# EU 화장품 규제 (EC 1223/2009) 상세 가이드

## 개요

**Regulation (EC) No 1223/2009**는 EU 화장품 규제의 핵심 법령으로, 세계에서 가장 체계적이고 포괄적인 화장품 규제 체계를 구축하고 있습니다.

### 기본 정보
| 항목 | 내용 |
|------|------|
| **법령명** | Regulation (EC) No 1223/2009 on cosmetic products |
| **발효일** | 2009년 12월 22일 |
| **적용일** | 2013년 7월 11일 |
| **적용 범위** | EU 27개국 + EEA (노르웨이, 아이슬란드, 리히텐슈타인) |
| **관리 기관** | European Commission DG GROW |

### 법적 효력
- **Regulation (규정)**: EU 전역에 직접 적용 (국내 입법 불필요)
- 회원국 간 **단일 시장 원칙** 적용
- 한 회원국에서 합법적 판매 제품은 타 회원국에서도 판매 가능

## 핵심 개념

### 1. Responsible Person (RP, 책임자)

EU 시장에 화장품을 출시하는 데 **법적 책임**을 지는 자

#### RP의 정의
```
EU 제조사 → 제조사가 RP
EU 수입사 → 수입사가 RP
RP 지정 → 제3자 지정 가능 (서면 동의 필요)
```

#### RP의 의무
1. **안전성 확보**: CPSR 작성 및 보관
2. **규정 준수**: 성분, 라벨, 클레임 규정 준수
3. **통지**: CPNP 등록 및 업데이트
4. **PIF 보관**: 시장 출시 후 10년간 보관
5. **사후 관리**: 부작용 보고, 리콜 대응

#### RP 요건
- EU/EEA 내 법인 또는 자연인
- 물리적 주소 보유 필수
- EU 외 기업은 반드시 EU 내 RP 지정

### 2. CPSR (Cosmetic Product Safety Report)

모든 화장품은 시장 출시 전 **안전성 평가** 필수

#### CPSR 구성

**Part A: 화장품 안전성 정보 (Safety Information)**
```
1. 정량적/정성적 성분 구성
   - 전 성분 목록 (INCI명, CAS No., 농도)
   - 성분별 기능 및 사양

2. 물리화학적 특성 및 안정성
   - pH, 점도, 밀도 등
   - 안정성 시험 결과

3. 미생물학적 품질
   - 미생물 한도 시험
   - Challenge Test (방부력 시험)

4. 불순물, 미량물질, 포장재 정보
   - 중금속, 잔류용매
   - 용기 적합성

5. 정상 및 합리적 예측 가능 사용
   - 사용 방법 및 부위
   - 오용 가능성

6. 화장품 노출
   - 사용 빈도 및 양
   - 체류 시간

7. 성분 노출
   - 성분별 노출량 계산
   - SED (Systemic Exposure Dose)

8. 성분의 독성학적 프로파일
   - NOAEL/LOAEL
   - 발암성, 변이원성, 생식독성

9. 바람직하지 않은 효과
   - 알려진 부작용
   - 과거 부작용 보고

10. 화장품에 대한 정보
    - 기존 데이터
    - 인체적용시험 결과
```

**Part B: 화장품 안전성 평가 (Safety Assessment)**
```
1. 평가 결론
   - 안전함/안전하지 않음 판정
   - 사용 조건

2. 경고문 및 사용 지침
   - 필수 경고문
   - 사용법

3. 평가 근거
   - 판단 이유
   - 참고 문헌

4. 평가자 자격 및 서명
   - 평가자 이름, 자격
   - 날짜 및 서명
```

#### Safety Assessor 자격
- 약학, 독성학, 의학, 또는 관련 분야 **대학 학위**
- **독성학적 평가 이해** 입증 가능
- EU Directive 2005/36/EC에 따른 자격 인정

### 3. PIF (Product Information File)

RP가 보관해야 하는 **제품 정보 파일**

#### PIF 구성
```
PIF Contents:
├── 제품 설명
├── CPSR (Part A + B)
├── 제조방법 기술서
├── GMP 적합성 증명 (ISO 22716)
├── 효능 입증 자료 (클레임 있는 경우)
├── 동물실험 데이터 (기존 자료만, 2013년 이전)
└── 라벨 사본
```

#### PIF 보관 요건
- **장소**: RP 주소지
- **기간**: 마지막 배치 출시 후 **10년**
- **접근성**: 당국 요청 시 즉시 제공

### 4. CPNP (Cosmetic Products Notification Portal)

EU 판매 화장품의 **통지 포털**

#### CPNP 등록 절차
```
1. SAAS 계정 생성
   └── EU Login 계정 필요

2. 조직 등록
   └── RP 정보 입력

3. 제품 등록
   ├── 제품명
   ├── 카테고리
   ├── Frame Formulation 또는 전성분
   ├── 라벨 이미지
   ├── 원산지
   ├── RP 정보
   ├── 나노물질 정보 (해당 시)
   └── CMR 물질 정보 (해당 시)

4. 시장 출시
   └── 등록 완료 후 가능
```

#### Frame Formulation
정확한 농도 공개 없이 범위로 표시:
```
예시:
- AQUA: 50-80%
- GLYCERIN: 5-10%
- NIACINAMIDE: 1-5%
- PRESERVATIVES: <1%
```

#### CPNP 업데이트 의무
- 성분 변경 시
- RP 변경 시
- 제품 단종 시

## 나노물질 규제

### 정의 (Article 2(1)(k))
> "나노물질"이란 의도적으로 제조되어 1-100nm 크기의 입자가 수 기준으로 **50% 이상**인 불용성 또는 생체 지속성 물질

### 규제 요건

#### 1. 사전 통지 (Article 16)
```
요건:
├── 시장 출시 6개월 전 Commission 통지
├── CPNP 별도 등록
└── SCCS 검토 대기 가능

통지 정보:
├── 물질 식별 (INCI, IUPAC, CAS)
├── 입자 크기 분포
├── 물리화학적 특성
├── 예상 사용량
├── 독성학적 프로파일
├── 안전성 데이터
└── 노출 조건
```

#### 2. 라벨 표기 (Article 19(1)(g))
```
표기 방법: [INCI명] (nano)
예시:
├── TITANIUM DIOXIDE (NANO)
├── ZINC OXIDE (NANO)
└── SILICA (NANO)
```

#### 3. 안전성 평가
나노 형태 특유의 독성학적 고려:
- 피부 투과성
- 흡입 독성
- 광독성/광촉매 활성
- 생체 축적성

## 클레임 규제

### EU Regulation 655/2013

화장품 클레임 **6대 원칙**:

1. **Legal Compliance (법률 준수)**
   - 의약품 영역 침범 금지
   - 국가별 추가 규정 준수

2. **Truthfulness (진실성)**
   - 허위 또는 오해 유발 금지
   - 존재하지 않는 특성 주장 금지

3. **Evidential Support (증거 기반)**
   - 클레임 입증 자료 보유
   - 적절한 시험/연구 수행

4. **Honesty (정직성)**
   - 공정한 표현
   - 과장 금지

5. **Fairness (공정성)**
   - 경쟁사 비방 금지
   - 공정 경쟁

6. **Informed Decision Making (정보에 기반한 결정)**
   - 소비자가 이해 가능한 표현
   - 명확한 정보 제공

### 금지 클레임 예시
- "질병을 치료합니다"
- "주름을 영구히 제거합니다"
- "100% 천연" (합성 방부제 포함 시)
- "무해합니다" (절대적 표현)

```

├── 1. 적용 범위

│   └── 세척 및 소독

│   ├── 구매 및 입고

│   ├── 작업 지시서
```

## 동물실험 금지

### 규정 (Article 18)
```
금지 사항:
├── 동물실험 화장품 판매 (2013.3.11~)
├── 동물실험 성분 사용 (2013.3.11~)
├── 동물실험 데이터 활용 (신규)
└── 예외 없음 (복잡한 독성 포함)

허용 사항:
├── 2013.3.11 이전 기존 데이터
├── 다른 법령 목적 시험 (REACH 등)
└── 대체시험법 사용
```

### 대체시험법
- In vitro 피부 자극 시험
- In vitro 안 자극 시험
- In vitro 피부 감작성 시험
- QSAR (구조-활성 관계)
- Read-across

## 부작용 보고

### Cosmetovigilance (Article 23)

#### RP 의무
- 심각한 바람직하지 않은 효과 (SUE) 인지 시 즉시 보고
- 회원국 당국 및 RP 소재국 당국에 통지
- 제품 안전성 재평가

#### SUE 정의
> 사망, 생명 위협, 입원, 선천적 기형, 지속적 장애를 초래하는 바람직하지 않은 효과

## 요약

EU 화장품 규제의 핵심:

1. **Responsible Person**: EU 내 법적 책임자 필수
2. **CPSR**: 모든 제품 안전성 평가 필수
3. **CPNP**: 시장 출시 전 통지 필수
4. **Annex 준수**: 금지/제한 물질 철저 확인
5. **동물실험 금지**: 대체시험법 사용
6. **GMP**: ISO 22716 준수 권장

EU는 세계에서 가장 엄격한 화장품 규제를 운영하므로, EU 기준으로 제품을 개발하면 다른 시장 진출이 용이합니다.
