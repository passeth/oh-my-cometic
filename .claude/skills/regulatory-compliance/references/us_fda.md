# 미국 FDA 화장품 규제 상세 가이드

## 개요

**FDA (Food and Drug Administration)**는 미국의 화장품 규제를 담당하며, 전통적으로 **자율 규제** 기반이었으나 **MoCRA (2022)**로 크게 강화되었습니다.

### 기본 정보
| 항목 | 내용 |
|------|------|
| **규제 기관** | FDA (Food and Drug Administration) |
| **근거 법령** | Federal Food, Drug, and Cosmetic Act (FD&C Act) |
| **주요 개정** | MoCRA (2022년 12월 29일 시행) |
| **적용 범위** | 미국 내 판매 모든 화장품 |
| **관리 부서** | CFSAN (Center for Food Safety and Applied Nutrition) |

---

## Cosmetic vs Drug 구분

미국 화장품 규제의 **가장 중요한 개념**

### 법적 정의

#### Cosmetic (화장품)
> "인체를 세정, 미화, 매력 증진, 외관 변경하기 위해 인체에 바르거나 붓거나 뿌리거나 분사하거나 소개하거나 기타 방법으로 적용하는 물품"

#### Drug (의약품)
> "인간 또는 동물의 질병 진단, 치료, 완화, 치료 또는 예방에 사용하는 물품" 또는 "인체의 구조나 기능에 영향을 미치는 물품 (음식 제외)"

### 분류 기준

| 기준 | Cosmetic | Drug |
|------|----------|------|
| **목적** | 미화, 청결 | 치료, 예방, 구조/기능 영향 |
| **클레임** | 외관 변화 | 치료적 효과 |
| **규제** | 자율적 (MoCRA로 강화) | FDA 승인 필수 |
| **라벨** | FD&C Act 21 CFR 701 | 21 CFR 201 |

### 제품 분류 예시

#### Cosmetic으로 분류
```
- 클렌저 (세정)
- 모이스처라이저 (보습)
- 립스틱 (색조)
- 향수 (향기)
- 샴푸 (청결 목적)
- 네일 폴리시 (미화)
- 바디로션 (피부 연화)
```

#### Drug으로 분류
```
- 자외선차단제 (SPF 표시)
- 여드름 치료제 (Acne treatment)
- 비듬 샴푸 (Dandruff treatment)
- 탈모 치료제 (Hair loss treatment)
- 데오도란트 (땀 억제 주장)
- 치약 (충치 예방)
- 피부 미백제 (Skin bleaching)
```

#### 경계선 제품 (Drug + Cosmetic)
```
보습 자외선차단제
├── Cosmetic: 보습 효과
└── Drug: 자외선차단 효과
→ 두 규제 모두 적용

항비듬 샴푸
├── Cosmetic: 모발 청결
└── Drug: 비듬 치료
→ 두 규제 모두 적용
```

## OTC Drug Monograph 시스템

Drug으로 분류된 제품 중 **OTC (Over-the-Counter)** 의약품은 Monograph 시스템 적용

### Monograph 정의

FDA가 발행하는 **일반 규칙**으로, 특정 OTC 약품 카테고리의 활성 성분, 농도, 라벨, 테스트 요건을 규정

### 주요 OTC Monograph (화장품 관련)

#### 1. Sunscreen Monograph (21 CFR 352)
```
활성 성분 (UV 필터):
├── Avobenzone: max 3%
├── Homosalate: max 15%
├── Octinoxate: max 7.5%
├── Octisalate: max 5%
├── Octocrylene: max 10%
├── Oxybenzone: max 6%
├── Titanium Dioxide: max 25%
├── Zinc Oxide: max 25%
└── 기타 승인 성분...

라벨 요건:
├── SPF 값 표시 (SPF 15+)
├── Broad Spectrum 여부
├── Water Resistance 시간
├── 경고문 (피부암 관련)
└── 사용 지침

테스트 요건:
├── SPF 테스트 (in vivo)
├── Broad Spectrum 테스트
└── Water Resistance 테스트
```

#### 2. Acne Monograph (21 CFR 333.310)
```
활성 성분:
├── Benzoyl Peroxide: 2.5-10%
├── Salicylic Acid: 0.5-2%
├── Sulfur: 3-10%
└── Sulfur + Resorcinol: 3-8% + 2%

라벨 요건:
├── "For treatment of acne"
├── 사용 지침
└── 경고문
```

#### 3. Dandruff/Seborrheic Dermatitis Monograph (21 CFR 358)
```
활성 성분:
├── Pyrithione Zinc: 0.3-2%
├── Selenium Sulfide: 1%
├── Coal Tar: 0.5-5%
├── Salicylic Acid: 1.8-3%
└── Sulfur: 2-5%

라벨 요건:
├── "Anti-dandruff" 또는 "Dandruff shampoo"
└── 경고문 (눈 접촉 등)
```

#### 4. Antiperspirant Monograph (21 CFR 350)
```
활성 성분:
├── Aluminum Chloride
├── Aluminum Chlorohydrate
├── Aluminum Zirconium complexes
└── 기타 알루미늄 염

클레임:
├── "Reduces underarm perspiration"
├── 24/48-hour protection (테스트 시)
└── "Extra strength" (조건 충족 시)
```

#### 5. Skin Protectant Monograph (21 CFR 347)
```
활성 성분:
├── Allantoin: 0.5-2%
├── Dimethicone: 1-30%
├── Petrolatum: 30-100%
├── Zinc Oxide: 1-25%
└── 기타...

클레임:
├── "Helps prevent and temporarily protects chafed, cracked, or chapped skin"
└── 특정 용도별 클레임
```

## MoCRA (Modernization of Cosmetics Regulation Act of 2022)

### 개요

2022년 12월 29일 시행된 **FDA 화장품 규제 현대화법**으로, 미국 화장품 규제 역사상 가장 큰 변화

### 핵심 요건

#### 1. 시설 등록 (Facility Registration)
```
의무: 화장품 제조/가공 시설 FDA 등록
기한: 2023년 12월 29일 (기존 시설)
      신규 시설: 제조 시작 60일 이내
갱신: 2년마다 갱신

등록 정보:
├── 시설명 및 주소
├── 모든 화장품 브랜드명
├── 화장품 카테고리
├── RP 정보 (해외 제조 시)
└── 시설 유형 (제조, 가공, 포장)
```

#### 2. 제품 리스팅 (Product Listing)
```
의무: 모든 시판 화장품 FDA 등록
기한: 2023년 12월 29일 (기존 제품)
      신규 제품: 출시 120일 이내
갱신: 매년 업데이트

등록 정보:
├── 제품명
├── 책임자 정보
├── 제조 시설
├── 성분 목록 (INCI명)
├── 제품 카테고리
└── 향료/착색제 (일반 용어 가능)
```

#### 3. 안전성 입증 (Safety Substantiation)
```
요건:
├── "합리적으로 예측 가능한 조건에서 안전"해야 함
├── 적절한 테스트 또는 기타 데이터
├── 전문가 평가

FDA 권한:
├── 안전성 기록 요청
└── 안전하지 않은 제품 시장 퇴출
```

#### 4. 부작용 보고 (Adverse Event Reporting)
```
의무: 심각한 부작용 15영업일 내 FDA 보고

심각한 부작용:
├── 사망
├── 생명 위협
├── 입원
├── 지속적 장애
├── 선천적 기형
└── 의료 개입 필요

기록 보관: 6년간 부작용 기록 유지
```

#### 5. GMP (Good Manufacturing Practice)
```
요건: FDA GMP 규정 준수
시행: FDA 규정 발행 후 2년
기준: ISO 22716 등 국제 표준 참조 예상

핵심 요소:
├── 시설 및 장비 관리
├── 원료 관리
├── 생산 및 품질 관리
├── 기록 관리
└── 불만/리콜 절차
```

#### 6. 향료 알레르겐 표시
```
요건: 향료 내 알레르겐 성분 개별 표시
시행: FDA 규정 발행 후
대상: FDA 지정 알레르겐 (목록 미정)
```

#### 7. 라벨 요건 강화
```
신규 요건:
├── 연락처 정보 (전화, 웹사이트 등)
├── 부작용 보고 방법 안내
└── 전문가용 제품 특별 표시
```

### 면제 대상

```
MoCRA 일부 요건 면제:
├── 소규모 사업자 (연 매출 $1M 미만)
│   └── 시설 등록, 제품 리스팅 면제
├── 자체 사용 제품
├── 연구/개발 목적
└── 특정 전문가 서비스
```

## 색소 규제

미국은 화장품 색소를 **별도로 규제**

### 색소 분류

#### 1. Certified Colors (인증 색소)
```
FD&C Colors:
├── FD&C Blue No. 1
├── FD&C Green No. 3
├── FD&C Red No. 3
├── FD&C Red No. 40
├── FD&C Yellow No. 5
├── FD&C Yellow No. 6
└── 기타...

D&C Colors:
├── D&C Red No. 6
├── D&C Red No. 7
├── D&C Yellow No. 10
└── 기타...

Ext. D&C Colors:
├── 외부 사용만 허용
└── 눈, 입술 주변 금지

요건:
├── 배치마다 FDA 인증 필수
├── LOT 번호 표시
└── 순도 기준 충족
```

#### 2. Exempt Colors (면제 색소)
```
인증 면제 (자연 유래):
├── Annatto
├── Beta-Carotene
├── Caramel
├── Carmine
├── Titanium Dioxide
├── Zinc Oxide
├── Iron Oxides
├── Mica
├── Ultramarine Blue
└── 기타...

요건:
├── FDA 인증 불필요
├── 순도 기준 충족
└── 사용 제한 준수
```

### 사용 제한

| 색소 | 허용 부위 |
|------|-----------|
| FD&C | 일반적으로 모든 부위 |
| D&C | 눈 제외 (일부 예외) |
| Ext. D&C | 외부만 (눈, 입술 금지) |
| Iron Oxides | 눈 포함 허용 |
| Carmine | 알레르기 경고 필요 |

## 라벨 요건

### 21 CFR 701

#### 필수 표시 사항
```
1. 제품 식별 (Identity Statement)
   - 제품 유형/용도

2. 순 내용량 (Net Quantity)
   - 중량 또는 부피
   - Metric + US customary

3. 책임자 정보
   - 제조사/유통사/수입사명
   - 주소 (도시, 주, ZIP)

4. 성분 목록 (Ingredient Declaration)
   - 함량 순 (1% 이상)
   - 1% 미만: 순서 무관
   - INCI명 사용 권장

5. 경고문 (필요 시)
   - 특정 성분 관련
   - 사용법 관련
```

#### MoCRA 추가 요건
```
- 연락처 정보 (전화, 이메일, 웹사이트)
- 부작용 보고 정보
```

#### 경고문 예시
```
자외선차단제:
"Sun Alert: Limiting sun exposure, wearing protective clothing,
and using sunscreens may reduce the risks of skin aging,
skin cancer, and other harmful effects of the sun."

에어로졸:
"WARNING: Flammable. Do not use near fire or flame."
"Avoid spraying in eyes. Contents under pressure."

손톱 제품 (톨루엔):
"Intentional misuse by deliberately concentrating and
inhaling contents can be harmful or fatal."
```

## 테스트 및 안전성

### FDA 권장 테스트

```
기본 테스트:
├── 미생물 시험 (Microbial testing)
├── 방부력 시험 (Preservative efficacy test)
├── 안정성 시험 (Stability testing)
└── 호환성 시험 (Compatibility testing)

안전성 테스트:
├── 피부 자극 시험 (Patch test)
├── 감작성 시험 (HRIPT, RIPT)
├── 안 자극 시험 (Eye irritation)
└── 광독성 시험 (필요 시)
```

### 동물실험

```
미국 현황:
├── 연방 수준: 금지하지 않음
├── 주 수준: 캘리포니아, 네바다, 일리노이 등 금지
└── 대체시험법 권장

FDA 입장:
├── 필수는 아님
├── 안전성 입증 방법 중 하나
└── 대체시험법 인정
```

## 수입 화장품

### FDA 요건

```
수입 절차:
├── CBP (Customs and Border Protection) 신고
├── FDA 자동 검사 대상 여부 확인
├── Entry bond 제출
└── 통관

FDA 검사:
├── 라벨 적합성
├── 금지 성분 포함 여부
├── 색소 인증 확인
└── Import Alert 대상 확인

Import Alert:
├── DWPE (Detention Without Physical Examination)
├── 특정 제조사/국가 자동 억류
└── 해제를 위한 입증 필요
```

### MoCRA 추가 요건

```
해외 제조 시:
├── 미국 내 대리인 지정
├── 시설 등록 (해외 시설 포함)
├── 제품 리스팅
└── 부작용 보고 체계
```

## 주(State) 규제

### 캘리포니아

#### Prop 65 (Safe Drinking Water and Toxic Enforcement Act)
```
요건: 암/생식독성 물질 함유 시 경고 필요
대상: 캘리포니아 판매 제품
경고문: "WARNING: This product can expose you to chemicals
including [물질명], which is known to the State of California
to cause cancer/birth defects/reproductive harm."
```

#### Cruelty-Free Cosmetics Act (SB 1249)
```
시행: 2020년 1월 1일
내용: 동물실험 화장품 판매 금지
적용: 캘리포니아 내 판매
```

### 기타 주

```
동물실험 금지:
├── 네바다
├── 일리노이
├── 버지니아
├── 메릴랜드
├── 메인
├── 하와이
└── 뉴저지

PFAS 금지:
├── 메인
├── 워싱턴
├── 캘리포니아
└── 콜로라도 (예정)
```

## 실무 가이드

### 미국 시장 진입 체크리스트

```
□ 제품 분류 확인 (Cosmetic vs Drug)
  □ Drug인 경우 OTC Monograph 준수
□ MoCRA 요건 확인
  □ 시설 등록
  □ 제품 리스팅
  □ 미국 대리인 (해외 제조 시)
□ 성분 검토
  □ 금지 물질 확인 (21 CFR 700)
  □ 색소 규정 확인
□ 라벨 준비
  □ 필수 정보 포함
  □ 경고문 (해당 시)
  □ 성분 목록 (영어)
□ 테스트
  □ 안전성 입증 자료
  □ OTC Drug 테스트 (해당 시)
□ 주별 규제 확인
  □ Prop 65 (캘리포니아)
  □ 동물실험 금지법
```

## 요약

미국 화장품 규제의 핵심:

1. **Cosmetic vs Drug**: 클레임에 따른 분류 중요
2. **OTC Monograph**: Drug 분류 시 준수 필수
3. **MoCRA**: 2022년 대폭 강화된 규제
   - 시설 등록 의무
