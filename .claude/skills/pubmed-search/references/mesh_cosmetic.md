## 6. MeSH 검색 활용법

### 기본 사용
```
dermatology[MeSH Terms]
→ 피부과학 관련 모든 하위 용어 포함
```

### 하위 용어 제외
```
dermatology[MeSH Major Topic]
→ 주요 주제로 태깅된 논문만
```

### 정확한 용어만
```
"Skin Aging"[MeSH Terms:noexp]
→ 하위 용어 확장 없이 정확한 용어만
```

### 자동 확장 (Automatic Term Explosion)
```
Skin[MeSH Terms]
→ Epidermis, Dermis, Stratum Corneum 등 모든 하위 개념 포함
```

## 8. MeSH Browser 활용

### 온라인 도구
- **MeSH Browser**: https://meshb.nlm.nih.gov/
- **MeSH on Demand**: https://meshb.nlm.nih.gov/MeSHonDemand

### 활용 방법
1. 용어 검색 → 정확한 MeSH Term 확인
2. Tree 구조 확인 → 상위/하위 개념 파악
3. Entry Terms 확인 → 동의어 목록 획득
4. Scope Note 확인 → 용어 정의 이해
