# Penetration Enhancers for Cosmetic Actives

## Overview

투과 촉진제(Penetration Enhancers)는 활성 성분의 피부 투과를 증가시키는 물질 또는 방법입니다. 이 문서는 화학적 촉진제, 물리적 방법, 그리고 나노 전달 시스템을 포함한 포괄적인 투과 촉진 전략을 다룹니다.

## Chemical Penetration Enhancers

### 1. 용매계 촉진제 (Solvents)

#### 에탄올 (Ethanol)

**작용 기전:**
- 각질층 지질 추출
- 지질 유동성 증가
- 약물 열역학적 활성 증가

**특성:**
| 항목 | 내용 |
|-----|------|
| 최적 농도 | 20-40% |
| 촉진 배율 | 2-5배 |
| 적용 제형 | 토너, 에센스, 로션 |
| 장점 | 빠른 건조, 청량감 |
| 단점 | 건조함, 자극 가능 |

**사용 시 주의:**
```
- 장기 사용 시 피부 장벽 손상
- 민감성 피부에서 제한적 사용
- 지질 성분과 병용 권장
- 농도 > 50%에서 자극 급증
```

#### 프로필렌글라이콜 (Propylene Glycol)

**작용 기전:**
- 각질층 수화
- 케라틴과 상호작용
- 약물 용해도 증가

**특성:**
| 항목 | 내용 |
|-----|------|
| 최적 농도 | 5-20% |
| 촉진 배율 | 2-3배 |
| 적용 제형 | 크림, 로션, 젤 |
| 장점 | 안전성 높음, 보습 효과 |
| 단점 | 끈적임, 알레르기 가능 |

**조합 효과:**
```
PG 단독: 2-3배 촉진
PG + 에탄올: 5-8배 촉진 (시너지)
PG + 올레산: 4-6배 촉진
```

#### DMSO (디메틸설폭사이드)

**작용 기전:**
- 케라틴 단백질 변성
- 지질 이중층 교란
- 수분 대체 효과

**특성:**
| 항목 | 내용 |
|-----|------|
| 최적 농도 | 60-90% |
| 촉진 배율 | 10-20배 |
| 장점 | 매우 효과적 |
| 단점 | 자극, 냄새, 규제 제한 |

**화장품 적용:**
```
일반 화장품: 사용 제한 (EU 금지)
의약외품/OTC: 제한적 허용
의약품: 치료 목적 사용
```

#### 기타 용매

| 용매 | 촉진 배율 | 특징 |
|-----|----------|------|
| 이소프로판올 | 1.5-3배 | 에탄올과 유사, 덜 자극적 |
| 부틸렌글라이콜 | 1.5-2배 | PG 대체제, 저자극 |
| 에톡시디글라이콜 | 2-4배 | 우수한 용해력 |
| 디프로필렌글라이콜 | 1.5-2배 | PG 대비 저자극 |
| 헥실렌글라이콜 | 2-3배 | 항균 효과 겸함 |

### 2. 지방산 (Fatty Acids)

#### 올레산 (Oleic Acid, C18:1)

**작용 기전:**
- 각질층 지질 유동화
- "Kink" 구조가 패킹 교란
- 세라마이드 대체

**특성:**
| 항목 | 내용 |
|-----|------|
| 최적 농도 | 1-10% |
| 촉진 배율 | 5-10배 |
| 적용 제형 | 오일, 크림 |
| 장점 | 자연 유래, 효과적 |
| 단점 | 코메도제닉, 자극 가능 |

**화장품 적용:**
```
올리브오일 (70% 올레산): 천연 투과 촉진제
마카다미아오일: 올레산 + 팔미톨레산
아보카도오일: 올레산 풍부
```

#### 기타 지방산

| 지방산 | 탄소수 | 촉진 효과 | 특징 |
|-------|-------|----------|------|
| 라우르산 | C12:0 | 중간 | 코코넛오일 성분 |
| 미리스트산 | C14:0 | 중간 | 침투+안정화 |
| 팔미트산 | C16:0 | 낮음 | 피부 친화성 |
| 스테아르산 | C18:0 | 낮음 | 주로 제형 안정화 |
| 리놀레산 | C18:2 | 높음 | 장벽 회복 효과도 |
| 리놀렌산 | C18:3 | 매우 높음 | 불안정, 산화 주의 |

**체인 길이와 효과:**
```
촉진 효과: C10-C14 > C16-C18 > C6-C8
포화 vs 불포화: 불포화 >> 포화
cis vs trans: cis > trans
```

### 3. 계면활성제 (Surfactants)

#### 작용 기전
- 각질층 지질 가용화
- 케라틴 단백질 변성
- 막 투과성 증가

#### 계면활성제 유형별 효과

| 유형 | 예시 | 촉진 효과 | 자극성 |
|-----|------|----------|--------|
| 음이온 | SLS, SLES | 높음 | 높음 |
| 양이온 | CTAB, BKC | 높음 | 높음 |
| 양쪽성 | 코카미도프로필베타인 | 중간 | 낮음 |
| 비이온 | 폴리소르베이트, PEG | 중-높음 | 낮음 |

#### 화장품에서의 적용

**Tween 80 (폴리소르베이트 80)**
```
농도: 0.5-5%
촉진 배율: 2-4배
장점: 저자극, 유화 겸용
적용: 로션, 크림
```

**Poloxamer (플루로닉)**
```
농도: 1-10%
촉진 배율: 2-5배
특징: 온도 감응성 겔화
적용: 겔, 마스크팩
```

### 4. 테르펜 및 테르페노이드 (Terpenes)

#### 작용 기전
- 각질층 지질 추출
- 지질 유동성 증가
- 케라틴 상호작용

#### 주요 테르펜 촉진제

| 테르펜 | 출처 | 촉진 배율 | 특징 |
|-------|------|----------|------|
| 리모넨 | 시트러스 | 3-8배 | 향료 겸용, 자극 가능 |
| 멘톨 | 박하 | 2-5배 | 청량감, 잘 견딤 |
| 시네올 | 유칼립투스 | 3-6배 | 항균 효과 |
| 카르본 | 캐러웨이 | 2-4배 | 독특한 향 |
| 테르피넨-4-올 | 티트리 | 2-3배 | 항균 우수 |
| 리날룰 | 라벤더 | 2-4배 | 진정 효과 |

**천연 에센셜 오일:**
```
유칼립투스 오일: 1,8-시네올 주성분
티트리 오일: 테르피넨-4-올
페퍼민트 오일: 멘톨 + 멘톤
라벤더 오일: 리날룰 + 리날릴아세테이트

적용 농도: 0.5-3%
```

### 5. 아민 및 아마이드 (Amines/Amides)

#### Azone (1-dodecylazacycloheptan-2-one)

**특성:**
| 항목 | 내용 |
|-----|------|
| 작용 기전 | 지질 유동화, 단백질 상호작용 |
| 최적 농도 | 0.1-5% |
| 촉진 배율 | 5-15배 |
| 규제 상태 | 화장품 사용 제한 |

#### 요소 유도체

```
요소 (Urea):
  - 각질 연화, 수화
  - 농도: 2-10% (화장품)
  - 촉진 배율: 1.5-3배

N-메틸피롤리돈 (NMP):
  - 강력한 용매
  - 규제 주의 (EU 제한)
```

### 6. 화학적 촉진제 조합

#### 시너지 조합

```
조합 1: 프로필렌글라이콜 + 올레산
  - PG 10% + OA 5%
  - 단독 대비 3-5배 시너지

조합 2: 에탄올 + 테르펜
  - EtOH 30% + 리모넨 2%
  - 빠른 작용 + 효과 지속

조합 3: PG + Tween 80 + 올레산
  - 삼중 작용 기전
  - 6-10배 촉진
```

#### 최적화된 촉진제 시스템

**친수성 활성 성분용:**
```
에탄올 20%
프로필렌글라이콜 10%
Tween 80 2%
물 68%
```

**친유성 활성 성분용:**
```
올레산 5%
IPM (이소프로필미리스테이트) 10%
에탄올 15%
오일 70%
```

## Physical Penetration Methods

### 1. 마이크로니들 (Microneedles)

#### 작용 원리
- 피부에 미세 구멍 생성 (50-600 μm)
- 각질층 물리적 우회
- 성분 직접 전달

#### 마이크로니들 유형

| 유형 | 구조 | 적용 | 장점 |
|-----|------|------|------|
| 솔리드 | 금속/실리콘 | 사전 처리 후 적용 | 높은 강도 |
| 중공형 | 속이 빈 니들 | 액체 주입 | 정확한 용량 |
| 용해형 | 당/고분자 | 니들 자체가 용해 | 간편, 안전 |
| 코팅형 | 약물 코팅 | 삽입 후 용해 | 빠른 전달 |
| 팽윤형 | 하이드로겔 | 삽입 후 팽창 | 서방 효과 |

#### 화장품 마이크로니들

```
가정용 롤러:
  - 니들 길이: 0.25-0.5 mm
  - 용도: 성분 흡수 증가
  - 주의: 감염 위험 관리

전문가용:
  - 니들 길이: 0.5-2.0 mm
  - 용도: 깊은 주름, 흉터
  - 전문 시술 필요

용해형 패치 (화장품):
  - 니들 길이: 200-400 μm
  - 재료: 히알루론산, 콜라겐
  - 활성: 펩타이드, 비타민
```

### 2. 이온토포레시스 (Iontophoresis)

#### 작용 원리
- 전기장으로 이온화 분자 구동
- 전기삼투로 비이온 분자 동반
- 막 투과성 일시적 증가

#### 적용 조건

```
전류: 0.1-0.5 mA/cm²
시간: 10-30분
전극: 양극(+) - 양이온 약물
      음극(-) - 음이온 약물

적합한 약물:
  - 이온화 가능 (전하 보유)
  - MW < 10,000 Da
  - 수용성
```

#### 화장품 적용

| 활성 성분 | 전하 | 전극 | 효과 |
|----------|------|------|------|
| 비타민 C (AA) | 음(-) | 음극 | 미백 |
| 아미노산 | 양쪽성 | 양/음 | 영양 |
| 히알루론산 | 음(-) | 음극 | 보습 |
| 펩타이드 | 다양 | 의존 | 주름개선 |

### 3. 초음파 (Sonophoresis)

#### 작용 원리
- 초음파 에너지로 각질층 교란
- 캐비테이션 (기포 형성/붕괴)
- 열 효과
- 음향 스트리밍

#### 조건별 효과

```
저주파 (20-100 kHz):
  - 강한 캐비테이션
  - 높은 촉진 효과 (10-100배)
  - 깊은 침투
  - 자극 위험 있음

고주파 (1-3 MHz):
  - 온화한 작용
  - 중간 촉진 (2-10배)
  - 표층 침투
  - 안전성 높음
```

#### 화장품 기기

```
가정용 초음파 기기:
  - 주파수: 1-3 MHz
  - 강도: 0.5-2 W/cm²
  - 사용: 에센스/세럼과 함께
  - 효과: 2-5배 흡수 증가
```

### 4. 전기천공 (Electroporation)

#### 작용 원리
- 고전압 펄스로 막에 순간적 기공 형성
- 대형 분자 통과 가능
- 기공은 수 분 내 회복

#### 적용 파라미터

```
전압: 50-500 V
펄스 폭: 0.1-100 ms
펄스 수: 1-10회
촉진 효과: 100-10,000배

적합한 분자:
  - 친수성 대형 분자
  - DNA, 단백질
  - 고분자량 펩타이드
```

**화장품 적용 제한:** 전문 시술용으로만 사용

### 5. 레이저/광 보조 (Laser/Light-Assisted)

#### 프랙셔널 레이저

```
작용: 미세 열 손상 채널 생성
종류: CO2, Er:YAG
침투 깊이: 0.5-2 mm
적용: 시술 후 약물 전달 가능
```

#### 저출력 레이저/LED

```
작용: 광화학적 효과
파장: 630-850 nm
효과: 세포 활성화, 미약한 투과 증가
화장품: LED 마스크 제품
```

## Nano Delivery Systems

### 1. 리포좀 (Liposomes)

#### 구조 및 특성

```
구조: 인지질 이중층 소포
크기: 50-500 nm (최적: 100-200 nm)
구성:
  - 인지질 (포스파티딜콜린)
  - 콜레스테롤 (막 안정화)
  - 전하 지질 (선택적)
```

#### 피부 투과 기전

1. **융합 (Fusion)**: 각질층 지질과 융합
2. **흡착-침투**: 표면 흡착 후 내용물 방출
3. **경부속기 (Transappendageal)**: 모낭 경로
4. **유연한 통과**: 탄성 리포좀의 경우

#### 리포좀 유형

| 유형 | 특징 | 적용 |
|-----|------|------|
| 일반 리포좀 | 기본 인지질 | 친수성 활성 |
| 트랜스퍼좀 | 계면활성제 포함, 유연 | 깊은 침투 |
| 에토좀 | 에탄올 포함 | 증가된 투과 |
| 니오좀 | 비이온 계면활성제 | 안정성 향상 |
| 프로니오좀 | 건조 니오좀 | 저장 안정성 |

#### 화장품 리포좀 제형

```
기본 조성:
  - 레시틴: 2-5%
  - 콜레스테롤: 0.5-1%
  - 활성 성분: 0.5-2%
  - 물: 잔여량

제조 방법:
  1. 박막 수화법
  2. 역상 증발법
  3. 에탄올 주입법
  4. 고압 균질화법
```

### 2. 나노에멀전 (Nanoemulsion)

#### 구조 및 특성

```
구조: 초미세 유적자 분산
크기: 20-200 nm
타입:
  - O/W (수중유): 친수성 기제
  - W/O (유중수): 친유성 기제
외관: 투명 ~ 반투명
```

#### 장점

- 높은 안정성 (열역학적)
- 향상된 용해도
- 빠른 침투
- 우수한 제형 심미성

#### 제조 방법

```
고에너지 방법:
  - 고압 균질화
  - 초음파 처리
  - 마이크로플루이다이저

저에너지 방법:
  - 상전이 온도 (PIT)
  - 상전이 조성 (PIC)
  - 자발적 에멀전화
```

#### 화장품 나노에멀전 조성

```
O/W 나노에멀전:
  - 오일상: 5-15%
    (IPM, 카프릴/카프릭 트리글리세라이드)
  - 계면활성제: 3-10%
    (PEG-40 수소화피마자유, Tween 80)
  - 공계면활성제: 0-5%
    (글리세린, PG)
  - 활성 성분: 0.5-5%
  - 물: 잔여량
```

### 3. 고체 지질 나노입자 (SLN)

#### 구조 및 특성

```
구조: 고체 지질 매트릭스
크기: 50-1000 nm
지질: 글리세릴 베헤네이트, 세틸팔미테이트
안정화: 계면활성제 (Tween, Poloxamer)
```

#### 장점

- 높은 안정성
- 서방 효과
- 보호 효과 (산화, 광분해)
- 피부 부착성

#### 피부 투과 기전

1. **폐쇄 효과 (Occlusion)**: 수분 증발 방지
2. **지질 교환**: 각질층 지질과 상호작용
3. **모낭 축적**: 모낭 타겟팅

#### SLN 제조

```
고온 균질화:
  1. 지질 융해 (활성 성분 용해)
  2. 계면활성제 수용액 가열
  3. 혼합 및 고압 균질화
  4. 냉각 (고체화)

냉온 균질화:
  1. 지질-약물 융해
  2. 빠른 냉각 (고체화)
  3. 분쇄
  4. 계면활성제 수용액에 분산
  5. 고압 균질화 (냉온)
```

### 4. 나노구조 지질 캐리어 (NLC)

#### SLN 대비 개선점

```
SLN 한계:
  - 결정화로 약물 배출
  - 제한된 봉입률
  - 저장 중 불안정

NLC 해결책:
  - 고체 + 액체 지질 혼합
  - 불완전 결정 → 공간 확보
  - 높은 봉입률
  - 누출 감소
```

#### NLC 유형

| 유형 | 구조 | 특징 |
|-----|------|------|
| Type I (불완전 결정) | 액체 지질 분자 분산 | 가장 일반적 |
| Type II (무정형) | 비결정 매트릭스 | 최고 봉입률 |
| Type III (다중) | W/O/W 유사 | 친수성 성분용 |

### 5. 사이클로덱스트린 (Cyclodextrin)

#### 구조 및 특성

```
구조: 고리형 올리고당
유형:
  - α-CD: 6 글루코스, 작은 공동
  - β-CD: 7 글루코스, 중간 공동
  - γ-CD: 8 글루코스, 큰 공동

특성:
  - 외부: 친수성
  - 내부: 소수성 공동
  - 포접 복합체 형성
```

#### 화장품 적용

| 활성 성분 | CD 유형 | 효과 |
|----------|--------|------|
| 레티놀 | β-CD | 안정화, 투과 증가 |
| 비타민 E | γ-CD | 수용성화, 안정화 |
| 살리실산 | β-CD | 자극 감소 |
| 향료 | 다양 | 서방, 안정화 |

**포접 복합체 제조:**
```
방법:
  1. 공침전 (Co-precipitation)
  2. 슬러리 복합체화
  3. 분무건조
  4. 동결건조

비율: 약물:CD = 1:1 ~ 1:3 (몰비)
```

## Safety Considerations

### 자극 및 독성

#### 촉진제별 자극 등급

| 촉진제 | 자극 등급 | 농도 제한 | 비고 |
|-------|----------|----------|------|
| 에탄올 | 중간 | < 40% | 건조, 장벽 손상 |
| PG | 낮음 | < 30% | 드문 알레르기 |
| 올레산 | 중-높음 | < 10% | 코메도제닉 |
| DMSO | 높음 | 화장품 금지 | 독성 우려 |
| SLS | 높음 | < 1% | 세정제만 |
| 리모넨 | 중간 | < 5% | 산화물 자극 |

### 장벽 회복

```
투과 촉진 후 장벽 회복 전략:
  1. 세라마이드 포함 제품 사용
  2. 필수 지방산 (리놀레산) 보충
  3. 콜레스테롤 함유 보습제
  4. 니아시나마이드로 장벽 강화
```

### 전신 흡수 고려

```
전신 노출 위험 평가:
  - 투과 플럭스 × 적용 면적 = 총 흡수량
  - NOAEL과 비교하여 안전성 마진 계산
  - 민감군 (어린이, 임산부) 추가 고려

위험 성분:
  - 고독성 활성 성분
  - 내분비계 교란 우려 물질
  - 변이원성/발암성 물질
```

## Delivery System Selection Guide

### 활성 성분 특성별 선택

```
친수성 (log P < 0):
  선택: 리포좀 (내부 봉입), W/O 에멀전
  촉진제: PG, 에탄올

중간 친유성 (0 < log P < 3):
  선택: 모든 시스템 가능
  권장: 나노에멀전, 탄성 리포좀

친유성 (log P > 3):
  선택: SLN/NLC, O/W 나노에멀전
  촉진제: 올레산, 테르펜

고분자량 (MW > 500):
  선택: 마이크로니들, 이온토포레시스
  나노: 탄성 리포좀 (제한적)
```

### 목표 피부층별 선택

```
각질층 (표면):
  - 일반 크림/로션
  - 보습제, 자외선차단제

표피 (중간):
  - 리포좀, 나노에멀전
  - 미백제, 각질제거제

진피 (깊은):
  - 마이크로니들
  - 탄성 리포좀 + 촉진제
  - 주름개선제, 탄력제
```

### 제형 유형별 권장 시스템

| 제형 | 권장 전달 시스템 | 촉진제 |
|-----|-----------------|--------|
| 토너/에센스 | 나노에멀전, 리포좀 | 에탄올, PG |
| 세럼 | 리포좀, 사이클로덱스트린 | 저농도 계면활성제 |
| 로션 | 나노에멀전, SLN | PG, 올레산 |
| 크림 | NLC, 리포좀 | 올레산, 테르펜 |
| 오일 | 지질 나노입자 | 올레산, IPM |
| 패치 | 마이크로니들, 하이드로겔 | 멘톨, 유칼립톨 |
