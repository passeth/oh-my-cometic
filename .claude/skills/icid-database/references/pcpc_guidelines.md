# PCPC INCI 명명 가이드라인

## PCPC 소개

**PCPC (Personal Care Products Council)**는 미국의 화장품 및 개인용품 산업 협회로, INCI(International Nomenclature of Cosmetic Ingredients) 명명 체계를 개발하고 관리합니다.

### 조직 정보

| 항목 | 내용 |
|------|------|
| **정식 명칭** | Personal Care Products Council |
| **이전 명칭** | CTFA (Cosmetic, Toiletry, and Fragrance Association) |
| **설립** | 1894년 |
| **본부** | Washington, D.C., USA |
| **웹사이트** | https://www.personalcarecouncil.org/ |
| **역할** | INCI 관리, 업계 대변, 규제 협력 |

## INCI 등록 프로세스

### 등록 대상

```
INCI 등록이 필요한 경우:
  - 완전히 새로운 화학 물질
  - 기존에 등록되지 않은 천연 추출물
  - 새로운 복합 성분/혼합물
  - 기존 성분의 새로운 형태 (나노 등)

INCI 등록이 불필요한 경우:
  - 이미 등록된 성분 사용
  - 기존 INCI로 표현 가능한 혼합물
  - 순수 용매만으로 구성된 경우
```

## 명명 원칙 상세

### 1. 화학 성분 명명

#### 기본 원칙
```
우선순위:
1. 일반적으로 인정된 화학명
2. IUPAC 명명법
3. CAS 명칭
4. 기타 체계적 명칭

간소화 원칙:
- 불필요한 복잡성 제거
- 발음/표기 용이성 고려
- 기존 유사 명칭과 일관성 유지
```

#### 명명 예시
```
화학명 → INCI명 변환 예시:

3-Pyridinecarboxamide → NIACINAMIDE
(복잡한 화학명)         (간소화된 INCI)

3,7,11,15-Tetramethyl-2-hexadecen-1-ol → PHYTOL
(IUPAC 체계명)                           (관용명)

Poly(dimethylsiloxane) → DIMETHICONE
(폴리머 체계명)          (간소화)
```

### 2. 식물 성분 명명

#### 학명 사용 규칙
```
원칙: 이명법 (Binomial nomenclature)

[속명] [종명]
  - 라틴어 표기
  - 대문자로 전환
  - 현재 인정된 학명 사용

예시:
Camellia sinensis → CAMELLIA SINENSIS
Aloe barbadensis → ALOE BARBADENSIS
Glycyrrhiza glabra → GLYCYRRHIZA GLABRA
```

#### 학명 변경 대응
```
식물 분류학 변경 시:

구 학명 → 신 학명 (예시)
Chamomilla recutita → Matricaria chamomilla

INCI 대응:
- 두 명칭 모두 유효로 인정
- 새로운 등록은 현재 인정 학명 사용
- 기존 제품은 전환 기간 허용
```

#### 부위 및 형태 표기
```
표준화된 용어 사용:

부위 (Part):
LEAF, FLOWER, FRUIT, SEED, ROOT, BARK, STEM, PEEL, KERNEL

형태 (Form):
EXTRACT, OIL, BUTTER, WAX, WATER, POWDER, JUICE

조합 예시:
CAMELLIA SINENSIS LEAF EXTRACT
[──속명──] [종명] [부위] [형태]
```

### 3. 폴리머 명명

#### PEG 계열
```
명명 규칙:
PEG-[평균 분자량 지표]

지표 계산:
분자량 ÷ 44 = 대략적 숫자

예시:
MW ~400  → PEG-8
MW ~600  → PEG-12
MW ~1000 → PEG-20
MW ~4000 → PEG-100
```

#### Polyquaternium 계열
```
명명 규칙:
POLYQUATERNIUM-[등록 순서 번호]

예시:
POLYQUATERNIUM-7  (양이온성 폴리아크릴아미드)
POLYQUATERNIUM-10 (셀룰로오스 유도체)
POLYQUATERNIUM-44 (비닐피롤리돈 공중합체)

번호 = 등록 순서, 화학 구조와 무관
```

#### 기타 폴리머
```
관용명 사용:
CARBOMER (카르복시비닐 폴리머)
XANTHAN GUM (잔탄검)

공중합체:
ACRYLATES COPOLYMER
ACRYLATES/C10-30 ALKYL ACRYLATE CROSSPOLYMER
VP/VA COPOLYMER
```

### 4. 나노물질 명명

#### EU 규정 대응
```
EU Regulation 1223/2009 정의:
- 불용성/생체지속성 물질
- 1-100 nm 크기의 입자 포함
- 응집체/집합체도 해당 가능

표기법:
[INCI NAME] (NANO)

예시:
TITANIUM DIOXIDE (NANO)
ZINC OXIDE (NANO)
SILICA (NANO)
```

#### 나노 표기 적용 기준
```
해당 성분:
- 무기 자외선차단제 (TiO2, ZnO)
- 실리카
- 금속 산화물
- 카본블랙 등

표기 의무:
- EU: 성분 라벨에 (NANO) 필수 표기
- 다른 지역: 규정에 따라 상이
```

## 특수 명명 규칙

### 1. 색소 (Colorants)

```
CI 번호 체계:
CI [5자리 숫자]

분류:
CI 10000-19999: 모노아조 염료
CI 20000-29999: 디스아조 염료
CI 40000-49999: 트리아릴메탄
CI 50000-59999: 잔텐
CI 70000-79999: 무기 안료
CI 75000-79999: 천연 색소

표기 방식:
CI 77891 (TITANIUM DIOXIDE)
CI 77491 (IRON OXIDES)
```

### 2. 향료 (Fragrance)

```
일반 표기:
FRAGRANCE (미국식)
PARFUM (EU식)

알레르겐 표기:
EU 26종 알레르겐 성분 개별 표기 의무

농도 기준:
- Leave-on: 0.001% 초과 시 표기
- Rinse-off: 0.01% 초과 시 표기

예시 알레르겐:
LIMONENE, LINALOOL, CITRONELLOL, GERANIOL,
HEXYL CINNAMAL, COUMARIN, EUGENOL 등
```

### 3. 혼합물/복합 성분

```
AND 연결:
CETEARYL ALCOHOL (AND) CETEARYL GLUCOSIDE
→ 단일 상품으로 공급되는 혼합물

슬래시 연결:
CAPRYLIC/CAPRIC TRIGLYCERIDE
→ 화학적으로 유사한 혼합물

탄소수 범위:
C12-15 ALKYL BENZOATE
→ C12, C13, C14, C15 알킬 혼합
```

## 명칭 변경/업데이트

### 변경 사유
```
INCI명 변경이 발생하는 경우:

1. 학명 변경 (식물 분류학 업데이트)
2. 화학 구조 정정
3. 명명 규칙 표준화
4. 오류 수정
5. 규제 요구사항
```

## FAQ

### Q: 비회원도 INCI 등록 가능?
```
A: 직접 등록은 불가능합니다.

방법:
1. PCPC 회원사를 통해 스폰서 신청
2. 컨설팅 회사 활용
3. 원료 공급업체에 요청

원료 공급업체 대부분이 회원사이므로,
새 원료 구매 시 INCI 등록 요청 가능
```

### Q: 등록 소요 기간?
```
A: 일반적으로 4-8주

변동 요인:
- 제출 정보 완성도
- 복잡한 성분 구조
- 추가 정보 요청
- 명명 규칙 적합성 검토

긴급 처리: 추가 비용으로 2-3주 가능
```

### Q: 등록 전 임시 명칭 사용?
```
A: 가능합니다.

권장 방법:
1. 화학명 사용 (예: "3-Pyridinecarboxamide")
2. CAS 번호 참조 (예: "CAS 98-92-0")
3. 상품명 + 설명 (예: "ABC Ingredient (INCI pending)")

주의:
- 라벨에는 공식 명칭 필요
- 내부 문서/처방전에는 임시 명칭 가능
- 규제 제출 시 공식 명칭 필수
```

### Q: INCI명 변경 통보 방법?
```
A: 여러 채널을 통해 확인 가능

1. PCPC 웹사이트 업데이트
2. 구독자 이메일 통보
3. 분기별 업데이트 노트
4. 연간 인쇄본 부록

권장:
- 분기별 Dictionary 업데이트 확인
- 주요 원료 변경사항 모니터링
- 규제 기관 공지 확인
```
