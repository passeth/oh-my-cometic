# 화장품 제형 유형별 특성

## 제형 분류

### 1. 토너/스킨
- 점도: 1-50 cps
- pH: 4.5-6.5
- 수분: 85-98%
- 오일: 0-5%

### 2. 에센스/세럼
| 항목 | 에센스 | 세럼 |
|------|--------|------|
| 점도 | 50-500 cps | 500-5000 cps |
| 농축도 | 중간 | 높음 |
| 텍스처 | 가벼움 | 약간 무거움 |

### 3. 로션
- 점도: 1,000-10,000 cps
- 오일: 5-20%
- 에멀전: O/W

### 4. 크림
| 유형 | 점도 | 오일 함량 |
|------|------|-----------|
| 라이트 | 10k-50k cps | 15-25% |
| 리치 | 50k-200k cps | 25-40% |

### 5. 겔
- 점도: 5,000-100,000 cps
- 수분: 80-95%
- 점증제: Carbomer, HEC

## 기본 배합 예시

### 토너
```
정제수          to 100%
Butylene Glycol    5.0%
Glycerin           3.0%
활성 성분       as needed
Phenoxyethanol     0.5%
```

### 수성 세럼
```
정제수          to 100%
Glycerin           5.0%
Butylene Glycol    5.0%
Sodium HA          0.5%
Carbomer           0.2%
TEA              q.s.
활성 성분       as needed
Phenoxyethanol     0.8%
```

### O/W 크림
```
A상 (수상) 70°C
정제수          to 100%
Glycerin           5.0%

B상 (유상) 70°C
Cetearyl Alcohol   3.0%
Glyceryl Stearate  2.0%
PEG-100 Stearate   1.5%
오일상          15-20%

C상 (40°C 이하)
활성 성분       as needed
Phenoxyethanol     0.8%
```

### 하이드로겔
```
정제수          to 100%
Glycerin           5.0%
Carbomer           0.3%
TEA              q.s.
활성 성분       as needed
Phenoxyethanol     0.8%
```
