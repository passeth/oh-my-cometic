# CIR Assessment Process

CIR (Cosmetic Ingredient Review) 평가 프로세스 상세 가이드

## Expert Panel 구성

### 패널 구성원

CIR Expert Panel은 다음 분야의 전문가 7~9명으로 구성됩니다:

| 전문 분야 | 역할 |
|----------|------|
| **피부과학 (Dermatology)** | 피부 안전성, 자극/감작성 평가 |
| **독성학 (Toxicology)** | 전신 독성, 유전독성, 발암성 평가 |
| **약리학 (Pharmacology)** | 피부 흡수, 대사, 약물동태학 |
| **병리학 (Pathology)** | 조직학적 변화, 장기 독성 |
| **화학 (Chemistry)** | 성분 특성, 안정성, 불순물 |
| **소비자 안전 (Consumer Safety)** | 사용 패턴, 노출량 평가 |

### 독립성 보장

- 패널 위원은 PCPC와 독립적으로 활동
- 이해충돌 시 해당 성분 평가에서 제외
- 투표 및 결정 과정 공개

## 평가 단계 (Assessment Stages)

### Stage 1: Priority List (우선순위 목록)

```
목적: 평가 대상 성분 선정 및 우선순위 결정

선정 기준:
- 사용 빈도 및 사용량
- 안전성 우려 제기 여부
- 규제 기관 관심 성분
- 재평가 필요 성분 (15년 주기)
- 업계 요청

프로세스:
1. 성분 후보 목록 작성
2. 사용 데이터 수집 (VCRP, FDA, 업계)
3. 기존 안전성 데이터 예비 검토
4. 우선순위 결정 및 Priority List 발표
```

### Stage 2: Scientific Literature Review (문헌 조사)

```
목적: 이용 가능한 모든 과학적 데이터 수집

수집 대상:
- 발표된 연구 논문 (peer-reviewed)
- 규제 기관 평가 자료 (FDA, EU SCCS, etc.)
- 원료사 제출 자료 (unpublished)
- 국제 데이터베이스 (ECHA, HSDB, etc.)
- 동물대체시험 데이터

데이터 유형:
- 물리화학적 특성
- 피부 흡수/침투
- 급성 독성
- 만성 독성
- 유전독성
- 발암성
- 생식발달독성
- 피부 자극/감작성
- 안점막 자극성
- 광독성/광알레르기
- 사용 농도 및 노출량
```

### Stage 3: Draft Report (초안 보고서)

```
목적: 수집된 데이터를 바탕으로 초안 작성

구성:
1. Introduction (서론)
   - 성분 정의 및 특성
   - 사용 목적 및 기능

2. Chemistry (화학적 특성)
   - 구조 및 물성
   - 제조 방법
   - 불순물 프로파일

3. Use (사용 현황)
   - 사용 제품 유형
   - 사용 농도 범위
   - 노출량 추정

4. Toxicological Studies (독성학 연구)
   - 각 독성 엔드포인트별 데이터 요약
   - 데이터 품질 평가

5. Summary (요약)
   - 주요 발견사항
   - 데이터 갭 식별

공개 및 검토:
- 웹사이트에 Draft Report 공개
- 60일 공개 의견수렴 기간
- 업계, 학계, 일반 시민 의견 접수
```

### Stage 4: Expert Panel Meeting (전문가 패널 회의)

```
목적: Draft Report 검토 및 예비 결론 도출

회의 진행:
1. CIR 스태프의 데이터 프레젠테이션
2. 패널 위원 토론
3. 공개 의견 검토
4. 추가 데이터 요청 여부 결정
5. 예비 결론 투표

가능한 결정:
- Safe as used (현재 사용 조건에서 안전)
- Safe with qualifications (조건부 안전)
- Insufficient data (데이터 부족)
- Unsafe (안전하지 않음)
- Table (보류 - 추가 데이터 필요)

회의 공개:
- 회의 일정 사전 공지
- 회의 방청 가능
- 회의록 공개
```

### Stage 5: Tentative Report (잠정 보고서)

```
목적: 예비 결론 포함 잠정 보고서 발행

내용:
- Draft Report 업데이트
- Expert Panel 토론 요약
- 예비 안전성 결론
- 조건부 안전인 경우 상세 조건

공개 의견수렴:
- 60일 추가 의견수렴 기간
- 결론에 대한 이의 제기 가능
- 추가 데이터 제출 기회
```

### Stage 6: Final Report (최종 보고서)

```
목적: 공식 안전성 결론 확정 및 발표

프로세스:
1. 공개 의견 검토
2. 필요시 결론 수정
3. Final Report 작성
4. Expert Panel 최종 승인
5. International Journal of Toxicology 게재
6. CIR 웹사이트 공개

최종 결론 유형:
- Safe as used in present practices of use and concentration
- Safe as used when formulated to be non-irritating/non-sensitizing
- Safe as used in the present practices of use and concentration
  when [specific conditions]
- Data insufficient to support safety
- Unsafe for use in cosmetic products

법적 효력:
- CIR 결론은 법적 구속력 없음 (권고 사항)
- 그러나 FDA 및 국제 규제 기관에서 참고 자료로 활용
- 업계 자율 규제의 근거
```

## 데이터 요구사항 (Data Requirements)

### 필수 데이터

안전성 평가를 위해 일반적으로 필요한 데이터:

#### 1. 물리화학적 특성
```
- 분자 구조 및 분자량
- 물리적 형태 (액체, 고체, 분말)
- 용해도 (물, 오일, 용매)
- pH (해당 시)
- 안정성 (온도, 빛, pH)
- 순도 및 불순물
- 입자 크기 (나노 물질인 경우)
```

#### 2. 피부 흡수 데이터
```
- In vitro 피부 투과 시험 (Franz diffusion cell)
- In vivo 흡수 연구 (해당 시)
- 대사 연구
- Molecular weight 기반 예측
- LogP/LogKow (지질 친화성)
```

#### 3. 독성학 데이터

| 엔드포인트 | 시험 유형 | 중요도 |
|-----------|----------|-------|
| **급성 독성** | LD50, LC50 | 중요 |
| **피부 자극** | Patch test, RhE | 필수 |
| **눈 자극** | Draize, EpiOcular | 필수 |
| **피부 감작** | LLNA, DPRA, KeratinoSens | 필수 |
| **유전독성** | Ames, 소핵시험, 염색체이상 | 필수 |
| **반복투여독성** | 28일/90일 | 중요 |
| **발암성** | 2년 설치류, 역학 | 중요 |
| **생식독성** | OECD 421/422 | 중요 |
| **발달독성** | Teratogenicity | 중요 |
| **광독성** | 3T3 NRU PT | 해당 시 |
| **광알레르기** | LLNA with UV | 해당 시 |

#### 4. 사용 데이터
```
- 사용 제품 유형 및 빈도
- 사용 농도 범위 (업계 설문)
- 노출 시나리오
- 소비자 사용 패턴
```

### 데이터 갭 (Data Gaps)

Insufficient data 결론 시 요구되는 추가 데이터:

```
일반적인 데이터 갭:
1. Dermal absorption data (피부 흡수 데이터)
   - 분자량 500 Da 이상인 경우에도 요구될 수 있음

2. Sensitization data (감작성 데이터)
   - 특히 단백질 결합 가능성 있는 성분

3. Reproductive/developmental toxicity data (생식발달독성)
   - 피부 흡수가 유의미한 경우

4. Genotoxicity battery (유전독성 배터리)
   - 최소 3가지 시험 필요 (Ames + in vitro + in vivo)

5. Use concentration data (사용 농도 데이터)
   - 현재 시장 사용 현황 파악

6. Impurity profile (불순물 프로파일)
   - 특히 합성 성분의 경우
```

## Re-review 주기 (Re-review Cycle)

### 15년 재평가 정책

```
CIR은 약 15년 주기로 기존 평가를 재검토합니다:

재평가 트리거:
1. 15년 경과 (자동 재평가)
2. 새로운 안전성 우려 제기
3. 규제 환경 변화
4. 신규 과학적 데이터 발표
5. 업계 요청

재평가 프로세스:
1. 기존 보고서 검토
2. 새로운 데이터 수집
3. 사용 현황 업데이트
4. Expert Panel 재검토
5. 결론 재확인 또는 수정
```

### 재평가 결과 유형

```
1. Conclusion Reaffirmed (결론 재확인)
   - 기존 결론 유지
   - 새로운 데이터가 결론을 뒷받침

2. Conclusion Revised (결론 수정)
   - 안전성 결론 변경
   - 새로운 조건 추가 또는 삭제

3. Re-review Deferred (재평가 연기)
   - 추가 데이터 필요
   - 진행 중인 연구 완료 대기

4. Split into Multiple Reports (분할)
   - 그룹 평가를 개별 평가로 분할
   - 또는 개별을 그룹으로 통합
```

## 평가 히스토리 예시

### Parabens (파라벤류) 평가 이력

```
1984년: 최초 평가
- 결론: Safe as used
- 대상: Methyl-, Ethyl-, Propyl-, Butylparaben

2006년: 재평가
- 에스트로겐 활성 우려 제기 후 재검토
- 결론: Safe as used (유지)
- 근거: 화장품 사용 수준에서 내분비계 영향 무시할 수 있음

2019년: 2차 재평가
- EU 규제 변화 및 추가 연구 반영
- 결론: Safe as used (유지)
- 추가 확인: 조합 사용에서도 안전
```

### Retinol (레티놀) 평가 이력

```
1987년: 최초 평가 (Vitamin A 그룹)
- 결론: Safe with qualifications
- 조건: 농도 제한, 자외선 노출 주의

2012년: 재평가
- 광독성, 피부 자극 추가 데이터 검토
- 결론: Safe with qualifications (유지)
- 업데이트된 농도 기준 제시

2021년: 최신 재평가
- 임신 중 안전성 검토
- 결론: Safe with qualifications
- 추가 권고: 임산부 사용 관련 경고 권장
```

## 의견 제출 방법

### 공개 의견수렴 참여

```
1. CIR 웹사이트에서 진행 중인 평가 확인
2. Draft Report 또는 Tentative Report 다운로드
3. 의견서 작성:
   - 성분명 및 보고서 유형 명시
   - 구체적 데이터 또는 우려사항 기술
   - 참고문헌 포함 (가능한 경우)
   - 연락처 정보

4. 제출:
   - 이메일: cirinfo@cir-safety.org
   - 우편: CIR, 1620 L Street, NW, Suite 1200, Washington, DC 20036

5. 응답:
   - CIR에서 의견 검토 후 회신
   - Expert Panel 회의에서 의견 논의
```
