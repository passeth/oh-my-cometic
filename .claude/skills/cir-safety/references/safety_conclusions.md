# CIR Safety Conclusions

CIR 안전성 결론 유형 및 주요 성분별 요약

## 안전성 결론 유형

### 1. Safe as Used (현재 사용 조건에서 안전)

#### 정의
```
"Safe as used in cosmetics" 또는
"Safe as used in the present practices of use and concentration"

의미:
- 현재 화장품에 사용되는 농도 및 용도에서 안전
- 추가적인 제한이나 조건 없이 사용 가능
- 일반적인 화장품 용도 전체에 적용

적용 기준:
- 충분한 안전성 데이터 확보
- 현재 사용 수준에서 독성학적 우려 없음
- 피부 자극/감작성 우려 낮음
```

#### Safe as Used 성분 예시

| 성분 (INCI) | 한글명 | 주요 용도 | 평가 연도 | 비고 |
|------------|--------|----------|----------|------|
| Glycerin | 글리세린 | 보습제 | 2019 재확인 | 가장 널리 사용 |
| Niacinamide | 나이아신아마이드 | 피부컨디셔닝 | 2020 재평가 | 5%까지 일반적 |
| Hyaluronic Acid | 히알루론산 | 보습제 | 2022 | 분자량별 평가 |
| Tocopherol | 토코페롤 | 항산화제 | 2018 재평가 | 비타민E |
| Panthenol | 판테놀 | 헤어컨디셔닝 | 2018 | 프로비타민B5 |
| Allantoin | 알란토인 | 피부보호제 | 2021 | 진정 효과 |
| Squalane | 스쿠알란 | 피부연화제 | 2020 | 식물 유래 포함 |
| Butylene Glycol | 부틸렌글라이콜 | 용매/보습 | 2019 | 프로판디올류 |
| Cetyl Alcohol | 세틸알코올 | 유화안정제 | 2019 | 지방알코올류 |
| Stearic Acid | 스테아릭애씨드 | 유화제 | 2019 | 지방산류 |

### 2. Safe with Qualifications (조건부 안전)

#### 정의
```
"Safe as used when [specific conditions]" 또는
"Safe as used in the present practices of use and concentration
 when formulated to be [condition]"

의미:
- 특정 조건 하에서만 안전
- 조건 미충족 시 안전성 보장되지 않음
- 제조사가 조건 충족 여부 확인 필요

조건 유형:
1. 농도 제한 (최대 사용 농도)
2. 용도 제한 (특정 제품 유형 제외)
3. 제형 조건 (pH, 순도, 입자 크기 등)
4. 경고 표시 요건
5. 흡입 노출 회피 조건
```

#### 농도 제한 예시

| 성분 (INCI) | 한글명 | 제한 농도 | 용도별 차이 | 평가 연도 |
|------------|--------|----------|------------|----------|
| Retinol | 레티놀 | 0.5% (Leave-on) | Body 0.05% | 2021 |
| Retinyl Palmitate | 레티닐팔미테이트 | 0.5% (Leave-on) | - | 2021 |
| Salicylic Acid | 살리실릭애씨드 | 2% (Leave-on), 3% (Rinse-off) | - | 2018 |
| Benzoyl Peroxide | 벤조일퍼옥사이드 | 2.5% (Leave-on) | - | 2010 |
| Lactic Acid | 락틱애씨드 | 10% (Leave-on) | pH 3.5 이상 | 1998 |
| Glycolic Acid | 글라이콜릭애씨드 | 10% (Leave-on) | pH 3.5 이상 | 1998 |
| Kojic Acid | 코직애씨드 | 1% | 감작성 주의 | 2010 |

#### 용도 제한 예시

| 성분 (INCI) | 제외 용도 | 이유 | 평가 연도 |
|------------|----------|------|----------|
| Talc | 분말 제품 (흡입) | 폐 침착 우려 | 2013 |
| Propylene Glycol | 립 제품 (고농도) | 섭취 노출 | 2019 |
| Butylated Hydroxytoluene | 립 제품 | 섭취 노출 | 2002 |

#### 제형 조건 예시

| 성분 (INCI) | 조건 | 상세 내용 | 평가 연도 |
|------------|------|----------|----------|
| Titanium Dioxide | 나노 입자 조건 | 코팅 처리, 피부 침투 방지 | 2016 |
| Zinc Oxide | 나노 입자 조건 | 피부 손상 부위 사용 금지 | 2017 |
| AHAs (Alpha Hydroxy Acids) | pH 3.5 이상 | 자극 최소화 | 1998 |
| Hydrogen Peroxide | 치아미백 제외 | 화장품 용도만 | 2009 |

### 3. Insufficient Data (데이터 부족)

#### 정의
```
"The available data are insufficient to support the safety of [ingredient]
 as used in cosmetics"

의미:
- 안전성 판단을 위한 정보가 부족
- 추가 데이터 제출 시 재평가 가능
- 사용 안전성이 확인되지 않음 (금지는 아님)

요구 데이터 유형:
- 피부 흡수 데이터
- 감작성 시험 데이터
- 유전독성 배터리
- 발달독성 데이터
- 현재 사용 농도 정보
- 불순물 프로파일
```

#### Insufficient Data 결론 사례

| 성분 (INCI) | 부족한 데이터 | 비고 |
|------------|--------------|------|
| Bakuchiol | 생식발달독성 | 레티놀 대체제로 관심 증가 |
| Cannabidiol (CBD) | 전반적 독성 데이터 | 규제 불확실성 |
| 일부 식물 추출물 | 조성 명확화, 독성 데이터 | 표준화 필요 |
| 신규 펩타이드류 | 피부 흡수, 전신 독성 | 사용 증가 추세 |

---

### 4. Unsafe (안전하지 않음)

#### 정의
```
"Unsafe for use in cosmetic products" 또는
"Unsafe for use in [specific application]"

의미:
- 현재 사용 조건에서 안전하지 않음
- 사용 금지 또는 대폭 제한 권고
- 매우 드문 결론 (CIR 역사상 소수)

Unsafe 결론 근거:
- 심각한 독성 확인
- 발암성/유전독성 우려
- 감작성/자극성 심각
- 안전한 사용 농도 설정 불가
```

#### Unsafe 결론 사례

| 성분 (INCI) | Unsafe 용도 | 이유 | 연도 |
|------------|------------|------|------|
| Coal Tar | Hair Dye (피부 접촉) | 발암성 우려 | 1985 |
| Lead Acetate | Hair Dye | 납 흡수, 독성 | 2007 |
| 4-ABP | 염모제 성분 | 방광암 연관성 | 1985 |

## 최근 평가 동향 (2020-2024)

### 2024년 주요 평가

```
- Cannabidiol (CBD): Insufficient data (데이터 부족)
- 신규 실리콘 유도체: 그룹 평가 진행 중
- 마이크로바이옴 관련 성분: 평가 준비 중
```

### 2023년 주요 평가

```
- Bakuchiol: 재평가 중 (데이터 보완 요청)
- Centella Asiatica 추출물: Safe as used
- Ceramide 복합체: Safe as used 확인
```

### 2022년 주요 평가

```
- Hyaluronic Acid: Safe as used (분자량별 재확인)
- Niacinamide: Safe as used (재평가 완료)
- Peptides (신규): 개별 평가 진행
```

## 결론별 대응 가이드

### Safe as Used 활용

```
1. 마케팅 클레임 근거로 활용 가능
2. 규제 대응 자료로 CIR 보고서 인용
3. 소비자 문의 대응 참고
4. 단, CIR 결론 = 법적 승인이 아님 주의
```

### Safe with Qualifications 대응

```
1. 모든 조건 상세 확인
2. 처방/제형이 조건 충족하는지 검증
3. 필요시 제품 라벨에 경고문 추가
4. 조건 미충족 시 사용 재검토
```

### Insufficient Data 대응

```
1. 요구 데이터 유형 확인
2. 원료사에 데이터 요청
3. 자체 시험 검토
4. CIR에 데이터 제출하여 재평가 요청 가능
5. 사용 시 자체 안전성 평가 필요
```

### Unsafe 대응

```
1. 해당 용도에서 사용 중단
2. 대체 성분 검토
3. 해당 용도가 제품과 관련 있는지 확인
4. 안전한 다른 용도 가능성 검토
```
