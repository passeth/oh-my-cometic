# 임상 근거 수준 평가 가이드

화장품 성분의 임상 근거를 평가하고 등급화하기 위한 표준 가이드라인

## 1. 근거 수준 등급 체계

### 주요 등급 정의

| Grade | 명칭 | 정의 | 근거 유형 |
|-------|------|------|----------|
| **A** | Strong Evidence | 강력한 임상 근거 | 다수의 RCT, 메타분석, 체계적 문헌고찰 |
| **B** | Moderate Evidence | 적절한 임상 근거 | 1-2개 RCT 또는 다수의 well-designed 임상시험 |
| **C** | Limited Evidence | 제한적이나 유망한 근거 | 소규모 임상시험, 파일럿 연구, 비대조 연구 |
| **D** | Preliminary Evidence | 기초 연구 수준 | In-vitro, Ex-vivo, 동물 실험만 존재 |
| **E** | Insufficient | 근거 부족 | 데이터 부족 또는 상반된 결과 |

### 등급별 상세 기준

#### Grade A: Strong Evidence (강력한 근거)

**필수 요건 (모두 충족)**:
- [ ] 3개 이상의 RCT 존재
- [ ] 총 피험자 수 > 200명
- [ ] 일관된 결과 (결과의 방향성 동일)
- [ ] 메타분석 또는 체계적 문헌고찰 존재

**추가 품질 지표**:
- 이중맹검 설계
- 위약 대조군 존재
- 명확한 1차 평가변수
- 적절한 통계 분석
- 이해충돌 최소

**권장 표현**:
```
"강력한 임상 근거가 있음"
"다수의 무작위대조시험에서 일관되게 효능이 입증됨"
"메타분석에서 유의한 효과가 확인됨"
```

---

#### Grade B: Moderate Evidence (적절한 근거)

**필수 요건 (하나 이상 충족)**:
- [ ] 1-2개의 RCT 존재 (n≥30 each)
- [ ] 3개 이상의 well-designed 임상시험 (RCT 아닐 수 있음)
- [ ] 총 피험자 수 > 100명

**추가 품질 지표**:
- 대조군 존재 (위약 또는 비처리)
- 명확한 평가 방법
- 객관적 측정 지표 포함
- 통계적 유의성 보고

**권장 표현**:
```
"적절한 임상 근거가 있음"
"임상시험에서 효능이 입증됨"
"대조군 대비 유의한 개선이 관찰됨"
```

---

#### Grade C: Limited Evidence (제한적 근거)

**필수 요건**:
- [ ] 최소 1개의 인체 임상시험 존재
- [ ] 피험자 수 ≥ 10명
- [ ] 정량적 결과 보고

**연구 유형**:
- 파일럿 연구
- 오픈라벨 연구
- 비교군 없는 단일군 연구
- 자기 대조 연구 (split-face)
- 관찰 연구

**권장 표현**:
```
"제한적이나 유망한 근거가 있음"
"초기 임상 연구에서 긍정적 결과 보고됨"
"추가 연구가 필요하나 잠재력이 있음"
```

---

#### Grade D: Preliminary Evidence (기초 연구 수준)

**근거 유형**:
- In-vitro 세포 실험
- Ex-vivo 피부 조직 실험
- In-vivo 동물 실험
- 기전 연구

**인체 연구 부재**

**권장 표현**:
```
"기초 연구 수준의 근거"
"세포/동물 실험에서 활성이 확인됨"
"인체 임상 연구가 필요함"
```

---

#### Grade E: Insufficient (근거 부족)

**해당 상황**:
- 관련 연구 없음
- 연구 결과 상반됨
- 심각한 방법론적 결함
- 이해충돌 심각

**권장 표현**:
```
"현재 충분한 근거가 없음"
"연구 결과가 일관되지 않음"
"추가 연구가 필요함"
```

## 2. 연구 품질 평가 체크리스트

### RCT 품질 평가 (Jadad Scale 기반)

| 항목 | 점수 | 기준 |
|-----|------|------|
| 무작위 배정 | 0-2 | 무작위화 언급 (+1), 적절한 방법 기술 (+1) |
| 이중맹검 | 0-2 | 맹검 언급 (+1), 적절한 방법 기술 (+1) |
| 탈락 보고 | 0-1 | 탈락자 수와 사유 기술 (+1) |
| **총점** | **0-5** | 3점 이상: 고품질 |

### 비무작위 연구 품질 평가

| 항목 | 점수 | 기준 |
|-----|------|------|
| 연구 설계 | 0-2 | 전향적 (+2), 후향적 (+1) |
| 대조군 | 0-2 | 적절한 대조군 (+2), 자기대조 (+1) |
| 평가 맹검 | 0-1 | 맹검 평가자 (+1) |
| 샘플 크기 | 0-2 | n≥50 (+2), n≥20 (+1) |
| 통계 분석 | 0-1 | 적절한 통계 방법 (+1) |
| 이해충돌 | 0-1 | 공개 및 최소 (+1) |
| **총점** | **0-9** | 6점 이상: 고품질 |

## 3. 효능별 평가 기준

### 미백/브라이트닝

**주요 평가 지표**:
| 지표 | 측정 방법 | 유의미한 변화 |
|-----|----------|--------------|
| ITA° (Individual Typology Angle) | 분광측색계 | ≥ 3° 증가 |
| Melanin Index | Mexameter | ≥ 10% 감소 |
| L* value | Chromameter | ≥ 2 증가 |
| 색소 면적 | Image Analysis | ≥ 20% 감소 |
| 주관적 평가 | VAS, 5-point scale | ≥ 1점 개선 |

**권장 연구 기간**: 8-12주

---

### 항노화/주름

**주요 평가 지표**:
| 지표 | 측정 방법 | 유의미한 변화 |
|-----|----------|--------------|
| 주름 깊이 | Profilometry | ≥ 10% 감소 |
| 주름 등급 | Crow's feet scale | ≥ 0.5 등급 개선 |
| 탄력도 | Cutometer (R2, R5) | ≥ 10% 증가 |
| 피부 두께 | Ultrasound | ≥ 5% 증가 |
| 콜라겐 밀도 | Confocal microscopy | 질적 개선 |

**권장 연구 기간**: 12-24주

---

### 보습

**주요 평가 지표**:
| 지표 | 측정 방법 | 유의미한 변화 |
|-----|----------|--------------|
| 피부 수분도 | Corneometer | ≥ 10% 증가 |
| TEWL | Tewameter | ≥ 10% 감소 |
| 피부 거칠기 | SELS | ≥ 10% 감소 |
| 주관적 건조감 | VAS | ≥ 2점 개선 |

**권장 연구 기간**: 4-8주

---

### 진정/항염

**주요 평가 지표**:
| 지표 | 측정 방법 | 유의미한 변화 |
|-----|----------|--------------|
| 홍반도 | Mexameter (Erythema Index) | ≥ 10% 감소 |
| a* value | Chromameter | ≥ 1 감소 |
| 자극 회복 시간 | SLS challenge | ≥ 20% 단축 |
| 가려움 VAS | 시각 아날로그 척도 | ≥ 30% 감소 |

**권장 연구 기간**: 2-4주

---

### 피지/모공

**주요 평가 지표**:
| 지표 | 측정 방법 | 유의미한 변화 |
|-----|----------|--------------|
| 피지 분비량 | Sebumeter | ≥ 15% 감소 |
| 모공 크기 | Image Analysis | ≥ 10% 감소 |
| 모공 수 | Visiometer | ≥ 10% 감소 |

**권장 연구 기간**: 4-8주

|-------|------|--------|---|----------|---------------|-------------|---------|

## 5. 등급별 권장 클레임 수준

### Grade A 성분

**허용 클레임**:
- "임상적으로 입증된"
- "과학적으로 검증된"
- "효과가 확인된"
- 구체적 수치 언급 가능 (연구 인용 시)

**예시**:
```
"임상시험에서 12주 후 주름 깊이가 평균 25% 감소함이 입증되었습니다."
```

---

### Grade B 성분

**허용 클레임**:
- "연구를 통해 효과가 확인된"
- "임상시험에서 개선이 관찰된"
- "피부 과학에 기반한"

**예시**:
```
"임상 연구에서 피부 톤 개선 효과가 확인되었습니다."
```

---

### Grade C 성분

**허용 클레임**:
- "~에 도움을 줄 수 있는"
- "~를 위해 연구된"
- "잠재적 효과가 있는"

**예시**:
```
"피부 보습에 도움을 줄 수 있습니다."
```

---

### Grade D 성분

**허용 클레임**:
- "~로 알려진"
- "전통적으로 ~에 사용된"
- 효능 클레임 제한적

**예시**:
```
"항산화 특성으로 알려진 성분입니다."
```

---

### Grade E 성분

**권장사항**:
- 효능 클레임 자제
- 성분 특성만 언급
- 추가 연구 필요 명시

## 6. 평가 시 주의사항

### 일반적 함정

1. **농도 불일치**: 연구 농도 vs 제품 농도 확인
2. **적용 방식 차이**: 연구 프로토콜 vs 실제 사용법
3. **피험자 특성**: 연구 대상 vs 목표 소비자
4. **평가 시점**: 연구 기간 vs 예상 사용 기간
5. **이해충돌**: 원료사 후원 연구 주의

### 품질 저하 요인

- 작은 샘플 크기 (n < 20)
- 짧은 연구 기간 (< 4주)
- 맹검 부재
- 탈락률 > 20%
- 이해충돌 미공개
- 통계 방법 불명확
- 부적절한 대조군

### 등급 상향 요인

- 다수의 독립적 연구
- 일관된 결과
- 객관적 측정 지표
- 장기간 추적
- 다양한 인종/연령 포함
- 용량-반응 관계 확인

### 등급 하향 요인

- 단일 연구만 존재
- 상반된 결과
- 주관적 평가 의존
- 연구 농도 >> 제품 농도
- 심각한 방법론적 결함
- 높은 이해충돌
