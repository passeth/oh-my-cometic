## 1. 비타민류 (Vitamins)

### Niacinamide (나이아신아마이드)

| 항목 | 정보 |
|-----|------|
| **INCI Name** | NIACINAMIDE |
| **CAS Number** | 98-92-0 |
| **화학식** | C₆H₆N₂O |
| **분자량** | 122.12 g/mol |
| **용해도** | 수용성 (1g/1mL water) |
| **최적 pH** | 5.0-7.0 |
| **안정성** | 매우 안정 (열, 빛, pH 내성) |
| **권장 농도** | 2-5% (일반), 10% (전문가용) |
| **EWG 등급** | 1 |

**주요 효능**:
- 피지 조절 및 모공 축소
- 미백 (멜라닌 전달 억제)
- 피부 장벽 강화 (세라마이드 합성 촉진)
- 항산화 및 항염
- 주름 개선 (콜라겐 합성 촉진)

**작용 메커니즘**:
- NAD+/NADH 전구체로서 세포 에너지 대사 증진
- 멜라노좀의 각질세포 전달 억제
- PPAR 활성화를 통한 지질 합성 촉진
- NF-κB 경로 억제로 항염 효과

**시너지 성분**:
- Zinc (피지 조절 강화)
- Hyaluronic Acid (보습 시너지)
- Retinol (자극 완화 및 효능 보완)

**비호환 성분**:
- Vitamin C (낮은 pH에서 flush 반응 가능)

**작용 메커니즘**:
- RAR/RXR 핵수용체 활성화
- 유전자 전사 조절 (콜라겐 합성 유전자 활성화)

- 각질세포 분화 조절

**시너지 성분**:
- Vitamin E (안정성 향상)
- Niacinamide (자극 완화)
- Peptides (항노화 시너지)

**비호환 성분**:
- AHA/BHA (과도한 자극)
- Benzoyl Peroxide (불활성화)
- Vitamin C (pH 불호환)

**주의사항**:
- 광감작 가능 (자외선차단제 필수)
- 임산부 사용 금지
- 점진적 농도 증가 권장

- 항염

**작용 메커니즘**:

- 낮은 pH 유지 (< 3.5)
- 무수 제형
- 에어리스 용기
- 항산화제 병용
