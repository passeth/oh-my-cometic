# Labeling Requirements by Country

Comprehensive guide to cosmetic product labeling requirements across major markets.

## General Labeling Principles

All markets require labels to be:
- **Legible**: Clear, readable text
- **Indelible**: Permanent, not easily removed
- **Visible**: Prominently displayed
- **Accurate**: Truthful information
- **Complete**: All required elements present

## Korea (MFDS) Labeling Requirements

### Required Label Elements

| Element | Requirement | Location |
|---------|-------------|----------|
| Product Name | Korean or English | Primary display |
| Product Category | Cosmetic type | Primary display |
| Manufacturer/Importer | Full name and address | Any panel |
| Responsible Person | Name and contact | Any panel |
| Batch/Lot Number | Manufacturing identifier | Any panel |
| Expiration Date | yyyy.mm.dd or Period After Opening | Primary/Secondary |
| Full Ingredient List | INCI or Korean names, descending order | Any panel |
| Net Contents | Weight/Volume in metric | Primary display |
| Directions for Use | How to use | Any panel |
| Cautions/Warnings | Safety warnings | Any panel |
| Country of Origin | Manufacturing country | Any panel |
| Price | Consumer price | Required for retail |

### Korean Language Requirements

| Requirement | Details |
|-------------|---------|
| Primary Language | Korean mandatory |
| INCI Names | Korean translation or INCI accepted |
| Foreign Products | Korean sticker label acceptable |
| Font Size | Minimum 6pt (some elements 5pt) |

### Korea-Specific Warnings

| Product Type | Required Warning |
|--------------|------------------|
| Hair Dye | Allergy warning, patch test instructions |
| Sunscreen | "Not suitable for children under 3" (if applicable) |
| AHA Products | "May increase sun sensitivity" |
| Retinol Products | Pregnancy warning (functional) |
| Products for Children | Age restrictions if applicable |

### Korea Functional Cosmetic Labels

Additional requirements for functional cosmetics:

| Element | Requirement |
|---------|-------------|
| "Functional Cosmetic" | Must be stated |
| Functional Ingredients | Listed with concentrations |
| Efficacy Claims | Only approved claims |
| MFDS Approval Number | Required on label |

## EU (EC 1223/2009) Labeling Requirements

### Mandatory Label Elements (Article 19)

| Element | Requirement | Notes |
|---------|-------------|-------|
| Responsible Person | Name and address | EU-based RP |
| Country of Origin | "Made in [Country]" | If outside EU |
| Nominal Content | Weight or volume | At time of packaging |
| Date of Minimum Durability | "Best before" or PAO symbol | See below |
| Precautions for Use | Safety warnings | As required |
| Batch Number | Traceability | Can be on outer packaging |
| Function of Product | Unless obvious | Helps consumer understand |
| List of Ingredients | Full INCI list | Descending order |

### Date Marking Requirements

| Durability | Requirement | Format |
|------------|-------------|--------|
| <30 months | "Best before end of" + date | MM/YYYY or symbol + date |
| >30 months | PAO symbol (open jar) | Number + M (e.g., 12M) |

**PAO Symbol (Period After Opening):**
```
  ___
 /   \
|  12M |  ← Months after opening
 \___/
   |
```

### EU Language Requirements

| Requirement | Details |
|-------------|---------|
| Member State Language | Language of country where sold |
| INCI Names | Standard INCI (Latin/English) |
| Multiple Languages | Acceptable on same label |
| Minimum Font Size | Not specified but "legible" required |

### EU Warning Statements (Annex III)

| Ingredient/Product | Warning Required |
|--------------------|------------------|
| Fluoride toothpaste | "Children of 6 years and younger: use a pea-sized amount" |
| Hydrogen peroxide (hair) | "Contains hydrogen peroxide. Avoid contact with eyes." |
| Persulfates (hair) | "Can cause allergic reactions. Do not use on irritated scalp." |
| Retinol (>0.05%) | "Contains retinol. Not for use during pregnancy or nursing." |
| Formaldehyde releasers | "Contains formaldehyde" (if >0.05%) |
| Fragrance allergens | List individual allergens if >0.001% (leave-on) |

### EU Ingredient Listing Rules

| Rule | Details |
|------|---------|
| Order | Descending concentration |
| Threshold | Ingredients >1% in order; <1% any order |
| Colorants | CI numbers, any order after other ingredients |
| Fragrances | "Parfum" or "Aroma" + listed allergens |
| Nanomaterials | [nano] after ingredient name |

### EU Label Example

```
[Outer Packaging]
----------------------------------------
BRIGHTENING SERUM
Serum Eclat

Ingredients: Aqua, Niacinamide, Glycerin,
Butylene Glycol, Phenoxyethanol, Parfum,
Limonene, Linalool, CI 77891 [nano]

50 mL  e                    [Nominal content]
[PAO symbol: 12M]           [Period After Opening]

ABC Cosmetics Ltd
123 High Street, London EC1 UK  [Responsible Person]

Made in France              [Country of Origin]

Batch: EU2025001           [Batch Number]

Apply daily to face after cleansing.
Avoid contact with eyes.   [Directions & Cautions]

Contains retinol. Not recommended during
pregnancy.                 [Specific Warning]
----------------------------------------
```

## USA (FDA) Labeling Requirements

### FD&C Act and Fair Packaging and Labeling Act

| Element | Requirement | Location |
|---------|-------------|----------|
| Identity Statement | Product name | Principal Display Panel (PDP) |
| Net Quantity | Weight, volume, or count | PDP, lower 30% |
| Distributor Statement | Name and address | Information panel |
| Ingredient Declaration | Descending order | Information panel |
| Warning Statements | If applicable | Information panel |
| Directions for Use | If needed for safe use | Information panel |

### US Principal Display Panel (PDP)

| Requirement | Details |
|-------------|---------|
| Product Identity | Common name or descriptive |
| Net Contents | Lower 30% of PDP |
| Font Size | Based on package size |

**Net Contents Font Size:**

| Package Area | Minimum Type Size |
|--------------|-------------------|
| ≤5 sq. in. | 1/16 inch |
| 5-25 sq. in. | 1/8 inch |
| 25-100 sq. in. | 3/16 inch |
| 100-400 sq. in. | 1/4 inch |
| >400 sq. in. | 1/2 inch |

### US Ingredient Declaration Rules

| Rule | Details |
|------|---------|
| Order | Descending predominance |
| Names | Common/usual name or CTFA/INCI |
| Fragrance | "Fragrance" or "Parfum" |
| Flavor | "Flavor" |
| Colorants | Common name (e.g., "FD&C Red No. 40") |
| Trade Secret | Petition FDA for exemption |

### US Warning Requirements

| Product Type | Required Warning |
|--------------|------------------|
| Coal tar hair dyes | Caution statement + patch test |
| Products containing acids | "Avoid contact with eyes" |
| Aerosol products | Flammability warnings if applicable |
| Products with potential allergens | Allergen warning recommended |
| Feminine deodorant sprays | Specific warnings required |
| Self-pressurized containers | Warning against incineration |

### US OTC Drug Labeling (Cosmetics with Drug Claims)

If product makes drug claims, additional requirements apply:

| Element | Requirement |
|---------|-------------|
| Drug Facts Panel | Standardized format |
| Active Ingredients | With purpose |
| Uses | Indications |
| Warnings | Specific drug warnings |
| Directions | Dosage and administration |
| Inactive Ingredients | Listed after Drug Facts |

### US Label Example

```
[Principal Display Panel]
----------------------------------------
RADIANCE SERUM
Vitamin C Brightening Formula

Net Wt 1.7 OZ (50 mL)
----------------------------------------

[Information Panel]
INGREDIENTS: Water (Aqua), Ascorbic Acid,
Glycerin, Butylene Glycol, Niacinamide,
Phenoxyethanol, Fragrance (Parfum),
Yellow 5 (CI 19140), Red 40 (CI 16035)

Distributed by:
XYZ Beauty Inc.
123 Main Street
New York, NY 10001
www.xyzbeauty.com

DIRECTIONS: Apply to clean face morning
and evening. Follow with moisturizer.

CAUTION: For external use only. Avoid
contact with eyes. Discontinue use if
irritation occurs.
----------------------------------------
```

## China (NMPA) Labeling Requirements

### Cosmetic Label Management Measures (2021)

| Element | Requirement | Notes |
|---------|-------------|-------|
| Product Name | Chinese + original name | Chinese prominent |
| Category | Cosmetic type | Chinese |
| Manufacturer Name | Chinese + original | Full legal name |
| Registrant/Filer | NMPA registered entity | Must be in China |
| Production License | Chinese license number | Required |
| Product Standard | GB or enterprise standard | Reference number |
| Full Ingredients | Chinese + INCI | Descending order |
| Net Contents | Metric units | Chinese numerals |
| Shelf Life | Production + expiry date | Both required |
| Storage Conditions | If special requirements | Chinese |
| Safety Warnings | Chinese | Product-specific |
| Country of Origin | Manufacturing country | Chinese |

### Chinese Language Requirements

| Requirement | Details |
|-------------|---------|
| Primary Language | Simplified Chinese mandatory |
| Font Size | ≥1.8mm for mandatory info |
| Foreign Languages | Allowed but Chinese must be ≥ foreign |
| Brand Names | Can keep original script |
| INCI Names | Chinese name OR pinyin + INCI |

### China-Specific Label Elements

| Element | Requirement |
|---------|-------------|
| Import Filing Number | Required for imported products |
| Domestic Agent | Chinese company contact |
| "进口" (Imported) | If imported product |
| Special Cosmetic Number | If registered as special |

### China Ingredient Naming

| Language | Accepted |
|----------|----------|
| Chinese Name | From IECIC (International Nomenclature of Cosmetic Ingredients - Chinese) |
| INCI | Accepted alongside Chinese |
| Pinyin | Phonetic transliteration acceptable |

### China Warning Requirements

| Product Type | Warning |
|--------------|---------|
| Hair Dye | Allergy warning, 48-hour patch test |
| Products for Children | "儿童必须在成人监护下使用" |
| Sunscreen | SPF and PA claims with testing proof |
| Whitening Products | "本产品为特殊用途化妆品" |

## Japan (PMDA/MHLW) Labeling Requirements

### Pharmaceutical Affairs Law Requirements

| Element | Requirement | Notes |
|---------|-------------|-------|
| Product Name | Japanese + English allowed | Japanese prominent |
| Category | 化粧品 (Cosmetic) or 医薬部外品 (Quasi-drug) | Required |
| Manufacturer/Seller | Name and address | License holder |
| Manufacturing Location | If outside Japan | Country name |
| Content Volume | Metric units | Japanese numerals |
| Lot Number | Traceability | Any location |
| Ingredients | INCI or Japanese | All ingredients |
| Precautions | Safety warnings | Japanese |

### Japan Language Requirements

| Requirement | Details |
|-------------|---------|
| Primary Language | Japanese |
| Ingredient Names | INCI or JCIC (Japan Cosmetic Ingredient Codex) |
| Foreign Products | Japanese label required |
| Supplementary Languages | Allowed but Japanese primary |

### Japan Quasi-Drug Additional Requirements

| Element | Requirement |
|---------|-------------|
| "医薬部外品" | Must be prominently displayed |
| Active Ingredients | Listed separately with amounts |
| Approved Claims | Only approved efficacy claims |
| Approval Number | Required |

### Japan Ingredient Labeling

| Rule | Details |
|------|---------|
| Order | Descending by weight |
| Names | JCIC names or INCI |
| "All Ingredients Listed" | Required statement |
| Tar Colors | Specific Japanese names |

### Japan Warning Requirements

| Product Type | Warning |
|--------------|---------|
| Hair Dye | Detailed allergy warning |
| Peroxide Products | Bleaching/damage warning |
| AHA Products | Sun sensitivity warning |
| Eye Products | "For external use around eyes" |

## Claim Restrictions on Labels

### Prohibited Claim Language

| Claim Type | All Regions |
|------------|-------------|
| Medical claims | "Treats", "Cures", "Prevents disease" |
| Drug claims | "Medicine", "Drug", "Therapeutic" |
| Absolute claims | "100% effective", "Guaranteed results" |
| Misleading claims | Exaggerated efficacy |

### Allowed Claim Language

| Claim Type | Generally Allowed |
|------------|-------------------|
| Cosmetic benefits | "Moisturizes", "Softens", "Smooths" |
| Appearance | "Improves appearance of..." |
| Sensory | "Feels softer", "Looks brighter" |
| Maintenance | "Helps maintain...", "Supports..." |

### Region-Specific Claim Notes

| Region | Notes |
|--------|-------|
| EU | All claims must be substantiated |
| USA | Claims determine cosmetic vs. drug |
| China | Claims must match registration |
| Japan | Quasi-drug claims need approval |
| Korea | Functional claims need registration |

| Region | Minimum Size | Notes |
|--------|--------------|-------|

| Region | Size Threshold | Exemptions |
|--------|----------------|------------|

## Compliance Checklist by Region

### Universal Checklist

- [ ] Product name clearly displayed
- [ ] Net contents in metric units
- [ ] Full ingredient list in descending order
- [ ] Manufacturer/importer information
- [ ] Country of origin
- [ ] Batch/lot number
- [ ] Date marking (expiry or PAO)
- [ ] Required warnings present
- [ ] Directions for use (if needed)
- [ ] Appropriate language

### Region-Specific Additions

**Korea:**
- [ ] Korean language primary
- [ ] Functional cosmetic registration number (if applicable)
- [ ] Price displayed

**EU:**
- [ ] Responsible Person in EU
- [ ] PAO symbol or best before date
- [ ] Fragrance allergens listed (if above threshold)
- [ ] [nano] designation where required

**USA:**
- [ ] Net quantity in lower 30% of PDP
- [ ] Distributor statement
- [ ] Color additive names (FD&C/D&C format)

**China:**
- [ ] Chinese language primary
- [ ] Import filing number
- [ ] Production AND expiry dates
- [ ] Chinese domestic agent

**Japan:**
- [ ] Japanese language primary
- [ ] Category designation (化粧品/医薬部外品)
- [ ] Manufacturing license holder

**ASEAN:**
- [ ] Local language requirements met
- [ ] Country-specific registration numbers
- [ ] Local responsible party
