# Efficacy Test Standards (효능시험 표준)

## Overview

Efficacy tests demonstrate the functional benefits claimed for cosmetic products. In Korea, these tests are mandatory for functional cosmetics (기능성화장품) approval and must be conducted by designated testing institutions following standardized protocols.

---

## Whitening Efficacy Test (미백 효능시험)

### Regulatory Requirements

**MFDS Approved Claims:**
- "피부 미백에 도움을 줌" (Helps skin whitening)
- "기미·주근깨 등의 색소침착 완화에 도움을 줌"

**Required Documentation:**
- Human clinical trial data (인체적용시험)
- Minimum 8 weeks duration
- Comparison with control (vehicle or positive control)

### Study Design

**Subjects:**
- Number: 20-30 per group (40-60 total for controlled study)
- Age: 20-60 years
- Skin Type: Fitzpatrick III-IV (Asian skin)
- Condition: Visible hyperpigmentation (melasma, freckles, sun spots)

**Inclusion Criteria:**
- Korean women with stable hyperpigmentation
- No active skin diseases
- Willing to avoid sun exposure during study
- No whitening products for 2 weeks prior

**Exclusion Criteria:**
- Pregnancy or lactation
- History of skin cancer
- Active dermatological conditions
- Use of systemic retinoids within 6 months
- Laser or chemical peel within 3 months

**Duration:** 8-12 weeks (12 weeks preferred)

**Application:**
- Twice daily (morning and evening)
- After cleansing, before moisturizer
- Specified amount (typically 2 finger-tip units for face)

### Measurement Methods

#### 1. Mexameter MX18 (Melanin Index)

**Principle:** Narrow-band spectrophotometry measuring melanin absorption

**Procedure:**
1. Calibrate device before each session
2. Subject acclimates 15-30 min in controlled environment
3. Measure target sites (3 readings per site)
4. Record ambient conditions (temperature, humidity)

**Sites:**
- Cheek (malar area) - bilateral
- Forehead
- Hyperpigmented lesion (if specific)

**Data Recording:**
- Melanin Index (MI): Arbitrary units (0-999)
- Baseline and Week 4, 8, 12

**Expected Outcomes:**
| Change | Interpretation |
|--------|----------------|
| >5% decrease | Mild improvement |
| >10% decrease | Moderate improvement |
| >15% decrease | Significant improvement |

#### 2. Colorimeter / Spectrophotometer (L*a*b*)

**Principle:** CIE L*a*b* color space measurement

**Parameters:**
- **L*** (Lightness): 0 = black, 100 = white
- **a*** (Red-Green): positive = red, negative = green
- **b*** (Yellow-Blue): positive = yellow, negative = blue

**Procedure:**
1. Use standard D65 illuminant
2. 10° standard observer
3. Calibrate with white tile
4. 3 measurements per site

**Analysis:**
```
ΔL* = L*(Week X) - L*(Baseline)
Positive ΔL* indicates lightening
```

#### 3. ITA° (Individual Typology Angle)

**Calculation:**
```
ITA° = [arctan((L* - 50) / b*)] × (180/π)
```

**Skin Classification:**
| ITA° | Classification |
|------|----------------|
| >55° | Very light |
| 41-55° | Light |
| 28-41° | Intermediate |
| 10-28° | Tan |
| <10° | Brown/Dark |

#### 4. Image Analysis

**Photography Standards:**
- Standardized lighting (cross-polarized)
- Fixed camera position (VISIA or equivalent)
- Same background and positioning
- Trained photographer

**Analysis Software:**
- UV spots quantification
- Brown spots quantification
- Evenness assessment

#### 5. Visual Assessment (Expert Grading)

**MASI Score (Melasma Area and Severity Index):**

```
MASI = 0.3A(D+H)forehead + 0.3A(D+H)malar_R + 0.3A(D+H)malar_L + 0.1A(D+H)chin
```

Where:
- A = Area (0-6 scale)
- D = Darkness (0-4 scale)
- H = Homogeneity (0-4 scale)

**Area Score:**
| Score | % Involvement |
|-------|---------------|
| 0 | No involvement |
| 1 | <10% |
| 2 | 10-29% |
| 3 | 30-49% |
| 4 | 50-69% |
| 5 | 70-89% |
| 6 | 90-100% |

## Anti-Wrinkle Efficacy Test (주름개선 효능시험)

### Regulatory Requirements

**MFDS Approved Claims:**
- "피부 주름 개선에 도움을 줌"
- "피부 탄력 개선에 도움을 줌"

### Study Design

**Subjects:**
- Number: 20-30 per group
- Age: 35-65 years
- Condition: Visible crow's feet wrinkles (Grade II-IV)

**Test Sites:**
- Crow's feet (lateral canthal area) - primary
- Forehead wrinkles - secondary
- Nasolabial folds - optional

**Duration:** 8-12 weeks

### Measurement Methods

#### 1. Skin Replica Analysis

**Procedure:**
1. Clean and dry test site
2. Apply silicone material (Silflo or equivalent)
3. Wait for polymerization (3-5 minutes)
4. Remove and analyze with profilometry

**Parameters:**
| Parameter | Description | Unit |
|-----------|-------------|------|
| Ra | Arithmetic mean roughness | μm |
| Rz | Maximum roughness | μm |
| Rt | Total roughness | μm |
| Rmax | Maximum peak-to-valley | μm |

**Expected Outcomes:**
| Ra Change | Interpretation |
|-----------|----------------|
| >5% decrease | Mild improvement |
| >10% decrease | Moderate improvement |
| >15% decrease | Significant improvement |

#### 2. 3D Optical Profilometry (PRIMOS)

**Principle:** Fringe projection for 3D surface reconstruction

**Parameters:**
- Surface roughness (Ra, Rz)
- Wrinkle depth
- Wrinkle volume
- Wrinkle count

**Advantages:**
- Non-invasive
- High resolution
- Quantitative 3D data
- Software analysis

#### 3. Visioscan VC98

**Principle:** UV-A light illumination with digital image analysis

**Parameters (SELS - Surface Evaluation of Living Skin):**
| Parameter | Meaning |
|-----------|---------|
| SEr | Roughness |
| SEsc | Scaliness |
| SEsm | Smoothness |
| SEw | Wrinkles |

#### 4. Cutometer MPA580 (Skin Elasticity)

**Principle:** Suction-elongation method measuring mechanical properties

**Parameters:**
| Parameter | Formula | Meaning | Normal Range |
|-----------|---------|---------|--------------|
| R0 (Uf) | - | Maximum deformation | - |
| R1 (Uf-Ua) | - | Ability to return | - |
| R2 | Ua/Uf | Gross elasticity | 0.8-1.0 |
| R5 | Ur/Ue | Net elasticity | 0.8-1.0 |
| R6 | Uv/Ue | Viscoelastic ratio | - |
| R7 | Ur/Uf | Biological elasticity | 0.5-0.7 |

**Measurement Settings:**
- Suction pressure: 450 mbar
- Suction time: 2 seconds
- Relaxation time: 2 seconds
- Number of cycles: 3
- Probe aperture: 2 mm

**Interpretation:**
- R2 closer to 1 = more elastic
- R7 closer to 1 = better elastic recovery
- Increase in R2, R5, R7 = improvement

#### 5. Visual Wrinkle Grading

**Fitzpatrick Wrinkle Classification:**
| Class | Score | Description | Depth |
|-------|-------|-------------|-------|
| I | 1-3 | Fine wrinkles | Minimal |
| II | 4-6 | Moderate | Moderate |
| III | 7-9 | Deep | Deep furrowing |

**Crow's Feet Grading Scale:**
| Grade | Description |
|-------|-------------|
| 0 | No wrinkles |
| 1 | Very shallow, barely visible |
| 2 | Shallow, fine lines |
| 3 | Moderate depth, visible at rest |
| 4 | Deep wrinkles, visible prominent |
| 5 | Very deep, severe |

## Moisturizing Efficacy Test (보습 효능시험)

### Study Types

**1. Acute Hydration Test (Single Application)**
- Duration: 8 hours
- Measurements: Baseline, 1h, 2h, 4h, 6h, 8h

**2. Chronic Hydration Test (Multiple Applications)**
- Duration: 2-4 weeks
- Measurements: Baseline, Week 1, 2, 3, 4

### Measurement Methods

#### 1. Corneometer CM825

**Principle:** Electrical capacitance measurement of stratum corneum hydration

**Procedure:**
1. Subject acclimates 30 minutes (20-22°C, 40-50% RH)
2. Clean forearm with dry tissue
3. Apply probe perpendicular to skin
4. 3 measurements per site

**Units:** Arbitrary Units (AU), range 0-130

**Classification:**
| Value | Skin Condition |
|-------|---------------|
| <30 AU | Very dry |
| 30-45 AU | Dry |
| 45-60 AU | Normal |
| >60 AU | Well-hydrated |

#### 2. Tewameter TM300 (TEWL)

**Principle:** Open chamber evaporimetry measuring water loss

**Procedure:**
1. Probe stabilization (10-15 seconds)
2. Measurement duration (30 seconds)
3. Record stable reading

**Units:** g/h/m² (grams per hour per square meter)

**Normal Ranges (Forearm):**
| Value | Condition |
|-------|-----------|
| <10 g/h/m² | Normal barrier |
| 10-15 g/h/m² | Mild impairment |
| 15-25 g/h/m² | Moderate impairment |
| >25 g/h/m² | Severe barrier damage |

**Interpretation:**
- Decreased TEWL = improved barrier function
- Acute moisturizers may increase TEWL initially

#### 3. Skin Hydration Curve (Acute Test)

**Protocol:**
1. Baseline measurement
2. Apply product (2 mg/cm²)
3. Measure at 1h, 2h, 4h, 6h, 8h
4. Calculate AUC

**AUC Calculation:**
```
AUC = Σ[(Ti+1 - Ti) × (Hi+1 + Hi) / 2]

Where:
- Ti = Time point
- Hi = Hydration value at time Ti
```

**Comparison:**
- AUC(test) vs. AUC(control)
- % increase from baseline at each time point

## Statistical Analysis

### Parametric Tests

**Paired t-test (Pre-Post Comparison):**
```
t = (X̄d - 0) / (Sd / √n)

Where:
- X̄d = mean of differences
- Sd = standard deviation of differences
- n = sample size
```

**Independent t-test (Group Comparison):**
```
t = (X̄1 - X̄2) / √(Sp² × (1/n1 + 1/n2))

Where:
- Sp² = pooled variance
```

### Non-Parametric Tests

**Wilcoxon Signed-Rank Test:**
- For paired, non-normal data
- Compares baseline to post-treatment

**Mann-Whitney U Test:**
- For independent groups
- Non-normal distributions

### Repeated Measures ANOVA

**For Multiple Time Points:**
- Within-subject factor: Time
- Between-subject factor: Treatment group
- Interaction: Time × Treatment

**Post-hoc Tests:**
- Bonferroni correction
- Tukey HSD
- Dunnett's test (vs. control)

### Effect Size Calculation

**Cohen's d:**
```
d = (Mean_post - Mean_baseline) / SD_pooled
```

**Interpretation:**
| d | Effect Size |
|---|-------------|
| 0.2 | Small |
| 0.5 | Medium |
| 0.8 | Large |

### Percentage Improvement

```
% Improvement = [(Baseline - Post) / Baseline] × 100
```

**For Decreasing Parameters (Melanin, TEWL, Roughness):**
- Positive % = improvement

**For Increasing Parameters (L*, Hydration, Elasticity):**
```
% Improvement = [(Post - Baseline) / Baseline] × 100
```

## Report Writing Guidelines

### Essential Sections

1. **Study Objective**
   - Primary objective (efficacy endpoint)
   - Secondary objectives

2. **Study Design**
   - Type (parallel, crossover, single-arm)
   - Duration
   - Control (vehicle, active, untreated)
   - Blinding (single, double, open-label)

3. **Subject Selection**
   - Inclusion/exclusion criteria
   - Recruitment method
   - Informed consent

4. **Test Materials**
   - Product description
   - Batch number
   - Storage conditions
   - Application instructions

5. **Efficacy Assessments**
   - Instruments used
   - Measurement schedule
   - Operator training

6. **Statistical Methods**
   - Pre-specified analysis plan
   - Sample size justification
   - Handling of missing data

7. **Results**
   - Subject disposition
   - Demographics
   - Efficacy data (tables, figures)
   - Statistical analysis

8. **Safety**
   - Adverse events
   - Tolerability assessments
   - Product-related reactions

9. **Conclusion**
   - Summary of findings
   - Efficacy determination
   - Limitations

## Acceptance Criteria

### MFDS Functional Cosmetics Standards

**Whitening:**
- Statistically significant decrease in Melanin Index
- OR significant increase in L* value
- p < 0.05 vs. baseline or control

**Anti-Wrinkle:**
- Statistically significant decrease in skin roughness (Ra)
- OR significant increase in elasticity (R2)
- p < 0.05 vs. baseline or control

**Moisturizing:**
- Statistically significant increase in skin hydration
- Duration of effect demonstrated
- p < 0.05 vs. baseline or control

### Minimum Requirements Summary

| Efficacy Type | Subjects | Duration | Primary Endpoint |
|---------------|----------|----------|------------------|
| Whitening | ≥20 | ≥8 weeks | Melanin Index or L* |
| Anti-Wrinkle | ≥20 | ≥8 weeks | Roughness or Elasticity |
| Moisturizing | ≥20 | Acute or ≥4 weeks | Hydration (Corneometer) |
