## 10. 밀도 측정 및 추정

### 10.1 밀도 측정법

```
1. 비중병법 (Pycnometer)
   - 가장 정확한 방법
   - 알려진 부피의 용기 질량 측정

2. 밀도계 (Density Meter)
   - 진동관 원리
   - 자동화된 측정

3. 비중계 (Hydrometer)
   - 부력 원리
   - 액체 밀도 간편 측정
```

### 10.2 밀도 추정 (혼합물)

```python
# 이상 혼합 가정 (대략적 추정)
d_mix = Σ(w_i × d_i)

# w_i: 각 성분의 질량 분율
# d_i: 각 성분의 밀도

# 예시: 글리세린 30% + 정제수 70%
d_mix ≈ 0.30 × 1.261 + 0.70 × 0.997
      ≈ 0.378 + 0.698
      ≈ 1.076 g/mL

# 실측값: 1.073 g/mL (오차 약 0.3%)
```

### 10.3 온도 보정

```python
# 온도에 따른 밀도 변화 (선형 근사)
d_T2 = d_T1 × [1 - β × (T2 - T1)]

# β: 부피 팽창 계수
# 물: β ≈ 0.00021 /°C
# 오일: β ≈ 0.0007-0.0008 /°C
# 에탄올: β ≈ 0.0011 /°C
```
