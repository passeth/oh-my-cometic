## 1️⃣ 신규 제품 카피 기획 체크리스트 (8주 계획)

### 1주차: 조사 & 분석

```
□ 제품 스펙 수집
  □ 주요 성분 리스트업 (최대 5개)
  □ 각 성분의 효능 정확히 파악
  □ 임상 데이터 수집 (있으면)
  □ 최종 포뮬러 확정 날짜 확인

□ 타겟 정의
  □ 주 타겟 연령대 (예: 30-40대 여성)
  □ 피부 타입 (예: 복합성, 민감성)
  □ 주요 피부 고민 TOP 3
  □ 라이프스타일 특성
  □ 구매력 수준
  □ 주 활동 채널

□ 경쟁사 분석
  제품명: ________________
  가격: ________________
  태그라인: ________________
  주요 카피: ________________
  강점: ________________
  약점: ________________

  → 5개 경쟁사 분석 완료

□ 차별화 포인트 도출
  우리만의 장점:
  1. ________________
  2. ________________
  3. ________________
```

## 2️⃣ SNS별 카피 작성 템플릿

### Instagram Feed 포스트 (100-200자)

#### 템플릿 1: 문제 공감형
```
[후킹 문장]
"당신도 이런 경험 있나요?"

[구체적 상황]
"아침에 일어났을 때:
_______

저녁 8시쯤:
_______

이게 내 피부라고 생각했어요."

[해결책 제시]
"지금은 다릅니다.
[제품명] 하나로:
_______"

[감정적 마무리]
"거울 보는 게 즐거워졌어요.
당신도 이 경험이 필요할 것 같아요."

[CTA]
"프로필의 링크에서 더 알아보기"

[해시태그]
#피부고민 #해결 #[브랜드명]
```

#### 템플릿 2: 효능 강조형
```
[임팩트 있는 수치]
"4주 만에 톤이 3단계 밝아졌어요.
99명 중 94명이 같은 경험을 했습니다."

[이유 설명]
"왜 가능했을까?
_______"

[사용자 후기]
"'거울 속 나를 몰라봤어요'
'딸이 '엄마 피부 좋아졌다'고 했어요'
'5년 만에 자신감이 돌아왔습니다'"

[호출]
"당신도 이 변화를 경험할 수 있습니다."

[해시태그]
#임상증명 #효과입증 #[카테고리]
```

#### 템플릿 3: 차별성 강조형
```
[일반적 제품]
"일반 토너: 물 + 알코올"

[우리 제품]
"우리 토너:
물 + 발효액 + 미네랄 + [핵심기술]"

[결과 비교]
"결과:
- 일반: 당김, 자극, 단기 효과
- 우리: 생생함, 안심, 장기 효과"

[증명]
"증거: 임상 결과 [수치]"

[호출]
"이 차이를 느껴보세요."

[해시택그]
#차별성 #과학적증명 #최고효능
```

### TikTok & Reels (15-30초 스크립트)

#### 템플릿 1: 즉각 변화형

```
[0-3초] 후킹
"여드름만 없어도 인생이 달라진다는 거, 알아?"

[3-10초] Before 공감
"나도 그랬어.
매일 거울 보고 한숨.
화장도 커버 안 되고..."

[10-20초] After 변화
"이 제품 4주 썼어.
진짜 사라졌어.
거울에서 눈을 마주칠 수 있어."

[20-25초] 증거
"100명 임상, 94명이 같은 경험"

[25-30초] CTA
"링크에서 리뷰 봐"
```

#### 템플릿 2: 활용 팁형

```
[0-2초] 후킹
"이 에센스, 10가지 사용법 있는 거 알아?"

[2-8초] 팁 1
"밤샘 후: 얼굴 진정용"

[8-14초] 팁 2
"새 제품 후: 피부 안정용"

[14-20초] 팁 3

[20-28초] 결론

[28-30초] CTA
"나머지 7가지는 댓글에 ⬇️"

#### 템플릿 3: 비포-애프터형

```

"2주 전 vs 지금"

[사진/영상]

[DAY 3 / DAY 7 / DAY 14 사진]
내레이션: "근데 진짜 변한다..."

[14-24초] 지금 모습
[현재 사진]

"구매 링크 생물 정보에 있어"

## 📌 자주 묻는 질문 (FAQ)

### Q1. 감성 카피와 과학 카피, 어느 것이 더 잘 팔릴까요?

**A.** 제품과 타겟에 따라 다릅니다.

- **감성 카피가 효과적**: 럭셔리, 생활용품, 신규 브랜드
- **과학 카피가 효과적**: 의약외품, 기능성, 경쟁 심화 카테고리
- **최고의 전략**: 감정 + 과학의 결합

예: "피부과 의사가 추천하는 성분(과학)으로 거울을 보고 싶은 자신감(감정)을 되찾으세요"

---

### Q2. 카피가 너무 길어요. 어디를 줄어야 할까요?

**A.** 우선순위 순으로 줄이세요.

1. **꼭 필요**: 문제(고객 고민) + 해결책(제품) + 행동(CTA)
2. **있으면 좋음**: 이유(왜 우리 제품인가) + 증거(임상)
3. **선택사항**: 스토리, 감정 표현, 부가 정보

---

### Q3. 해시태그는 정말 필요할까요?

**A.** 채널에 따라 다릅니다.

- **Instagram**: 필수 (8-12개)
- **TikTok**: 2-5개 정도
- **Email**: 불필요
- **웹사이트**: 불필요

핵심: 많은 해시태그보다 **관련성 높은 해시태그**가 중요합니다.

---

### Q4. 카피를 얼마나 자주 바꿔야 하나요?

**A.** 성과에 따라 다릅니다.

- **태그라인**: 3-5년 유지 (변경 거의 없음)
- **SNS 포스트**: 주 2-3회 변경
- **캠페인 슬로건**: 월 1-2회 변경
- **톤앤매너**: 최소 1년 유지 후 평가
