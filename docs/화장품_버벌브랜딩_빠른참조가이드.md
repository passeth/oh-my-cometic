## 🎯 핵심 5가지 원칙

```
1. 진정성 (Authenticity)
   ❌ "가장 좋은 제품"
   ✅ "100명 중 89명이 효과 경험"

2. 특이성 (Distinctiveness)
   ❌ "효과 좋습니다"
   ✅ "클렌징만으로도 톤이 3단계 밝아진다"

3. 감정성 (Emotionality)
   ❌ "pH 5.5 약산성"
   ✅ "거울을 보고 싶다는 생각이 든다"

4. 선택지 제시 (Choice Architecture)
   ❌ "좋은 제품입니다"
   ✅ "당신은 문제 해결형? 예방형?"

5. 일관성 (Consistency)
   ❌ SNS는 친근, 광고는 공식적
   ✅ 모든 채널 동일한 톤앤매너
```

## 📝 카피 구조 공식

### 짧은 버전 (15초)
```
[0-3초] 후킹: "이 파운데이션 때문에 인생이 바뀔 뻔했어"
[3-12초] 스토리: "1주일 쓰니까 피부톤이 달라져"
[12-15초] CTA: "링크에서 리뷰 봐"
```

### 중간 버전 (200자)
```
[오프닝] 문제 제기: "당신의 피부 고민 어디서 시작됐나요?"
[증거] 과학 기반: "임상 테스트 결과..."
[경험] 사용자 후기: "거울을 보고 싶어졌어요"
[행동] CTA: "지금 구매하면 20% 할인"
```

### 긴 버전 (500자)
```
[오프닝(20%)] 상황 설정: 구체적이고 감정적
[증거(30%)] 과학 기반: 성분, 임상, 수치
[경험(25%)] 사용자 이야기: Before/After
[결론(25%)] 감정 마무리: 희망 또는 자신감
```

## 📊 2025년 카피라이팅 트렌드

```
✅ 감성 + 과학의 결합
   "임상 입증된 성분으로 자신감을 되찾다"

   "100명 중 94명 / '거울을 봤어요'"

   "이 에센스 때문에..."

✅ 사용자 참여 유도
   "당신의 피부 고민은? 댓글 달아줘"

✅ 해시태그 챌린지 활용
   "#[브랜드]_피부가행복하다"

❌ 과장광고
❌ 일방적 홍보
❌ 긴 설명
❌ 의약품처럼 보이는 표현
```

## 🔥 즉시 적용 가능한 템플릿

### 문제-해결형 (가장 많이 쓰임)
```
"당신도 이런 경험 있나요? [구체적 상황]
저도 그랬어요. [감정]
그런데 [제품]을 써봤어요.
지금은 [변화된 모습]입니다.
당신도 할 수 있어요."
```

### 차별성 강조형
```
"일반 제품: [특징A]
우리 제품: [특징B]

결과: 일반은 [결과A]
결과: 우리는 [결과B]

증거: [임상 수치]"
```

### 감정-경험형
```
"[감정 표현]
→ [구체적 상황]
→ [제품 경험]
→ [변화]
→ [감정 마무리]"
```

## ✅ 카피 최종 체크리스트 (게시 전)

```
□ 사실 검증: 과장 표현 없나?
□ 톤앤매너: 브랜드 스타일 맞나?
□ 타겟: 내 타겟이 이해할 수 있나?
□ 임팩트: 후킹이 강한가? (SNS)
□ CTA: 행동 유도가 명확한가?
□ 맞춤법: 띄어쓰기, 문장부호 맞나?
□ 길이: 플랫폼 최적 길이인가?
□ 해시태그: 관련성 높은가? (5-10개)
□ 금지어: 의약품처럼 보이는 말 없나?
□ 일관성: 이전 포스트와 톤 같나?
```
