## 6. 약어 규칙

### MEDLINE 약어 표준

1. **관사 생략**: The, A, An
2. **전치사 생략**: of, in, on, for
3. **일반적 약어**:
   - Journal → J
   - International → Int
   - American → Am
   - British → Br
   - European → Eur
   - Science → Sci
   - Research → Res
   - Technology → Technol
   - Dermatology → Dermatol
   - Pharmacology → Pharmacol
   - Biochemistry → Biochem
   - Clinical → Clin
   - Experimental → Exp

### 주의사항

- 단일 단어 저널명은 약어화하지 않음 (예: Dermatology, Molecules)
- 약어와 전체명 혼용 금지
- 문서 전체에서 일관성 유지
