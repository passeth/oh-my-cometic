# INCI 명명법 가이드

## INCI란?

**INCI**(International Nomenclature of Cosmetic Ingredients)는 화장품 성분의 국제 표준 명명 체계입니다.

- **관리 기관**: PCPC (Personal Care Products Council, 미국)
- **법적 근거**: EU Regulation 1223/2009, US FDA 등
- **목적**: 전 세계 화장품 라벨의 성분 표기 통일

## INCI명 기본 규칙

### 1. 언어 및 문자

| 규칙 | 예시 |
|------|------|
| 대문자 사용 | NIACINAMIDE (O), Niacinamide (X) |
| 라틴 알파벳 | 그리스 문자 사용 안함 |
| 숫자 허용 | TOCOPHEROL, RETINOL |
| 하이픈 허용 | BETA-CAROTENE, PEG-100 |
| 공백 허용 | HYALURONIC ACID, SHEA BUTTER |

### 2. 식물 유래 성분

```
[학명 속명] [학명 종명] ([일반명]) [부위] [형태]

예시:
OLEA EUROPAEA (OLIVE) FRUIT OIL
└─속명─┘ └─종명─┘  └일반명┘ └부위┘ └형태┘

CAMELLIA SINENSIS LEAF EXTRACT
└──학명 (녹차)──┘ └부위┘ └형태┘
```

### 식물 성분 형태 용어

| 용어 | 의미 | 예시 |
|------|------|------|
| EXTRACT | 추출물 | CHAMOMILLA RECUTITA FLOWER EXTRACT |
| OIL | 오일 | ARGANIA SPINOSA KERNEL OIL |
| WATER | 증류수/워터 | ROSA DAMASCENA FLOWER WATER |
| BUTTER | 버터 | BUTYROSPERMUM PARKII BUTTER |
| WAX | 왁스 | CERA ALBA (BEESWAX) |
| POWDER | 분말 | ORYZA SATIVA BRAN POWDER |
| JUICE | 주스/즙 | ALOE BARBADENSIS LEAF JUICE |

### 식물 부위 용어

| 용어 | 의미 |
|------|------|
| LEAF | 잎 |
| FLOWER | 꽃 |
| FRUIT | 열매 |
| SEED | 씨앗 |
| KERNEL | 종자/알맹이 |
| ROOT | 뿌리 |
| BARK | 껍질 |
| STEM | 줄기 |
| PEEL | 껍질 (과일) |

### 3. 합성 성분

#### 화학명 기반
```
SODIUM HYALURONATE        ← 화학명 기반
TOCOPHERYL ACETATE        ← 화학명 기반
CETEARYL ALCOHOL          ← 화학명 기반
```

#### PEG/PPG 계열
```
PEG-[숫자]                ← Polyethylene Glycol
PEG-40 HYDROGENATED CASTOR OIL

PPG-[숫자]                ← Polypropylene Glycol
PPG-15 STEARYL ETHER
```

#### 폴리머 계열
```
CARBOMER                  ← 카보머 (점증제)
ACRYLATES COPOLYMER       ← 아크릴레이트 공중합체
POLYQUATERNIUM-[숫자]     ← 양이온성 폴리머
```

### 4. 색소 (Colorants)

#### CI 번호 체계
```
CI [5자리 숫자]

CI 77891                  ← TITANIUM DIOXIDE
CI 77491                  ← IRON OXIDES (RED)
CI 77492                  ← IRON OXIDES (YELLOW)
CI 77499                  ← IRON OXIDES (BLACK)
CI 15985                  ← YELLOW 6
```

#### 색소 명명 예시
```
[+/- CI 77891, CI 77491]  ← 색상 범위 표기 (may contain)
```

### 5. 향료

```
FRAGRANCE                 ← 일반 향료 (미국)
PARFUM                    ← 일반 향료 (EU)
AROMA                     ← 방향 성분

개별 향료 성분:
LINALOOL
LIMONENE
CITRONELLOL
GERANIOL
HEXYL CINNAMAL
```

### 6. 나노물질

```
[INCI명] (NANO)

TITANIUM DIOXIDE (NANO)
ZINC OXIDE (NANO)
SILICA (NANO)
```

## 특수 케이스

### 물 (Water)
```
AQUA                      ← EU 표기
WATER                     ← 미국 표기
AQUA (WATER)              ← 통합 표기
```

### 혼합물
```
[성분1] (AND) [성분2]

CAPRYLIC/CAPRIC TRIGLYCERIDE
CETEARYL ALCOHOL (AND) CETEARYL GLUCOSIDE
```

### 동의어/대체명
```
BUTYROSPERMUM PARKII (SHEA) BUTTER
└────학명────┘  └일반명┘

CERA ALBA (BEESWAX)
└학명(라틴)┘ └일반명┘
```

## INCI명 vs 상품명 vs 화학명

| 유형 | 예시 | 용도 |
|------|------|------|
| **INCI명** | NIACINAMIDE | 라벨 표기 |
| **상품명** | Vitamin B3 | 마케팅 |
| **화학명** | Pyridine-3-carboxamide | 화학 문헌 |
| **CAS명** | 3-Pyridinecarboxylic acid amide | CAS 등록 |

## 자주 혼동되는 INCI명

| 흔히 쓰는 이름 | 정확한 INCI명 |
|---------------|---------------|
| Vitamin C | ASCORBIC ACID |
| Vitamin E | TOCOPHEROL |
| Vitamin A | RETINOL |
| 히알루론산 | HYALURONIC ACID 또는 SODIUM HYALURONATE |
| 알로에 | ALOE BARBADENSIS LEAF JUICE/EXTRACT |
| 녹차 추출물 | CAMELLIA SINENSIS LEAF EXTRACT |
| 감초 추출물 | GLYCYRRHIZA GLABRA ROOT EXTRACT |
| 코엔자임Q10 | UBIQUINONE |

## 라벨 표기 규정

### EU 규정 (EC 1223/2009)
- INCI명 사용 필수
- 농도 순 나열 (1% 이상)
- 1% 미만: 순서 무관
- 색소: `[+/-]` 표기 가능
- 향료 알레르겐: 별도 표기 필수

### 한국 규정 (화장품법)
- INCI명 또는 한글명 사용 가능
- 대한화장품협회 표준 한글명 권장
- 함량 순 나열

### 미국 규정 (FDA)
- INCI명 사용
- OTC 의약품 성분은 별도 표기
- WATER (AQUA 대신 WATER 사용)
