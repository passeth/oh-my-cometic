## Dermal Absorption Guidance

### When Measured Data is Unavailable

| Molecular Weight | LogP | Default Absorption |
|-----------------|------|-------------------|
| > 500 Da | Any | 10% |
| < 500 Da | > 0 | 100% (or 50% for lipophilic) |
| < 500 Da | < 0 | 100% |
| Polymers (> 1000 Da) | Any | 0% |
| Inorganic particles | Any | 0-10% |

### SCCS Dermal Absorption Guidelines

1. **Measured in vivo** human data preferred
2. **In vitro** data with human skin acceptable
3. **In vitro** with pig skin as surrogate
4. **Default 100%** if no data available (conservative)
5. **50% default** may be used for rinse-off products at assessor discretion

## MoS Calculation Reference

### Formula

```
MoS = NOAEL / SED

SED = (A × C/100 × DA/100 × F × RF) / BW
```

### Parameters

| Symbol | Description | Default Value |
|--------|-------------|---------------|
| A | Amount applied (mg/day) | Product-specific |
| C | Concentration (%) | Formula-specific |
| DA | Dermal absorption (%) | Ingredient-specific |
| F | Frequency (per day) | Product-specific |
| RF | Retention factor | 1.0 (leave-on), 0.01-0.1 (rinse-off) |
| BW | Body weight (kg) | 60 kg |

### MoS Interpretation

| MoS Value | Interpretation |
|-----------|----------------|
| ≥ 100 | Safe for use as assessed |
| 50 - 100 | May require additional justification |
| < 50 | Unacceptable without risk management |
