# 미백 기능성 고시원료 (22종)

## 개요

미백 기능성 화장품에 사용할 수 있는 식약처 고시원료 22종의 상세 정보입니다.
고시된 농도 범위 내에서 사용 시 기능성화장품 보고로 간소화된 절차를 통해 제품 출시가 가능합니다.

---

## 원료 목록

### 1. 나이아신아마이드 (Niacinamide)

| 항목 | 내용 |
|------|------|
| **INCI명** | Niacinamide |
| **화학명** | Nicotinamide, Vitamin B3 |
| **CAS No.** | 98-92-0 |
| **고시 농도** | 2~5% |
| **적용 제형** | 모든 leave-on 및 wash-off 제형 |

**특성**:
- 타이로시나제 억제 작용
- 멜라노좀 전이 억제
- 피부 장벽 강화 효과 (세라마이드 합성 촉진)
- 피지 조절 효과
- 안정성 우수

**배합 시 주의사항**:
- pH 5~7에서 안정
- 순수 비타민C(L-Ascorbic Acid)와 동시 사용 시 니코틴산 생성 가능 (피부 자극)
- 시간차 사용 또는 비타민C 유도체 사용 권장

## 원료 비교 가이드

### 효능 강도 순 (일반적 평가)

| 순위 | 원료 | 특징 |
|------|------|------|
| 1 | 4-부틸레조시놀 | 저농도(0.5%)로 강력한 효과 |

| 순위 | 원료 | pH 범위 |
|------|------|---------|
| 1 | 아스코빌글루코사이드 | 6~7 |

| 3 | 나이아신아마이드 | 5~7 |

## 복합 기능성 조합 예시

### 미백 + 주름개선

```
나이아신아마이드 3% + 아데노신 0.04%
→ 보고 대상 (양쪽 모두 고시 원료)
```

### 미백 + 미백 (복수 원료)

```
나이아신아마이드 2% + 알부틴 2%
→ 각 원료 고시 농도 충족 시 보고 가능
→ 복합 효과 기대
```

## 주의사항

### 농도 관련

- 고시 농도 미만: 기능성 표시 불가 (일반 화장품)
- 고시 농도 초과: 심사 대상 (추가 안전성/유효성 자료 필요)

### 안정성 관련

- 비타민C 유도체: 산화 방지 패키징 권장
- 레티노이드와 병용: 자외선 노출 주의 안내

### 표시 관련

- 기능성 원료명 필수 표기
- 배합 농도 표기 (의무)
- "피부 미백에 도움" 문구 사용
