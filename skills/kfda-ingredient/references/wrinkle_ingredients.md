# 주름개선 기능성 고시원료 (6종)

## 개요

주름개선 기능성 화장품에 사용할 수 있는 식약처 고시원료 6종의 상세 정보입니다.
피부에 탄력을 주어 피부의 주름을 완화 또는 개선하는 데 도움을 주는 기능을 가진 화장품에 사용됩니다.

## 원료 목록

### 1. 레티놀 (Retinol)

| 항목 | 내용 |
|------|------|
| **INCI명** | Retinol |
| **화학명** | Vitamin A, all-trans-Retinol |
| **CAS No.** | 68-26-8 |
| **고시 농도** | 2,500 IU/g |
| **적용 제형** | 모든 leave-on 제형 |

**특성**:
- 비타민A의 활성형
- 세포 분화 및 재생 촉진
- 콜라겐 합성 증가
- 표피 두께 증가
- FDA 및 식약처 모두 인정한 주름개선 성분

**작용 메커니즘**:
1. 핵 수용체(RAR/RXR)에 결합
2. 유전자 발현 조절
3. 콜라겐 I, III 합성 촉진
4. MMP(기질금속단백분해효소) 억제
5. 표피 세포 턴오버 정상화

**배합 시 주의사항**:
- **광안정성 불안정**: 빛에 의해 쉽게 분해
  - 불투명/차광 용기 사용 필수
  - 에어리스 펌프 권장
- **산소 민감성**: 산화되기 쉬움
  - 질소 충전 고려
  - 항산화제 병용 (토코페롤 등)
- **열 민감성**: 고온에서 분해
  - 저온 유화 권장
  - 냉장 보관 권장
- **pH 민감성**: 산성에서 불안정
  - 중성~약알칼리 권장
- **배합 금기**: 산성 성분(AHA, BHA, 비타민C)과 동시 사용 자제

**농도 계산**:
```
2,500 IU/g = 2,500 IU per gram of product
레티놀 순도에 따른 환산 필요
예: 순수 레티놀 1 IU = 약 0.3 μg
→ 2,500 IU = 약 0.75 mg/g = 0.075%
```

**소비자 주의사항 (필수 표시)**:
- 처음 사용 시 소량으로 시작
- 야간 사용 권장
- 자외선차단제 병행 사용
- 임산부 사용 자제 권고

### 2. 레티닐팔미테이트 (Retinyl Palmitate)

| 항목 | 내용 |
|------|------|
| **INCI명** | Retinyl Palmitate |
| **화학명** | Vitamin A Palmitate |
| **CAS No.** | 79-81-2 |
| **고시 농도** | 10,000 IU/g |
| **적용 제형** | 모든 제형 |

**특성**:
- 레티놀 + 팔미틱산 에스터
- 레티놀의 안정화된 형태
- 피부 내에서 레티놀로 전환
- 레티놀 대비 순한 자극
- 안정성 우수

**작용 메커니즘**:
```
레티닐팔미테이트 → 에스터라제에 의해 → 레티놀 → 레티날 → 레티노산
                (피부 내 효소)        (활성형)
```

**배합 시 주의사항**:
- 레티놀보다 안정적이나 여전히 빛, 산소에 민감
- 차광 용기 권장
- 효능은 레티놀 대비 완만

**농도 비교**:
```
레티놀: 2,500 IU/g
레티닐팔미테이트: 10,000 IU/g

→ 레티닐팔미테이트가 4배 높은 IU 허용
→ 전환 효율 고려하면 유사한 효능 수준
```

**장점**:
- 초보자에게 적합
- 민감성 피부 사용 가능
- 상온 유통 용이

### 3. 아데노신 (Adenosine)

| 항목 | 내용 |
|------|------|
| **INCI명** | Adenosine |
| **화학명** | 9-β-D-Ribofuranosyladenine |
| **CAS No.** | 58-61-7 |
| **고시 농도** | 0.04% |
| **적용 제형** | 모든 leave-on 제형 |

**특성**:
- 인체 내 자연 존재하는 뉴클레오시드
- DNA/RNA 구성 성분
- 세포 에너지(ATP) 전구체
- 뛰어난 안전성
- 자극 거의 없음

**작용 메커니즘**:
1. **세포 에너지 공급**: ATP 생성 촉진
2. **콜라겐 합성**: 섬유아세포 활성화
3. **세포 신호전달**: 아데노신 수용체 활성화
4. **항염 효과**: 염증 매개체 억제

**배합 시 주의사항**:
- **용해도**: 수용성, 물에 잘 녹음
- **pH**: 넓은 pH 범위에서 안정 (4~9)
- **열 안정성**: 비교적 안정
- **상용성**: 대부분의 원료와 호환

**장점**:
- 민감성 피부 적합
- 임산부 사용 가능
- 레티노이드 대안
- 주간 사용 가능
- 광안정성 문제 없음

**단점**:
- 낮은 고시 농도 (0.04%)
- 레티놀 대비 효과 완만

**제품 유형별 활용**:
| 제품 | 활용도 | 비고 |
|------|--------|------|
| 에센스 | ★★★★★ | 수용성 특성 활용 |
| 크림 | ★★★★★ | 안정성 우수 |
| 선크림 | ★★★★☆ | 복합 기능성 |
| 아이크림 | ★★★★★ | 순한 성분 |
| 앰플 | ★★★★★ | 고기능성 강조 |

## 레티노이드 농도 환산표

### IU 기준 환산

| 원료 | 고시 농도 | 순수 레티놀 환산 | 비고 |
|------|-----------|------------------|------|
| 레티놀 | 2,500 IU/g | 0.075% | 기준 |
| 레티닐아세테이트 | 5,500 IU/g | ~0.10% | 전환 효율 고려 |
| 레티닐팔미테이트 | 10,000 IU/g | ~0.15% | 전환 효율 고려 |

### 단위 변환

```
1 IU 레티놀 = 0.3 μg 레티놀
1 mg 레티놀 = 3,333 IU

2,500 IU/g = 0.75 mg/g = 0.075% w/w
```

## 복합 기능성 조합

### 주름개선 + 미백

**권장 조합**:
```
아데노신 0.04% + 나이아신아마이드 3%
→ 보고 대상
→ 안정성 우수, 주야간 사용 가능
```

**프리미엄 조합**:
```
레티놀 2,500 IU/g + 알부틴 3%
→ 보고 대상
→ 야간 사용 권장, 고기능성
```

### 주름개선 + 자외선차단

**권장 조합**:
```
아데노신 0.04% + SPF50+ PA++++
→ 보고 대상
→ 데이 케어 제품에 적합
```

**주의 조합**:
```
레티놀 + SPF 제품
→ 레티놀의 광안정성 문제
→ 별도 제품으로 분리 사용 권장
```

## 안정화 기술

### 레티놀 안정화 방법

1. **캡슐화 (Encapsulation)**
   - 리포좀 캡슐화
   - 마이크로캡슐
   - 나노에멀전

2. **유도체 사용**
   - 레티닐팔미테이트
   - 레티닐아세테이트

3. **처방 최적화**
   - 무수 처방
   - 질소 충전
   - 산화 방지제 병용 (토코페롤, BHT)

4. **포장 최적화**
   - 에어리스 펌프
   - 차광 용기
   - 알루미늄 튜브

### 아데노신 안정화

- 특별한 안정화 불필요
- 넓은 pH 범위에서 안정
- 열에 안정적

## 규제 관련 참고사항

### 보고 vs 심사

| 조건 | 심사 유형 |
|------|-----------|
| 고시원료 + 고시농도 | **보고** (간소화) |
| 고시원료 + 농도 초과 | **심사** (안전성 입증) |
| 비고시 레티노이드 | **심사** (전체 서류) |

### 표시 의무사항

**필수 표시**:
- "기능성화장품" 문구
- "주름개선에 도움" 문구
- 기능성 원료명 및 함량
- 심사(보고)번호

**레티놀 특별 주의문구**:
- "처음 사용 시 소량으로 테스트 후 사용"
- "자외선차단제 병행 사용 권장"
- "임산부는 사용 전 전문의와 상담"

### 광고 시 주의

**허용 표현**:
- "주름 개선에 도움을 줍니다"
- "피부 탄력에 도움을 줍니다"
