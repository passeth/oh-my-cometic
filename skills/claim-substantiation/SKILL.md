# Claim Substantiation Skill

화장품 효능 클레임의 과학적 입증 전략 수립 및 시험 설계를 지원하는 스킬입니다.

## Overview

화장품 클레임은 제품의 효능과 특성을 소비자에게 전달하는 핵심 마케팅 요소입니다. 그러나 모든 클레임은 과학적 근거에 기반해야 하며, 각국의 규제 요건을 충족해야 합니다. 이 스킬은 클레임 유형 분류부터 입증 방법 선택, 시험 설계, 규제 적합 문구 작성까지 전 과정을 안내합니다.

## When to Use

이 스킬은 다음 상황에서 활용됩니다:

1. **마케팅 클레임 검토**: 기획된 클레임의 입증 가능성 및 규제 적합성 평가
2. **시험 설계**: 클레임 입증을 위한 적절한 시험 방법 선택 및 프로토콜 설계
3. **규제 대응**: 각국 규제 기관의 클레임 근거 요청 대응
4. **신제품 개발**: 차별화된 클레임 전략 수립
5. **기존 제품 강화**: 추가 시험을 통한 클레임 확장

## Core Capabilities

### 1. 클레임 유형 분류

화장품 클레임은 그 성격에 따라 다양한 유형으로 분류되며, 각 유형별로 요구되는 근거 수준이 다릅니다.

#### 기능적 클레임 (Functional Claims)
제품의 기본적인 기능을 설명하는 클레임입니다.
- **보습**: "피부에 수분을 공급합니다"
- **진정**: "민감한 피부를 진정시킵니다"
- **세정**: "노폐물을 부드럽게 제거합니다"
- **자외선 차단**: "SPF 50+ PA++++"

**근거 수준**: 기본적인 in vitro 또는 기기 측정으로 입증 가능

#### 성능 클레임 (Performance Claims)
제품의 효과 지속 시간이나 정량적 성능을 명시하는 클레임입니다.
- **시간 기반**: "24시간 보습 유지", "48시간 지속력"
- **정량적**: "보습력 30% 향상", "주름 깊이 20% 감소"
- **즉각적**: "즉각적인 탄력 개선"

**근거 수준**: 인체적용시험(In vivo) 및 기기 측정 데이터 필수

#### 비교 클레임 (Comparative Claims)
타 제품 또는 기존 제품 대비 우수성을 주장하는 클레임입니다.
- **자사 비교**: "기존 제품 대비 2배 효과"
- **일반 비교**: "일반 크림보다 더 오래 지속"
- **성분 비교**: "비타민 C 5배 농축"

**근거 수준**: 비교 대상과의 동일 조건 비교 시험 필수, 법적 리스크 검토 필요

#### 소비자 인식 클레임 (Consumer Perception Claims)
소비자 조사 결과에 기반한 클레임입니다.
- **동의율**: "90%가 피부 개선을 느꼈습니다"
- **만족도**: "95% 고객 만족"
- **재구매 의향**: "10명 중 9명이 재구매 의향"

**근거 수준**: 통계적으로 유의한 소비자 조사(HUT) 데이터 필수

### 2. 입증 방법 선택

클레임 유형과 요구되는 근거 수준에 따라 적절한 입증 방법을 선택합니다.

#### In Vitro 시험 (세포/효소 수준)
세포 배양이나 효소 반응을 통해 성분의 효능을 입증합니다.

| 효능 | 시험 방법 | 세포/기질 |
|------|----------|----------|
| 항산화 | DPPH, ABTS, ORAC | 화학적 측정 |
| 미백 | Tyrosinase 억제 | B16F10 멜라노마 |
| 주름개선 | MMP-1 억제, Collagen 합성 | HDF 섬유아세포 |
| 항염 | NO, PGE2, IL-6 억제 | RAW 264.7 대식세포 |
| 보습 | HA 합성 촉진 | HaCaT 각질형성세포 |

**장점**: 비용 효율적, 빠른 결과, 작용 기전 규명
**한계**: 실제 피부 효과와 차이 가능, 단독 근거로 부족

#### Ex Vivo 시험 (피부 모델)
인공피부 또는 적출 피부 조직을 이용한 시험입니다.

- **3D 피부 모델**: EpiDerm, MatTek, KeraSkin
- **경피 흡수**: Franz diffusion cell
- **피부 자극**: HET-CAM, EpiSkin

**장점**: In vivo에 가까운 결과, 윤리적 우려 감소
**한계**: 비용 높음, 시스템적 효과 평가 불가

#### In Vivo 시험 (인체적용시험)
실제 인체에 제품을 적용하여 효능을 평가합니다.

| 클레임 | 평가 항목 | 측정 기기 |
|--------|----------|----------|
| 보습 | 피부 수분량 | Corneometer |
| 보습 | 경피수분손실 | Tewameter |
| 주름개선 | 주름 파라미터 | PRIMOS, Visiometer |
| 탄력 | 피부 탄성 | Cutometer |
| 미백 | 멜라닌 지수 | Mexameter |
| 진정 | 홍반 지수 | Chromameter |
| 모공 | 모공 크기 | Visiopor |

**장점**: 가장 신뢰도 높은 근거, 규제 기관 선호
**한계**: 높은 비용, 긴 시험 기간, IRB 승인 필요

#### 소비자 조사 (Home Use Test)
실제 사용 환경에서의 소비자 평가입니다.

- **사용 기간**: 2-8주
- **피험자 수**: 최소 30명 (통계적 유의성 확보 시 50-100명 권장)
- **평가 항목**: 효능 체감, 사용감, 만족도, 재구매 의향

**장점**: 실제 사용 조건 반영, 소비자 언어로 클레임 가능
**한계**: 주관적 평가, 플라시보 효과

### 3. 시험 설계 가이드

#### 피험자 수 산정

통계적 검정력(Power)에 기반한 피험자 수 계산:

```
n = 2 * (Zα + Zβ)² * σ² / δ²

n: 그룹당 필요 피험자 수
Zα: 유의수준에 해당하는 Z값 (α=0.05 → 1.96)
Zβ: 검정력에 해당하는 Z값 (β=0.20 → 0.84)
σ: 예상 표준편차
δ: 검출하고자 하는 최소 차이
```

**일반적 권장 피험자 수**:
- 단일군 비교: 20-30명
- 양측 비교: 군당 25-35명
- 다군 비교: 군당 20-25명
- 탈락률 고려: 10-20% 추가 모집

#### 대조군 설정

| 시험 유형 | 대조군 종류 | 적용 |
|----------|------------|------|
| 단일 제품 효능 | 무처리 대조 | 사용 전후 비교 |
| 비교 클레임 | 기준 제품 | 타제품 또는 자사 기존제품 |
| 성분 효능 | 기제 대조 | 활성 성분 제외 기제 |
| 이중맹검 | 플라시보 | 외관 동일한 대조 제품 |

#### 통계 분석 방법

**연속형 데이터** (기기 측정값):
- Paired t-test: 사용 전후 비교
- Two-sample t-test: 두 군 비교
- ANOVA: 다군 비교
- Repeated measures ANOVA: 시간에 따른 변화

**범주형 데이터** (설문 응답):
- Chi-square test: 비율 비교
- McNemar test: 짝지은 비율 비교
- Wilcoxon signed-rank: 순서형 데이터

**유의수준**:
- 양측 검정: p < 0.05 (기본)
- 다중 비교: Bonferroni 보정 적용

### 4. 클레임 문구 작성

#### 규제 적합 문구

**허용 표현**:
- "보습에 도움을 줍니다" (기능 보조)
- "피부 결 개선에 기여합니다" (효과 기여)
- "OO 시험에서 입증되었습니다" (근거 명시)

**주의 표현**:
- "치료", "완치", "예방" → 의약품 영역 침범
- "100% 효과", "확실한" → 과장 광고
- "세계 최초", "유일한" → 입증 어려움

#### 과장 표현 회피

| 과장 표현 | 권장 대체 표현 |
|----------|--------------|
| 주름이 사라집니다 | 주름 개선에 도움을 줍니다 |
| 완벽한 보습 | 풍부한 보습감을 선사합니다 |
| 피부를 치료합니다 | 피부 건강 관리에 도움을 줍니다 |
| 노화를 막습니다 | 피부 탄력 유지에 기여합니다 |

#### 근거 기반 표현

시험 결과를 클레임으로 전환하는 예시:

| 시험 결과 | 클레임 표현 |
|----------|-----------|
| Corneometer 30% 증가 (p<0.05) | "피부 수분량 30% 향상*" |
| 90%가 보습 효과 동의 | "사용자 90%가 보습 효과 체감*" |
| MMP-1 50% 억제 (in vitro) | "주름 개선 기능성 성분 함유" |

*주석 표기: "인체적용시험 결과, n=30, 4주 사용" 또는 "소비자 조사 결과, n=100"

## Common Workflows

### 신제품 클레임 전략 수립

```
1. 제품 컨셉 및 타겟 클레임 정의
   ↓
2. 클레임 유형 분류 및 근거 수준 결정
   ↓
3. 입증 방법 선택 (비용/시간/신뢰도 고려)
   ↓
4. 시험 프로토콜 설계
   ↓
5. 시험 수행 및 결과 분석
   ↓
6. 규제 적합 클레임 문구 작성
   ↓
7. 마케팅 검토 및 최종 확정
```

### 기존 제품 클레임 강화

```
1. 현재 클레임 및 근거 자료 검토
   ↓
2. 강화 가능한 클레임 영역 식별
   ↓
3. 추가 시험 필요성 평가
   ↓
4. 비용 대비 효과 분석
   ↓
5. 추가 시험 수행 (필요시)
   ↓
6. 확장된 클레임 문구 개발
```

## Best Practices

### 근거 수준 확보

1. **다층적 근거**: In vitro + In vivo 조합으로 신뢰도 향상
2. **충분한 피험자**: 통계적 유의성 확보를 위한 적정 인원
3. **적절한 시험 기간**: 클레임에 맞는 충분한 평가 기간
4. **재현성**: 가능하면 2회 이상 시험으로 결과 확인

### 규제 준수

1. **국가별 규제 파악**: 수출 대상국의 클레임 규제 사전 검토
2. **의약품 영역 회피**: 치료, 예방, 완치 등 금지 표현 배제
3. **근거 보관**: 클레임 근거 자료 최소 5년 이상 보관
4. **라벨링 정합성**: 제품 라벨, 홍보물, 온라인 일관성 유지

### 비용 효율성

| 전략 | 설명 |
|------|------|
| 성분 근거 활용 | 원료사 제공 데이터 활용 |
| 단계적 접근 | In vitro → In vivo 순차 진행 |
| 복합 시험 | 여러 클레임 동시 입증 |
| 기존 데이터 활용 | 유사 제품 시험 결과 참조 |

## Output Format

클레임 입증 전략 보고서 구성:

```markdown
# [제품명] 클레임 입증 전략

## 1. 타겟 클레임
- 주요 클레임: [클레임 내용]
- 클레임 유형: [기능적/성능/비교/소비자 인식]
- 필요 근거 수준: [기본/중간/높음]

## 2. 입증 방법
### 선택된 시험
- [시험 1]: [목적], [기대 결과]
- [시험 2]: [목적], [기대 결과]

### 시험 일정 및 비용
| 시험 | 기간 | 예상 비용 |
|------|------|----------|
| [시험명] | [기간] | [비용] |

## 3. 시험 설계
### 피험자 선정
- 대상: [연령, 피부 타입 등]
- 인원: [명] (탈락률 [%] 고려)

### 평가 항목
- [기기/설문]: [측정 항목]

### 통계 분석
- [분석 방법]: [유의수준]

## 4. 클레임 문구 (안)
- 기기 결과 기반: "[문구]*"
- 소비자 조사 기반: "[문구]*"
- *주석: [근거 출처]

## 5. 규제 검토
- 국내: [적합/검토필요]
- 수출국: [국가별 검토 결과]
```
