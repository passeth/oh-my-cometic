# Clinical Study Design Guide

화장품 인체적용시험(Human Application Test) 설계를 위한 종합 가이드입니다.

## 1. IRB 승인 절차

### IRB (Institutional Review Board) 개요

IRB는 인체적용시험의 윤리성과 과학적 타당성을 심의하는 기관입니다.

#### 심의 대상
- 인체에 직접 적용되는 모든 시험
- 피험자로부터 데이터를 수집하는 연구
- 피험자 사진, 영상 등 개인정보 수집 연구

#### 심의 제외 (기관별 상이)
- 기존 공개 데이터 분석
- 익명화된 자료 활용 연구

### IRB 제출 서류

```
1. 연구 계획서 (Protocol)
   - 연구 목적 및 배경
   - 연구 방법론
   - 피험자 선정/제외 기준
   - 평가 항목 및 일정
   - 예상 위험 및 대응 방안
   - 통계 분석 계획

2. 동의서 (Informed Consent Form)
   - 연구 목적 설명
   - 시험 절차 안내
   - 예상 위험 및 이익
   - 자발적 참여 및 철회 권리
   - 개인정보 보호 방침
   - 연구자 연락처

3. 피험자 모집 공고문
   - 연구 개요
   - 참여 조건
   - 보상 내용
   - 연락처

4. 시험 제품 정보
   - 전성분표
   - 안전성 자료 (패치테스트 결과 등)
   - 제조 정보

5. 연구자 자격 서류
   - 연구책임자 CV
   - 시험 기관 정보
```

### IRB 심의 일정

| 단계 | 소요 기간 | 비고 |
|------|----------|------|
| 서류 제출 | - | 월 1-2회 심의 |
| 서류 검토 | 1주 | 보완 요청 가능 |
| IRB 회의 심의 | 1주 | 참석 발표 가능 |
| 심의 결과 통보 | 3-5일 | 승인/조건부승인/보류/반려 |
| 조건 보완 | 1-2주 | 해당 시 |
| 최종 승인 | - | 승인번호 발급 |

**총 소요 기간**: 약 3-6주 (기관별 상이)

### IRB 승인 유형

1. **승인 (Approved)**: 즉시 시험 진행 가능
2. **조건부 승인 (Conditionally Approved)**: 경미한 수정 후 진행
3. **보류 (Deferred)**: 주요 수정 후 재심의
4. **반려 (Not Approved)**: 윤리적/과학적 문제로 승인 거부

## 2. 피험자 선정/제외 기준

### 일반적 선정 기준 (Inclusion Criteria)

```
공통 기준:
- 만 19세 이상 만 65세 이하 성인 남녀
- 본 연구의 목적을 이해하고 자발적으로 참여 동의서에 서명한 자
- 시험 기간 동안 지정된 방문 일정을 준수할 수 있는 자
- 시험 기간 동안 시험 제품 외 동일 부위에 다른 제품 사용을 자제할 수 있는 자

피부 타입별 기준 (예시):
- 건성 피부: Corneometer 측정값 < 40 AU
- 민감성 피부: 스팅 테스트 양성 또는 자가 보고
- 주름 피부: 주름 등급 3 이상 (Fitzpatrick Scale)
- 과색소침착: Mexameter MI > 200 또는 육안 확인
```

### 일반적 제외 기준 (Exclusion Criteria)

```
건강 관련:
- 급성 또는 만성 피부 질환이 있는 자 (아토피, 건선, 접촉성 피부염 등)
- 시험 부위에 상처, 염증, 감염이 있는 자
- 면역 관련 질환이 있는 자
- 현재 피부과 치료 중인 자

제품/시술 관련:
- 시험 4주 이내 동일 부위에 피부과 시술을 받은 자
  (레이저, 필링, 보톡스, 필러 등)
- 시험 2주 이내 전신 스테로이드 또는 면역억제제 복용자
- 시험 제품 성분에 알레르기 이력이 있는 자

기타:
- 임신 중이거나 수유 중인 여성
- 시험 기간 중 임신 계획이 있는 자
- 최근 6개월 이내 다른 임상시험 참여자
- 연구자 판단에 의해 부적합하다고 판단되는 자
```

### 클레임별 특수 선정 기준

| 클레임 | 추가 선정 기준 |
|--------|--------------|
| 보습 | Corneometer < 40 AU, TEWL > 15 g/m2/h |
| 주름개선 | 눈가 또는 이마 주름 Grade 3+ |
| 미백 | 색소침착 부위 보유, MI > 200 |
| 진정 | 민감성 피부 자가 보고자 |
| 모공 | 모공 확대 Grade 2+ |
| 탄력 | Cutometer R2 < 0.8 |

## 3. 무작위배정 및 눈가림

### 무작위배정 (Randomization)

#### 단순 무작위배정 (Simple Randomization)
```
방법: 난수표 또는 컴퓨터 생성 난수
장점: 단순, 객관적
단점: 소규모 시험에서 불균형 가능
```

#### 블록 무작위배정 (Block Randomization)
```
방법: 일정 블록 내에서 균등 배정
예시: 블록 크기 4 → AABB, ABAB, ABBA 등
장점: 군간 균형 유지
적용: 대부분의 화장품 인체적용시험
```

#### 층화 무작위배정 (Stratified Randomization)
```
방법: 주요 변수(연령, 피부타입 등)별 층화 후 배정
장점: 교란 변수 통제
적용: 피부타입이 결과에 영향 줄 때
```

### 눈가림 (Blinding)

#### 단일맹검 (Single-Blind)
- 피험자만 배정군 모름
- 연구자는 알고 있음
- 적용: 대부분의 화장품 시험

#### 이중맹검 (Double-Blind)
- 피험자, 평가자 모두 배정군 모름
- 적용: 비교 시험, 플라시보 대조 시험

#### 눈가림 방법
```
- 동일 외관의 용기 사용
- 코드화된 제품 라벨
- 독립된 배정자 지정
- 유사한 제형, 색상, 향 맞춤
```

## 5. 유의성 검정

### 통계적 유의수준

```
유의수준 (α):
- 표준: α = 0.05 (95% 신뢰수준)
- 엄격: α = 0.01 (99% 신뢰수준)

검정력 (1-β):
- 표준: 80% (β = 0.20)
- 권장: 90% (β = 0.10)

양측 vs 단측 검정:
- 양측 (Two-tailed): 차이 방향 미정 시
- 단측 (One-tailed): 개선만 기대 시
```

### 통계 분석 방법

#### 기술 통계 (Descriptive Statistics)
```
연속형 변수: 평균 ± 표준편차, 중앙값, 범위
범주형 변수: 빈도, 백분율
```

#### 단일군 전후 비교

```python
# Paired t-test (정규분포 시)
from scipy import stats
t_stat, p_value = stats.ttest_rel(baseline, after_4weeks)

# Wilcoxon signed-rank test (비정규분포 시)
stat, p_value = stats.wilcoxon(baseline, after_4weeks)
```

**변화율 계산**:
```
변화율(%) = (측정값_후 - 측정값_전) / 측정값_전 × 100
```

#### 두 군 비교

```python
# Independent t-test (정규분포 시)
t_stat, p_value = stats.ttest_ind(group_test, group_control)

# Mann-Whitney U test (비정규분포 시)
stat, p_value = stats.mannwhitneyu(group_test, group_control)
```

#### 다군 비교 및 시간 경과

```python
# One-way ANOVA
f_stat, p_value = stats.f_oneway(group1, group2, group3)

# Repeated measures ANOVA (반복측정)
# 시간 × 그룹 상호작용 확인
```

#### 다중 비교 보정

```
Bonferroni 보정:
- 보정 유의수준 = α / 비교 횟수
- 예: 3회 비교 시 0.05/3 = 0.017

Tukey HSD:
- ANOVA 후 사후 검정
- 모든 쌍별 비교
```

### 결과 해석 및 보고

#### 통계적 유의성 기술

```
"시험군의 피부 수분량은 사용 4주 후 기저치 대비
유의하게 증가하였다 (p < 0.05, paired t-test)."

"시험군과 대조군 간 주름 깊이 변화율에 유의한
차이가 관찰되었다 (p = 0.023, independent t-test)."
```

#### 효과 크기 (Effect Size)

```
Cohen's d:
- 0.2: 작은 효과
- 0.5: 중간 효과
- 0.8: 큰 효과

계산: d = (평균_후 - 평균_전) / 표준편차
```

### 결과표 예시

```
Table. 피부 수분량 변화 (N=30)

| 평가 시점 | Mean ± SD | 변화량 | 변화율(%) | p-value |
|----------|-----------|--------|----------|---------|
| Baseline | 35.2 ± 5.8 | - | - | - |
| 2주 | 42.1 ± 6.2 | 6.9 ± 3.1 | 19.6 | <0.001* |
| 4주 | 45.8 ± 5.9 | 10.6 ± 4.2 | 30.1 | <0.001* |

*Paired t-test, p < 0.05
```

## 6. 시험 기관 선정

### 국내 주요 피부과학 시험 기관

| 기관 유형 | 특징 | 적합 시험 |
|----------|------|----------|
| 대학 부속 연구소 | 학술적 신뢰도 높음 | 기능성 입증, 논문 게재 |
| 전문 시험 기관 | 효율성, 비용 경쟁력 | 일반 클레임 입증 |
| 병원 피부과 | 의료진 참여 | 민감 피부, 의학적 평가 |

### 시험 기관 선정 기준

1. **인증**: GLP, KFDA 인체적용시험기관 지정
2. **장비**: 필요 측정 기기 보유 여부
3. **경험**: 유사 시험 수행 실적
4. **비용**: 시험 단가 및 패키지 가격
5. **일정**: 시험 가능 시기 및 소요 기간
