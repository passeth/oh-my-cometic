# Claim Types and Regulatory Guide

화장품 클레임 유형 분류 및 국가별 규제 가이드입니다.

## 1. 클레임 유형 분류

### 성격별 분류

#### 기능적 클레임 (Functional Claims)
제품의 기본 기능을 설명하는 클레임

```
예시:
- "피부에 수분을 공급합니다"
- "노폐물을 깨끗이 제거합니다"
- "피부를 보호합니다"
- "메이크업을 부드럽게 해줍니다"

근거 수준: 기본 (성분 특성, 제형 특성)
```

#### 효능 클레임 (Efficacy Claims)
제품 사용으로 인한 피부 개선 효과

```
예시:
- "주름 개선에 도움을 줍니다"
- "피부 탄력을 높여줍니다"
- "피부톤을 밝게 해줍니다"
- "모공을 조여줍니다"

근거 수준: 중간~높음 (In vitro, In vivo 시험)
```

#### 성능 클레임 (Performance Claims)
정량적 효과 또는 지속 시간

```
예시:
- "24시간 보습 지속"
- "수분량 30% 증가"
- "주름 깊이 20% 감소"
- "SPF 50+ PA++++"

근거 수준: 높음 (In vivo 기기 측정 필수)
```

#### 비교 클레임 (Comparative Claims)
타 제품 대비 우수성

```
예시:
- "기존 제품 대비 2배 보습력"
- "일반 크림보다 더 오래 지속"
- "비타민 C 5배 농축"

근거 수준: 최고 (동일 조건 비교 시험 필수)
법적 리스크: 높음
```

#### 소비자 인식 클레임 (Consumer Perception Claims)
소비자 조사 결과 기반

```
예시:
- "90%가 피부 개선을 체감했습니다"
- "10명 중 9명이 재구매 의향"
- "사용자 95% 만족"

근거 수준: 중간 (통계적 유의성 확보 HUT)
```

#### 성분 클레임 (Ingredient Claims)
특정 성분 함유 또는 미함유

```
예시:
- "히알루론산 함유"
- "비타민 C 유도체 10,000ppm"
- "무파라벤, 무미네랄오일"
- "비건 인증"

근거 수준: 기본 (전성분표 확인)
```

#### 감성 클레임 (Sensory Claims)
사용감, 향, 질감 관련

```
예시:
- "산뜻한 사용감"
- "빠른 흡수"
- "촉촉한 마무리"
- "은은한 꽃향기"

근거 수준: 기본 (소비자 조사 또는 관능 평가)
```

## 2. 허용 클레임 vs 금지 클레임

### 화장품법상 허용 표현

#### 허용되는 효능 표현 (한국)

**기능성화장품 3개 카테고리**
```
1. 미백: "피부 미백에 도움을 줌"
2. 주름개선: "피부 주름 개선에 도움을 줌"
3. 자외선차단: "자외선으로부터 피부 보호"

+ 기타 인정된 기능성:
- 탈모 완화
- 여드름성 피부 완화
- 아토피 피부 보습
- 튼살 완화
```

**일반화장품 허용 표현**
```
- 피부의 건조함 개선
- 피부 보습
- 피부 정돈, 정화
- 피부 진정
- 피부 결 정돈
- 각질 제거
- 피부 윤기
- 모공 관리
```

### 금지 또는 주의 표현

#### 의약품 영역 침범 표현 (절대 금지)

```
금지 키워드:
- 치료, 완치, 예방
- 의약품, 약효
- 질병명 (아토피, 습진, 건선 등)
- 신체 기능 변화 (호르몬, 세포 재생 등)

금지 예시:
✗ "아토피를 치료합니다"
✗ "피부암을 예방합니다"
✗ "세포를 재생시킵니다"
✗ "호르몬 균형을 맞춥니다"
```

#### 과장/허위 표현 (금지)

```
절대적 표현:
✗ "100% 효과 보장"
✗ "완벽한 피부"
✗ "영원히 지속"
✗ "즉각적인 주름 제거"

근거 없는 비교:
✗ "세계 최고"
✗ "업계 유일"
✗ "비교 불가"
```

#### 비교 광고 주의사항

```
허용 조건:
- 객관적 비교 시험 근거 보유
- 비교 대상 명확히 특정
- 사실에 기반한 비교

금지:
✗ 경쟁사 비방
✗ 근거 없는 우위 주장
✗ 오해 소지 있는 비교
```

## 3. 국가별 클레임 규제

### 한국 (MFDS)

**규제 기관**: 식품의약품안전처 (MFDS)

**화장품 정의**:
"인체를 청결, 미화하여 매력을 더하고 용모를 밝게 변화시키거나
피부, 모발의 건강을 유지 또는 증진하기 위하여 인체에 바르고
문지르거나 뿌리는 등 이와 유사한 방법으로 사용되는 물품"

**클레임 규제**:
```
기능성화장품:
- 미백, 주름개선, 자외선차단 등
- MFDS 사전 심사 필요
- "기능성화장품" 표시 의무

일반화장품:
- 기능성 범위 외 클레임만 허용
- 허위/과장 광고 금지 (화장품법 제13조)
```

**주요 규정**:
- 화장품법
- 화장품 표시/광고 실증에 관한 규정
- 기능성화장품 기준 및 시험방법

---

### 미국 (FDA)

**규제 기관**: Food and Drug Administration (FDA)

**화장품 정의**:
"인체에 문지르거나, 붓거나, 뿌리거나, 스프레이 하거나 또는
기타 방법으로 적용하여 청결, 미화, 매력 증진, 외모 변화를
목적으로 하는 제품"

**클레임 규제**:
```
Drug Claims (금지):
- 신체 구조/기능 변화 주장
- 질병 치료/예방 주장
- 예: "주름 제거", "세포 재생", "콜라겐 증가"

Cosmetic Claims (허용):
- 외관 개선
- 일시적 효과
- 예: "주름 외관 개선", "더 젊어 보이는 피부"
```

**OTC Drug vs Cosmetic**:
```
선스크린: OTC Drug (FDA monograph 준수)
항노화 크림: Cosmetic (구조/기능 변화 주장 시 Drug)
여드름 치료: OTC Drug (Salicylic acid 등)
```

---

### EU (유럽연합)

**규제 기관**: European Commission

**화장품 정의** (EC 1223/2009):
"인체 외부 (표피, 모발, 손톱, 입술, 외부 생식기) 또는 치아와
구강 점막에 접촉하여 청결, 향기, 외관 변화, 보호, 좋은 상태
유지를 주 목적으로 하는 물질 또는 혼합물"

**클레임 규제 (Claims Regulation)**:
```
EU Claims Regulation (655/2013)
6가지 공통 기준:
1. 법적 준수 (Legal compliance)
2. 진실성 (Truthfulness)
3. 근거 기반 (Evidential support)
4. 정직성 (Honesty)
5. 공정성 (Fairness)
6. 정보에 입각한 선택 (Informed decision-making)
```

**허용/금지 예시**:
```
허용:
- "주름 외관 개선"
- "24시간 보습" (시험 근거 시)
- "90%가 더 부드러운 피부 체감"

금지:
- "치료", "치유"
- "의약품" 연상 표현
- 근거 없는 절대적 표현
```

---

### 중국 (NMPA)

**규제 기관**: National Medical Products Administration (NMPA)

**화장품 분류**:
```
특수용도화장품 (Special Use Cosmetics):
- 미백, 염모, 탈모방지 등 9개 카테고리
- NMPA 등록 필수 (동물시험 포함)

일반화장품 (General Cosmetics):
- 특수용도 외 제품
- 비안(备案) 신고제
```

**클레임 규제**:
```
금지 표현:
- 의료 효능 표현
- 절대적 표현 ("최고", "100%")
- 허위/과장 표현

필수 표시:
- 중문 라벨
- 전성분 표기
- 유통기한/제조일
```

---

### 일본 (MHLW)

**규제 기관**: Ministry of Health, Labour and Welfare

**화장품 분류**:
```
의약부외품 (Quasi-drug):
- 미백, 육모, 피지 분비 억제 등
- MHLW 승인 필요
- 허가된 유효성분 함유

화장품 (Cosmetics):
- 의약부외품 범위 외
- 신고제
```

**클레임 규제**:
```
허용 효능 표현 (56개 항목):
- 피부 보습
- 피부 유연화
- 피부 결 정돈
- 건조에 의한 소주름 억제 등

금지:
- 의약품적 효능
- 의약부외품 효능 (승인 없이)
```

## 4. 클레임-근거 매칭 가이드

### 클레임별 필요 근거 수준

| 클레임 유형 | 최소 근거 | 권장 근거 | 최적 근거 |
|------------|----------|----------|----------|
| 성분 함유 | 전성분표 | - | - |
| 기능적 | 성분 문헌 | In vitro | In vitro + In vivo |
| 효능 | In vitro | In vivo | In vitro + In vivo + HUT |
| 성능 (정량) | In vivo | In vivo (다기관) | In vivo + 논문 게재 |
| 비교 | 비교 In vivo | 비교 In vivo + HUT | 제3자 기관 비교 시험 |
| 소비자 인식 | HUT (n≥30) | HUT (n≥50) | HUT (n≥100) |

### 구체적 클레임-근거 매칭

#### 보습 클레임

| 클레임 | 필요 근거 |
|--------|----------|
| "보습에 도움" | 보습 성분 함유 확인 |
| "피부 수분 공급" | Corneometer 측정 (증가 확인) |
| "24시간 보습" | Corneometer 24시간 측정 |
| "수분량 30% 증가" | Corneometer p<0.05, 변화율 30% |
| "90%가 촉촉함 체감" | HUT n≥50, Top 2 Box 90% |

#### 주름개선 클레임

| 클레임 | 필요 근거 |
|--------|----------|
| "주름 개선 도움" | 기능성 고시 원료 함유 |
| "피부 탄력 개선" | Cutometer 측정, In vivo |
| "주름 깊이 20% 감소" | PRIMOS/Visiometer, p<0.05 |
| "8명 중 7명이 탄력 개선 체감" | HUT n≥50, Top 2 Box 87.5% |

#### 미백 클레임

| 클레임 | 필요 근거 |
|--------|----------|
| "미백에 도움" | 기능성 고시 원료 함유 |
| "피부톤 밝아짐" | Mexameter/Chromameter In vivo |
| "멜라닌 지수 감소" | Mexameter MI 감소, p<0.05 |
| "사용자 85%가 밝아짐 체감" | HUT n≥50, Top 2 Box 85% |

### 근거 자료 보관

**보관 기간**:
- 한국: 최종 판매일로부터 3년
- EU: 클레임 사용 기간 동안
- 미국: 클레임 사용 기간 동안 (FTC 요청 대비)

**보관 자료**:
```
- 시험 프로토콜
- 원데이터 (Raw data)
- 통계 분석 결과
- 최종 보고서
- 시험 기관 인증서
- IRB 승인서 (해당 시)
```

## 5. 클레임 작성 실무 가이드

### 클레임 개발 프로세스

```
1. 마케팅 목표 클레임 설정
   ↓
2. 규제 적합성 1차 검토
   ↓
3. 근거 자료 확보 가능성 평가
   ↓
4. 시험 계획 수립 및 수행
   ↓
5. 결과 분석 및 클레임 문구 작성
   ↓
6. 규제/법무 2차 검토
   ↓
7. 최종 클레임 확정
```

### 클레임 문구 작성 템플릿

**기기 시험 결과 기반**
```
[측정 항목]이 [변화율/변화량] [개선/증가/감소]되었습니다*
*[시험 유형], n=[피험자수], [사용기간]

예:
피부 수분량이 30% 증가했습니다*
*인체적용시험 결과, n=30, 4주 사용
```

**소비자 조사 결과 기반**
```
[비율]%가 [효과 표현]을 [체감/동의]했습니다*
*[조사 유형], n=[응답자수], [사용기간]

예:
사용자 90%가 피부가 촉촉해졌다고 느꼈습니다*
*자사 소비자 조사, n=100, 4주 사용
```

### 주석 표기 가이드

**필수 정보**:
- 시험/조사 유형
- 피험자/응답자 수
- 시험/사용 기간

**선택 정보**:
- 시험 기관
- 시험 시점
- 피험자 특성

**표기 위치**:
- 클레임 직후 별표(*)
- 페이지 하단 또는 별도 영역에 상세 설명
