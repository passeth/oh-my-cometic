# Claim Review Checklist

화장품 클레임 검토를 위한 체크리스트입니다.

## 1. 근거 자료 확보 여부

### 시험 근거

- [ ] 클레임 유형에 적합한 시험 방법 선택
- [ ] 시험 프로토콜 문서화
- [ ] IRB 승인 (인체적용시험의 경우)
- [ ] 충분한 피험자/표본 수 확보 (통계적 유의성)
- [ ] 적절한 시험 기간 (클레임에 맞는 기간)
- [ ] 원데이터(Raw data) 보유
- [ ] 통계 분석 완료 (p-value < 0.05)
- [ ] 최종 시험 보고서 확보
- [ ] 시험 기관 신뢰성 (GLP, 인증 기관 등)

### 근거 수준

- [ ] In vitro 시험 (해당 시)
- [ ] Ex vivo 시험 (해당 시)
- [ ] In vivo 기기 측정 (해당 시)
- [ ] 전문가 육안 평가 (해당 시)
- [ ] 소비자 조사/HUT (해당 시)
- [ ] 문헌 근거 (성분 효능)
- [ ] 원료사 제공 자료 (해당 시)

### 근거 자료 보관

- [ ] 모든 근거 자료 3년 이상 보관 계획
- [ ] 디지털 및 물리적 백업 확보
- [ ] 접근 및 검색 용이성 확보

---

## 2. 규제 적합성

### 국내 규제 (한국)

- [ ] 화장품법 제13조 (광고 규정) 준수
- [ ] 기능성화장품 해당 여부 확인
- [ ] 기능성 해당 시 MFDS 심사/신고 완료
- [ ] 화장품 표시광고 실증에 관한 규정 준수
- [ ] 의약품 영역 침범 표현 없음

### 수출국 규제

#### 미국
- [ ] FDA cosmetic 범위 내 클레임
- [ ] Drug claim 해당 없음
- [ ] FTC 광고 규정 준수

#### EU
- [ ] EC 1223/2009 준수
- [ ] Claims Regulation (655/2013) 6개 기준 충족
- [ ] 금지 클레임 목록 확인

#### 중국
- [ ] 특수용도화장품 해당 여부 확인
- [ ] NMPA 등록/비안 완료 (해당 시)
- [ ] 중국어 클레임 표현 검토

#### 일본
- [ ] 의약부외품 해당 여부 확인
- [ ] 56개 허용 효능 범위 내 확인

### 규제 문서화

- [ ] 규제 검토 기록 보관
- [ ] 법무/RA 팀 검토 완료
- [ ] 수출국별 클레임 버전 관리

---

## 3. 과장 표현 여부

### 절대적 표현 점검

- [ ] "100%", "완벽한", "확실한" 등 제거
- [ ] "최고", "최초", "유일" 등 제거 (근거 없는 경우)
- [ ] "영원히", "즉각적으로" 등 과장 제거
- [ ] "모든", "누구나" 등 일반화 표현 제거

### 의약품 연상 표현 점검

- [ ] "치료", "완치", "예방" 제거
- [ ] "약", "의약품", "효능" 제거
- [ ] 질병명 언급 제거 (아토피, 건선 등)
- [ ] 신체 기능 변화 표현 제거

### 비교 표현 점검

- [ ] 비교 대상 명확히 특정
- [ ] 비교 시험 근거 확보
- [ ] 경쟁사 비방 표현 제거
- [ ] 오해 소지 있는 비교 제거

### 정량적 표현 점검

- [ ] 수치 근거 명확 (시험 결과 기반)
- [ ] 반올림 기준 명확
- [ ] 통계적 유의성 확보 (p < 0.05)
- [ ] 측정 방법 및 조건 명시 가능

---

## 4. 소비자 오인 가능성

### 표현 명확성

- [ ] 일반 소비자가 이해할 수 있는 표현
- [ ] 전문 용어 최소화 또는 설명 추가
- [ ] 이중적 해석 가능성 제거
- [ ] 조건부 효과 명확히 표시

### 기대 수준 관리

- [ ] 과도한 기대 유발 표현 제거
- [ ] 개인차 있을 수 있음 안내 (필요시)
- [ ] 사용 조건 명확히 안내

### 주석 및 부가 설명

- [ ] 근거 출처 주석 표기
- [ ] 시험 조건 명시 (n=, 기간 등)
- [ ] 주석 크기 및 위치 가독성 확보
- [ ] 별표(*) 및 참조 명확

### 시각적 표현

- [ ] 과장된 Before/After 이미지 없음
- [ ] 이미지 보정 적정 수준
- [ ] 모델 사진과 실제 효과 괴리 없음
- [ ] 그래프/차트 왜곡 없음

---

## 5. 최종 확인

### 내부 검토

- [ ] 마케팅팀 검토 완료
- [ ] 연구개발팀 근거 확인
- [ ] 규제팀(RA) 검토 완료
- [ ] 법무팀 검토 완료 (필요시)
- [ ] 경영진 최종 승인

### 문서화

- [ ] 클레임 버전 관리
- [ ] 검토 이력 기록
- [ ] 승인 문서 보관
- [ ] 근거 자료 연결 확인

### 모니터링 계획

- [ ] 클레임 관련 소비자 피드백 모니터링
- [ ] 규제 변경 사항 추적
- [ ] 정기적 클레임 재검토 일정

---

## 클레임별 체크리스트 요약

### 기능적 클레임
```
[ ] 성분 특성 확인
[ ] 규제 허용 범위 내
[ ] 과장 표현 없음
```

### 효능 클레임
```
[ ] In vitro 또는 In vivo 근거
[ ] 통계적 유의성 확보
[ ] 기능성화장품 해당 여부 확인
[ ] 규제 적합 문구
```

### 성능 클레임 (정량적)
```
[ ] In vivo 기기 측정 근거
[ ] 정확한 수치 및 조건 명시
[ ] 주석 표기 완료
[ ] 재현 가능한 결과
```

### 비교 클레임
```
[ ] 동일 조건 비교 시험
[ ] 비교 대상 명확화
[ ] 법적 리스크 검토
[ ] 공정거래법 준수
```

### 소비자 인식 클레임
```
[ ] HUT 또는 소비자 조사 근거
[ ] 통계적 유의성 (n≥30, 권장 50-100)
[ ] Top 2 Box 기준 명확
[ ] 주석 완료 (조사 유형, n=, 기간)
```

---

## 위험 수준별 조치

### 높은 위험 (즉시 조치)
```
- 의약품 영역 침범 표현
- 근거 없는 정량적 클레임
- 허위 비교 클레임
→ 조치: 즉시 수정 또는 삭제
```

### 중간 위험 (검토 후 조치)
```
- 과장 가능성 있는 표현
- 주석 누락
- 오해 소지 있는 비교
→ 조치: 문구 수정 및 주석 추가
```

### 낮은 위험 (개선 권고)
```
- 더 명확한 표현 가능
- 추가 근거로 강화 가능
- 소비자 이해도 개선 가능
→ 조치: 선택적 개선
```
