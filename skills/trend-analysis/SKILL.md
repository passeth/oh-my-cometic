## Overview

화장품 산업의 트렌드를 체계적으로 분석하고, 이를 제품 기획에 적용하는 방법론을 제공합니다.
트렌드의 식별, 검증, 수명 주기 분석, 그리고 실제 제품 컨셉으로의 전환까지 전 과정을 다룹니다.

## When to Use

이 스킬은 다음 상황에서 활용하세요:

1. **신제품 기획 초기 단계** - 시장 기회 탐색 및 컨셉 방향 설정
2. **시장 조사** - 현재 시장 동향 파악 및 미래 예측
3. **경쟁 분석** - 경쟁사 제품 트렌드 분석 및 차별화 포인트 발굴
4. **포트폴리오 전략** - 기존 제품 라인업 재검토 및 갭 분석
5. **성분 선정** - 트렌드 성분 식별 및 적용 가능성 평가

## Core Capabilities

### 1. 트렌드 프레임워크

#### Macro Trends (거시 트렌드)
사회, 문화, 기술, 환경 변화가 뷰티 시장에 미치는 영향

| 카테고리 | 트렌드 예시 | 뷰티 영향 |
|---------|-----------|----------|
| **사회** | 고령화, 1인 가구 | 안티에이징, 소용량 제품 |
| **문화** | 젠더 뉴트럴, 다양성 | 유니섹스 라인, 다양한 쉐이드 |
| **기술** | AI, IoT, 3D 프린팅 | 맞춤형 화장품, 스마트 디바이스 |
| **환경** | 기후변화, 지속가능성 | 클린뷰티, 리필 시스템 |
| **건강** | 웰니스, 홀리스틱 | 이너뷰티, 기능성 화장품 |

#### Category Trends (카테고리 트렌드)

```
┌─────────────────────────────────────────────────────────────┐
│                    카테고리별 핵심 트렌드                      │
├─────────────────────────────────────────────────────────────┤
│ 스킨케어  │ 스킨 배리어, 마이크로바이옴, 하이브리드 제형         │
│ 메이크업  │ 스킨케어 포뮬러, No-메이크업 메이크업, 다기능        │
│ 헤어케어  │ 스칼프 케어, 본딩 기술, 헤어 웰니스                 │
│ 바디케어  │ 페이셜라이제이션, 향기 웰빙, 타겟 케어              │
│ 선케어   │ 광범위 보호, 환경 프렌들리, 하이브리드               │
└─────────────────────────────────────────────────────────────┘
```

#### Ingredient Trends (성분 트렌드)
> 자세한 내용은 `references/ingredient_trends.md` 참조

- **주목 성분**: 마이크로바이옴, 어댑토젠, 펩타이드, 발효 성분
- **지속가능 성분**: 업사이클, 바이오테크 유래
- **기능성 성분**: 엑소좀, 성장인자, CBD 유도체

### 2. 트렌드 수명 주기 (Trend Lifecycle)

```
 Impact
   ^
   │                    ╭──────╮
   │                   ╱        ╲
   │                  ╱          ╲
   │        ╭───────╮            ╲
   │       ╱                       ╲
   │      ╱                         ╲──────
   │     ╱
   │────╱
   └─────────────────────────────────────────> Time
     Emerging  Growing   Mature   Declining

   [혁신가]  [얼리어답터] [대중화]  [포화/쇠퇴]
```

#### 단계별 특성 및 전략

| 단계 | 특성 | 제품 전략 | 리스크 |
|-----|------|----------|--------|
| **Emerging (출현)** | 전문 채널, 니치 브랜드 | 선도적 포지셔닝, R&D 투자 | 높음 (실패 가능) |
| **Growing (성장)** | 미디어 주목, 대형 브랜드 진입 | 빠른 출시, 차별화 | 중간 (경쟁 심화) |
| **Mature (성숙)** | 대중화, 가격 경쟁 | 프리미엄화 or 가성비 | 낮음 (마진 압박) |
| **Declining (쇠퇴)** | 신규 트렌드로 대체 | 재포지셔닝 or 철수 | 낮음 (기회비용) |

### 3. 지역별 트렌드 차이

#### K-Beauty (한국)
- **키워드**: 혁신, 트렌드세터, 멀티스텝
- **특징**: 빠른 신제품 출시, 독특한 제형, SNS 드리븐
- **주요 트렌드**: 글래스 스킨, 스킨 배리어, 7스킨 법칙, 시카
- **글로벌 영향**: 세계 트렌드 리딩, 성분 혁신

#### J-Beauty (일본)
- **키워드**: 정교함, 미니멀리즘, 기술력
- **특징**: 장인 정신, 심플한 루틴, 고기능성
- **주요 트렌드**: 더블 클렌징, 에센스, 발효 성분
- **글로벌 영향**: 프리미엄 세그먼트, 기술 신뢰성

#### C-Beauty (중국)
- **키워드**: 전통 현대화, 로컬 파워, 디지털
- **특징**: 한방 성분, 라이브 커머스, 국수주의
- **주요 트렌드**: TCM 성분, 국조(궈차오), 캐릭터 콜라보
- **글로벌 영향**: 아시아 시장 확대, 성분 다양화

#### Western Trends (서양)
- **키워드**: 클린뷰티, 인클루시브, 과학적 증거
- **특징**: 성분 투명성, 다양성 존중, 임상 데이터 중시
- **주요 트렌드**: 클린 포뮬러, 쉐이드 다양화, 더마 브랜드
- **글로벌 영향**: 규제 리딩, 지속가능성 기준


### 5. 트렌드 → 제품 컨셉 전환

#### 전환 프로세스

```
[트렌드 식별] → [트렌드 검증] → [기회 평가] → [컨셉 개발] → [제품화]
     │              │              │              │
     ▼              ▼              ▼              ▼
  다중 소스       소비자 조사    브랜드 적합성   포지셔닝
  크로스체크      전문가 인터뷰   역량 평가       차별화
                 데이터 분석    경쟁 분석       스토리텔링
```

#### 트렌드 적용 체크리스트

```markdown
## 트렌드 적용 평가 체크리스트

### 1. 트렌드 적합성
- [ ] 타겟 고객의 니즈와 부합하는가?
- [ ] 브랜드 아이덴티티와 일관성이 있는가?
- [ ] 현재 포트폴리오와 시너지가 있는가?

### 2. 시장 기회
- [ ] 트렌드의 현재 수명 주기 단계는?
- [ ] 시장 규모와 성장 가능성은?
- [ ] 경쟁 강도와 진입 장벽은?

### 3. 실행 가능성
- [ ] 필요한 기술/성분 확보 가능한가?
- [ ] 규제 요건을 충족하는가?
- [ ] 예산과 타임라인 내 가능한가?

### 4. 차별화
- [ ] 경쟁 제품 대비 차별점은?
- [ ] 클레임/스토리 요소는?
- [ ] 가격 포지셔닝은?
```

#### 트렌드 컨셉 개발 템플릿

```markdown
- Key Ingredients: [핵심 성분]
```

## Usage Examples

### Example 1: 트렌드 분석 요청

**입력:**
"2025년 스킨케어 시장의 주요 트렌드를 분석해주세요."

**분석 접근:**
1. 거시 트렌드 영향 분석 (환경, 기술, 사회)
2. 성분 트렌드 현황 및 수명 주기
3. 패키징/포뮬레이션 트렌드
4. 지역별 차별점
5. 제품 기획 시사점

### Example 2: 트렌드 기반 제품 기획

**입력:**
"마이크로바이옴 트렌드를 활용한 제품 컨셉을 제안해주세요."

**분석 접근:**
1. 마이크로바이옴 트렌드 수명 주기 분석
2. 경쟁 제품 현황
3. 차별화 가능 영역 식별
4. 타겟 소비자 정의
5. 컨셉 및 포지셔닝 제안

## Best Practices

1. **다중 소스 검증**: 단일 소스에 의존하지 말고, 다양한 채널에서 트렌드 신호 확인
2. **수명 주기 고려**: 트렌드의 현재 위치와 예상 지속 기간을 고려한 타이밍 결정
3. **브랜드 적합성**: 트렌드 자체보다 브랜드 DNA와의 적합성 우선 평가
4. **실행 가능성**: 아이디어 단계에서 기술적, 규제적 실현 가능성 검토
5. **지속적 모니터링**: 트렌드는 진화하므로 정기적인 리뷰 및 업데이트 필수

## Limitations

- 트렌드 예측은 본질적으로 불확실성을 포함
- 시장 데이터는 공개 정보에 기반하며 최신 유료 리포트와 차이 가능
- 지역별 규제 환경에 따른 트렌드 적용 가능성 변동
