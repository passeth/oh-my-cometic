# Concentration Units Reference

화장품 및 화학 분야에서 사용되는 농도 단위 상세 정의 및 변환 공식

## 1. 퍼센트 농도 (Percent Concentration)

### 1.1 중량 퍼센트 % (w/w)

**정의**: 용질의 질량 / 용액의 총 질량 × 100

```
% (w/w) = (m_solute / m_solution) × 100
        = (m_solute / (m_solute + m_solvent)) × 100
```

**특징**:
- 화장품 배합에서 표준 표기법
- 온도에 따른 변화 없음 (질량 기준)
- 계량이 용이 (저울 사용)

**예시**:
```
나이아신아마이드 5g + 정제수 95g = 100g 용액
% (w/w) = (5 / 100) × 100 = 5%
```

### 1.2 질량/부피 퍼센트 % (w/v)

**정의**: 용질의 질량(g) / 용액의 부피(mL) × 100

```
% (w/v) = (g_solute / mL_solution) × 100
```

**특징**:
- 의약품, 주사제에서 표준 표기법
- 온도에 따라 값 변화 (부피 변화)
- 계량 시 부피 측정 필요

**예시**:
```
나이아신아마이드 5g을 정제수에 녹여 총 100mL
% (w/v) = (5 / 100) × 100 = 5%
```

### 1.3 부피 퍼센트 % (v/v)

**정의**: 용질의 부피 / 용액의 총 부피 × 100

```
% (v/v) = (V_solute / V_solution) × 100
```

**특징**:
- 액체/액체 혼합물에 사용
- 알코올, 향료, 용매 표기
- 혼합 시 부피 변화 주의 (비이상 용액)

**예시**:
```
에탄올 70mL + 정제수 30mL = ~97mL (실제 부피)
% (v/v) = (70 / 100) × 100 = 70% (혼합 전 부피 기준)
```

### 1.4 퍼센트 단위 간 변환

```python
# w/w ↔ w/v (밀도 d g/mL)
% (w/v) = % (w/w) × d_solution
% (w/w) = % (w/v) / d_solution

# w/w ↔ v/v (용질 밀도 d_solute, 용액 밀도 d_solution)
% (w/w) = % (v/v) × (d_solute / d_solution)
% (v/v) = % (w/w) × (d_solution / d_solute)

# w/v ↔ v/v
% (v/v) = % (w/v) / d_solute
% (w/v) = % (v/v) × d_solute
```

## 2. ppm / ppb / ppt

### 2.1 ppm (Parts Per Million)

**정의**: 백만분율 (10^-6)

```
1 ppm = 1 part per 1,000,000 parts
      = 1 mg/kg (고체/고체)
      = 1 mg/L (액체, d ≈ 1)
      = 1 μg/g
      = 0.0001%
```

**화장품에서의 용도**:
- 중금속 함량 (납, 비소, 카드뮴 등)
- 미량 불순물 (1,4-다이옥산, 니트로사민)
- 미량 첨가제

### 2.2 ppb (Parts Per Billion)

**정의**: 십억분율 (10^-9)

```
1 ppb = 1 part per 1,000,000,000 parts
      = 1 μg/kg
      = 1 ng/g
      = 0.0000001%
      = 0.001 ppm
```

**화장품에서의 용도**:
- 극미량 오염물질
- 민감한 분석 결과 표기
- 환경 규제 물질

### 2.3 ppt (Parts Per Trillion)

**정의**: 조분율 (10^-12)

```
1 ppt = 1 part per 1,000,000,000,000 parts
      = 1 ng/kg
      = 1 pg/g
      = 0.001 ppb
```

### 2.4 변환 관계

```
1% = 10,000 ppm = 10,000,000 ppb = 10,000,000,000 ppt
1 ppm = 1,000 ppb = 1,000,000 ppt
1 ppb = 1,000 ppt
```

## 3. 질량/질량 농도

### 3.1 mg/g (밀리그램/그램)

```
1 mg/g = 0.001 g/g = 0.1% = 1000 ppm
```

**용도**: 고체 원료 스펙, 추출물 유효 성분 함량

### 3.2 mg/kg (밀리그램/킬로그램)

```
1 mg/kg = 1 ppm = 0.0001%
```

**용도**: 규제 기준, 분석 결과

### 3.3 μg/g (마이크로그램/그램)

```
1 μg/g = 1 ppm = 1 mg/kg
```

### 3.4 ng/g (나노그램/그램)

```
1 ng/g = 1 ppb = 0.001 ppm
```

## 4. 질량/부피 농도

### 4.1 mg/mL (밀리그램/밀리리터)

```
1 mg/mL = 1 g/L = 0.1% (w/v) = 1000 ppm (밀도 1 기준)
```

**용도**: 용액 스펙, 의약품

### 4.2 g/L (그램/리터)

```
1 g/L = 1 mg/mL = 0.1% (w/v)
```

### 4.3 μg/mL (마이크로그램/밀리리터)

```
1 μg/mL = 1 mg/L = 1 ppm (밀도 1 기준)
```

### 4.4 ng/mL (나노그램/밀리리터)

```
1 ng/mL = 1 μg/L = 1 ppb (밀도 1 기준)
```

## 5. 몰농도 (Molar Concentration)

### 5.1 몰 농도 M (Molarity)

**정의**: 용액 1L 당 용질의 몰수

```
M (mol/L) = n_solute (mol) / V_solution (L)
          = m_solute (g) / (MW × V_solution (L))
```

**특징**:
- 화학 반응 계산에 유용
- 온도에 따라 값 변화 (부피 변화)
- 화학양론 계산의 기본 단위

### 5.2 mM (Millimolar)

```
1 mM = 0.001 M = 1 mmol/L = 10^-3 mol/L
```

### 5.3 μM (Micromolar)

```
1 μM = 0.001 mM = 0.000001 M = 10^-6 mol/L
```

### 5.4 nM (Nanomolar)

```
1 nM = 0.001 μM = 10^-9 mol/L
```

### 5.5 질량 농도 ↔ 몰농도 변환

```python
# 몰농도 → 질량 농도
g/L = M × MW
mg/mL = M × MW
% (w/v) = M × MW / 10

# 질량 농도 → 몰농도
M = (g/L) / MW
M = (mg/mL) / MW
M = (% w/v × 10) / MW
```

## 6. 기타 농도 단위

### 6.1 몰랄 농도 m (Molality)

**정의**: 용매 1kg 당 용질의 몰수

```
m (mol/kg) = n_solute (mol) / m_solvent (kg)
```

**특징**:
- 온도에 무관 (질량 기준)
- 끓는점 오름, 어는점 내림 계산에 사용
- 화장품에서는 거의 사용하지 않음

### 6.2 몰분율 (Mole Fraction)

**정의**: 특정 성분의 몰수 / 전체 몰수

```
χ_A = n_A / Σn_i
```

**특징**:
- 단위 없음 (무차원)
- 증기압 계산에 사용

### 6.3 노르말 농도 N (Normality)

**정의**: 용액 1L 당 당량수

```
N (eq/L) = equivalents / V_solution (L)
         = M × 당량수
```

**특징**:
- 산-염기 적정에 사용
- 산화-환원 반응 계산
- 현재는 몰농도 사용 권장

### 6.4 IU (International Unit)

**정의**: 국제 단위, 물질마다 다른 기준

```
비타민 A (레티놀): 1 IU = 0.3 μg
비타민 D: 1 IU = 0.025 μg
비타민 E (토코페롤): 1 IU = 0.67 mg (d-α-토코페롤)
                           = 0.45 mg (dl-α-토코페롤 아세테이트)
```

**화장품에서의 용도**:
- 레티놀 농도 표기 (IU/g)
- 비타민 성분 함량

## 7. 변환 마스터 테이블

### 7.1 퍼센트 기준 변환

| From | To % | Formula |
|------|------|---------|
| ppm | % | ÷ 10,000 |
| ppb | % | ÷ 10,000,000 |
| mg/g | % (w/w) | ÷ 10 |
| mg/kg | % | ÷ 10,000 |
| mg/mL | % (w/v) | ÷ 10 |
| g/L | % (w/v) | ÷ 10 |
| M | % (w/v) | × MW ÷ 10 |

### 7.2 ppm 기준 변환

| From | To ppm | Formula |
|------|--------|---------|
| % | ppm | × 10,000 |
| ppb | ppm | ÷ 1,000 |
| mg/g | ppm | × 1,000 |
| mg/kg | ppm | × 1 (동일) |
| μg/g | ppm | × 1 (동일) |
| mg/L | ppm | × 1 (밀도≈1) |

### 7.3 mg/mL 기준 변환

| From | To mg/mL | Formula |
|------|----------|---------|
| % (w/v) | mg/mL | × 10 |
| g/L | mg/mL | × 1 (동일) |
| μg/mL | mg/mL | ÷ 1,000 |
| M | mg/mL | × MW |
| mM | mg/mL | × MW ÷ 1,000 |

### 7.4 몰농도 기준 변환

| From | To M | Formula |
|------|------|---------|
| mM | M | ÷ 1,000 |
| μM | M | ÷ 1,000,000 |
| g/L | M | ÷ MW |
| mg/mL | M | ÷ MW |
| % (w/v) | M | × 10 ÷ MW |

## 9. 단위 표기 규칙

### 9.1 SI 단위 접두사

| 접두사 | 기호 | 배수 |
|--------|------|------|
| 테라 (tera) | T | 10^12 |
| 기가 (giga) | G | 10^9 |
| 메가 (mega) | M | 10^6 |
| 킬로 (kilo) | k | 10^3 |
| 밀리 (milli) | m | 10^-3 |
| 마이크로 (micro) | μ | 10^-6 |
| 나노 (nano) | n | 10^-9 |
| 피코 (pico) | p | 10^-12 |

### 9.2 올바른 표기

```
올바름: 5% (w/w), 100 ppm, 1.5 mg/mL, 0.1 M
틀림: 5 %(w/w), 100ppm, 1.5mg/mL, 0.1M

# 숫자와 단위 사이에 공백
# 단위 내 슬래시(/) 앞뒤는 공백 없음
```

### 9.3 유효 숫자

```
- 규칙: 가장 정밀도가 낮은 측정값에 맞춤
- 예시: 5.00% × 0.99 = 4.95% (3자리)
- ppm: 일반적으로 정수 또는 소수점 1자리
- %: 소수점 2-3자리
```
