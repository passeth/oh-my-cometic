# 연구 유형 분류

화장품 성분 효능 평가를 위한 연구 유형 분류 및 특성

## 1. 연구 유형 계층 구조

```
근거 피라미드 (Evidence Pyramid)
         ▲
        /  \
       / MA \     ← Meta-Analysis (메타 분석)
      /------\
     /   SR   \   ← Systematic Review (체계적 고찰)
    /----------\
   /    RCT     \  ← Randomized Controlled Trial
  /--------------\
 /  Cohort/CC    \ ← Observational Studies
/------------------\
    Case / In Vivo    ← Case Reports, Animal Studies
----------------------
    In Vitro / Ex Vivo ← Lab Studies
```

## 2. 임상 연구 (Clinical Studies)

### 2.1 Meta-Analysis (메타 분석)

| 항목 | 설명 |
|-----|------|
| **정의** | 여러 연구 결과를 통계적으로 종합 |
| **근거 수준** | 최고 (Level 1a) |
| **장점** | 검정력 증가, 효과 크기 정밀 추정 |
| **단점** | 이질성 문제, 출판 비뚤림 |
| **적용** | 성분 효능의 종합적 평가 |

**식별 키워드**:
- "meta-analysis"
- "pooled analysis"
- "quantitative synthesis"

### 2.2 Systematic Review (체계적 고찰)

| 항목 | 설명 |
|-----|------|
| **정의** | 체계적 방법으로 문헌 검색 및 평가 |
| **근거 수준** | 최고 (Level 1a) |
| **장점** | 편향 최소화, 재현 가능 |
| **단점** | 시간/비용 소요 |
| **적용** | 성분의 전반적 근거 검토 |

**식별 키워드**:
- "systematic review"
- "PRISMA"
- "literature review" + comprehensive search

### 2.3 Randomized Controlled Trial (RCT)

| 항목 | 설명 |
|-----|------|
| **정의** | 무작위 배정, 대조군 비교 |
| **근거 수준** | 높음 (Level 1b) |
| **장점** | 인과관계 추론 가능 |
| **단점** | 비용 높음, 윤리적 제약 |
| **적용** | 효능 입증의 금본위 |

**하위 분류**:

| 유형 | 설명 | 맹검 수준 |
|-----|------|----------|
| **Double-blind RCT** | 피험자, 연구자 모두 맹검 | 최고 |
| **Single-blind RCT** | 피험자만 맹검 | 높음 |
| **Open-label RCT** | 맹검 없음 | 중간 |
| **Split-face RCT** | 한 피험자의 좌우 비교 | 높음 |

**식별 키워드**:
- "randomized"
- "double-blind"
- "placebo-controlled"
- "split-face"

### 2.4 Controlled Trial (대조 임상시험)

| 항목 | 설명 |
|-----|------|
| **정의** | 대조군 있으나 무작위 아님 |
| **근거 수준** | 중상 (Level 2a) |
| **장점** | 실행 용이 |
| **단점** | 선택 비뚤림 가능 |
| **적용** | 탐색적 효능 평가 |

**식별 키워드**:
- "controlled trial" (without randomized)
- "comparative study"
- "vehicle-controlled"

### 2.5 Cohort Study (코호트 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 노출군/비노출군 추적 관찰 |
| **근거 수준** | 중간 (Level 2b) |
| **장점** | 자연 경과 관찰 가능 |
| **단점** | 교란 변수 통제 어려움 |
| **적용** | 장기 안전성, 사용 패턴 |

**식별 키워드**:
- "cohort"
- "prospective"
- "follow-up study"
- "longitudinal"

### 2.6 Case-Control Study (환자-대조군 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 질환 유무에 따른 노출 비교 |
| **근거 수준** | 중간 (Level 3a) |
| **장점** | 드문 결과 연구에 유리 |
| **단점** | 회상 비뚤림 |
| **적용** | 이상반응 원인 탐색 |

**식별 키워드**:
- "case-control"
- "retrospective"

### 2.7 Cross-sectional Study (단면 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 특정 시점 상태 측정 |
| **근거 수준** | 낮음 (Level 3b) |
| **장점** | 빠르고 경제적 |
| **단점** | 인과관계 파악 불가 |
| **적용** | 유병률, 사용 현황 조사 |

**식별 키워드**:
- "cross-sectional"
- "survey"
- "prevalence study"

### 2.8 Case Series / Case Report (증례 보고)

| 항목 | 설명 |
|-----|------|
| **정의** | 개별 또는 소수 사례 보고 |
| **근거 수준** | 매우 낮음 (Level 4) |
| **장점** | 새로운 발견 제시 |
| **단점** | 일반화 불가 |
| **적용** | 이상반응, 특이 반응 보고 |

**식별 키워드**:
- "case report"
- "case series"
- "case study"

## 3. 전임상 연구 (Preclinical Studies)

### 3.1 In Vivo Study (동물 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 살아있는 동물에서 실험 |
| **모델** | 무모 마우스, Guinea pig, 돼지 |
| **적용** | 효능 스크리닝, 안전성 평가 |

**주요 모델**:

| 모델 | 적용 효능 | 특징 |
|-----|----------|------|
| **Hairless mouse** | 피부 노화, 광손상 | UV 손상 모델 |
| **Guinea pig** | 미백 | 멜라닌 생성 모델 |
| **Pig skin** | 투과, 자극 | 인체 피부와 유사 |
| **SKH-1 mouse** | 광노화 | UV 유도 주름 |

**식별 키워드**:
- "in vivo"
- "animal study"
- "mouse model"
- "hairless mice"

### 3.2 In Vitro Study (세포 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 세포 배양 수준 실험 |
| **모델** | 각질형성세포, 멜라노사이트, 섬유아세포 |
| **적용** | 기전 연구, 초기 스크리닝 |

**주요 세포주**:

| 세포주 | 기원 | 적용 |
|-------|------|------|
| **HaCaT** | 인체 각질형성세포 | 장벽, 분화 |
| **B16F10** | 마우스 흑색종 | 멜라닌 생성 억제 |
| **Normal melanocyte** | 인체 멜라노사이트 | 미백 기전 |
| **HDFa / HDFn** | 인체 섬유아세포 | 콜라겐 합성 |
| **HDF-UV** | UV 조사 섬유아세포 | 광노화 |

**식별 키워드**:
- "in vitro"
- "cell culture"
- "HaCaT"
- "melanocyte"
- "fibroblast"

### 3.3 Ex Vivo Study (조직 연구)

| 항목 | 설명 |
|-----|------|
| **정의** | 생체 외 조직에서 실험 |
| **모델** | 피부 생검 조직, 각질층 |
| **적용** | 투과, 대사, 국소 효과 |

**주요 모델**:

| 모델 | 유래 | 적용 |
|-----|------|------|
| **Franz cell** | 적출 피부 | 경피 투과 |
| **Tape stripping** | 각질층 | 각질 분석 |
| **Skin explant** | 피부 생검 | 국소 효과 |

**식별 키워드**:
- "ex vivo"
- "skin explant"
- "Franz diffusion cell"
- "tape stripping"

## 4. 화장품 특화 연구 유형

### 4.1 사용 시험 (Use Test)

| 유형 | 설명 | 평가 항목 |
|-----|------|----------|
| **HRIPT** | 반복 첩포 자극 | 자극성, 감작성 |
| **SIOPT** | 단회 첩포 | 급성 자극 |
| **Phototoxicity test** | 광독성 | UV 하 자극 |
| **Photosensitization** | 광감작 | UV 하 알레르기 |
| **Eye irritation** | 점막 자극 | 안자극 |

### 4.2 효능 평가 시험

| 시험 유형 | 측정 항목 | 장비/방법 |
|----------|----------|----------|
| **보습 평가** | 수분량, TEWL | Corneometer, Tewameter |
| **미백 평가** | 멜라닌, 피부톤 | Mexameter, Chromameter |
| **항노화 평가** | 주름, 탄력 | Replica, Cutometer |
| **진정 평가** | 홍반, 민감도 | Mexameter, VAS |

### 4.3 소비자 인식 조사

| 유형 | 방법 | 평가 항목 |
|-----|------|----------|
| **자가 평가** | 설문 | 주관적 효능 인식 |
| **선호도 조사** | 비교 설문 | 사용감, 기호도 |
| **패널 테스트** | 전문가 평가 | 관능 평가 |

## 5. 연구 유형별 적용 가이드

### 5.1 효능별 권장 연구 유형

| 효능 | 최적 연구 유형 | 이유 |
|-----|---------------|------|
| **미백** | Split-face RCT | 개인차 최소화 |
| **항노화** | DB-RCT, 12주 이상 | 장기 효과 필요 |
| **보습** | Crossover RCT | 빠른 효과 확인 |
| **진정** | RCT + 유발 시험 | 자극 후 효과 측정 |
| **항산화** | In vitro + 바이오마커 | 직접 측정 어려움 |

### 5.2 근거 구축 전략

```
단계별 근거 구축:

1단계: In vitro 스크리닝
   ↓ 기전 확인
2단계: Ex vivo / In vivo
   ↓ 효능 가능성 확인
3단계: 탐색적 임상 (n=20-30)
   ↓ 용량, 기간 결정
4단계: 확증적 RCT (n≥50)
   ↓ 효능 입증
5단계: 장기/대규모 연구
   ↓ 안전성, 재현성
```

## 6. 연구 품질 평가 체크리스트

### 6.1 임상 연구 체크리스트

| 항목 | 평가 질문 |
|-----|----------|
| **대상자** | 포함/제외 기준이 명확한가? |
| **무작위화** | 적절한 무작위화 방법인가? |
| **맹검** | 적절한 맹검이 적용되었는가? |
| **대조군** | 적절한 대조군이 설정되었는가? |
| **결과 측정** | 객관적 측정 방법인가? |
| **탈락** | 탈락률과 사유가 보고되었는가? |
| **분석** | ITT 분석이 수행되었는가? |
| **이해충돌** | 이해충돌이 명시되었는가? |

### 6.2 전임상 연구 체크리스트

| 항목 | 평가 질문 |
|-----|----------|
| **모델** | 적절한 모델이 선택되었는가? |
| **농도** | 임상 적용 가능한 농도인가? |
| **대조군** | 적절한 양성/음성 대조가 있는가? |
| **반복** | 충분한 반복 실험이 수행되었는가? |
| **통계** | 적절한 통계 분석이 적용되었는가? |

## 7. 요약 테이블

### 연구 유형별 근거 수준

| 연구 유형 | 근거 수준 | 화장품 적용 |
|----------|----------|------------|
| Meta-analysis | Level 1a | 최종 근거 종합 |
| Systematic Review | Level 1a | 전반적 근거 검토 |
| RCT (DB) | Level 1b | 효능 입증 |
| RCT (Open) | Level 2a | 탐색적 효능 |
| Controlled Trial | Level 2b | 비교 효능 |
| Cohort | Level 2b | 장기 관찰 |
| Case-Control | Level 3a | 이상반응 탐색 |
| Cross-sectional | Level 3b | 현황 조사 |
| Case Report | Level 4 | 새로운 관찰 |
| In vivo | Level 5 | 전임상 효능 |
| In vitro | Level 5 | 기전 연구 |
