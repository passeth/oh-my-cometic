# 근거 평가 기준 상세

화장품 성분의 임상 근거 평가를 위한 상세 기준

## 1. 근거 등급 체계

### Grade A - Strong Evidence (강력한 근거)

| 항목 | 기준 |
|-----|------|
| RCT 수 | ≥ 3건 |
| 총 피험자 | > 200명 |
| 결과 일관성 | > 80% |
| 연구 품질 | 평균 Jadad ≥ 3 |

**클레임 가능 수준**:
- "임상적으로 입증된"
- "과학적으로 증명된"
- 구체적 수치 클레임 가능

**예시 표현**:
```
"임상시험에서 입증된 미백 효과"
"12주 사용 시 색소 침착 35% 감소 (임상 검증)"
```

---

### Grade B - Moderate Evidence (적절한 근거)

| 항목 | 기준 |
|-----|------|
| RCT 수 | 1-2건 |
| 또는 임상시험 | ≥ 3건 |
| 총 피험자 | > 100명 |
| 결과 일관성 | > 60% |

**클레임 가능 수준**:
- "연구에서 확인된"
- "임상 연구 결과"
- 일반적 효능 클레임 가능

**예시 표현**:
```
"피부 보습에 도움을 주는 것으로 연구됨"
"임상 연구에서 피부 탄력 개선 효과 확인"
```

---

### Grade C - Limited Evidence (제한적 근거)

| 항목 | 기준 |
|-----|------|
| 임상시험 | ≥ 1건 |
| 총 피험자 | ≥ 20명 |
| 연구 품질 | 중간 이상 |

**클레임 가능 수준**:
- "연구에 따르면"
- 조건부 효능 표현
- 메커니즘 기반 설명

**예시 표현**:
```
"연구에 따르면 피부 장벽 강화에 도움"
"성분이 가진 항산화 특성"
```

---

### Grade D - Preliminary Evidence (예비 근거)

| 항목 | 기준 |
|-----|------|
| 전임상 연구 | In-vitro/In-vivo만 |
| 임상 데이터 | 없음 |

**클레임 가능 수준**:
- 메커니즘 설명만 가능
- 직접적 효능 클레임 불가
- "~에 기여할 수 있는" 수준

**예시 표현**:
```
"콜라겐 합성을 촉진하는 것으로 알려진 성분"
"항산화 작용을 통해 피부를 보호하는 데 기여"
```

---

### Grade E - Insufficient Evidence (불충분한 근거)

| 항목 | 기준 |
|-----|------|
| 연구 | 없거나 상충 |
| 데이터 | 신뢰할 수 없음 |

**클레임 가능 수준**:
- 효능 클레임 불가
- 일반적 특성만 언급

**예시 표현**:
```
"피부에 수분을 공급하는 성분" (보습제 일반)
"피부 컨디셔닝 성분" (기능 명시 없이)
```

## 2. 연구 품질 평가

### 2.1 Jadad Score (Modified)

RCT 품질 평가를 위한 Jadad Score (0-5점)

| 항목 | 점수 | 평가 기준 |
|-----|------|----------|
| **무작위화 언급** | +1 | "randomized" 또는 동등 표현 |
| **적절한 무작위화** | +1 | 컴퓨터 생성, 난수표 등 적절한 방법 |
| | -1 | 부적절한 방법 (생년월일, 입원 순서 등) |
| **이중맹검 언급** | +1 | "double-blind" 명시 |
| **적절한 맹검** | +1 | 동일 외관의 위약 등 적절한 방법 |
| | -1 | 부적절한 방법 |
| **탈락자 보고** | +1 | 탈락 수, 사유, 처리 방법 기술 |

**점수 해석**:
- 4-5점: High quality
- 3점: Good quality
- 2점: Medium quality
- 0-1점: Low quality

### 2.2 Risk of Bias (Cochrane RoB 2.0 기반)

| 영역 | 평가 항목 | Low Risk 기준 |
|-----|----------|--------------|
| **Selection bias** | 무작위 배정 방법 | 적절한 무작위화 수행 |
| | 배정 은닉 | 배정 순서 은닉됨 |
| **Performance bias** | 참여자 맹검 | 참여자가 배정 모름 |
| | 연구자 맹검 | 연구자가 배정 모름 |
| **Detection bias** | 평가자 맹검 | 결과 평가자가 배정 모름 |
| **Attrition bias** | 불완전 결과 | ITT 분석 수행 |
| | 탈락률 | < 20% |
| **Reporting bias** | 선택적 보고 | 모든 결과 보고됨 |
| **Other bias** | 기타 | 명백한 다른 비뚤림 없음 |

### 2.3 Sample Size Adequacy

| 등급 | 피험자 수 | 검정력 |
|-----|----------|-------|
| Excellent | n ≥ 100 | 대부분의 효과 크기에 충분 |
| Good | n ≥ 50 | 중간 효과 크기 검출 가능 |
| Adequate | n ≥ 30 | 큰 효과 크기 검출 가능 |
| Limited | n ≥ 20 | 매우 큰 효과만 검출 |
| Insufficient | n < 20 | 파일럿 수준 |

## 3. 효능별 평가 지표

### 3.1 미백 (Brightening/Whitening)

| 지표 | 측정 장비 | 유의미한 변화 | 임상적 의미 |
|-----|----------|--------------|------------|
| **ITA°** | 분광측색계 | ≥3° 증가 | 피부톤 밝아짐 |
| **Melanin Index** | Mexameter | ≥10% 감소 | 멜라닌 생성 억제 |
| **L* value** | Chromameter | ≥2 증가 | 명도 증가 |
| **a* value** | Chromameter | 감소 | 홍반 감소 |
| **MASI score** | 임상 평가 | ≥25% 감소 | 기미 개선 |

### 3.2 항노화 (Anti-aging)

| 지표 | 측정 장비 | 유의미한 변화 | 임상적 의미 |
|-----|----------|--------------|------------|
| **주름 깊이** | Replica/Profilometry | ≥10% 감소 | 주름 완화 |
| **Ra (평균 거칠기)** | 레플리카 분석 | ≥10% 감소 | 피부결 개선 |
| **Rz (최대 높이)** | 레플리카 분석 | ≥15% 감소 | 깊은 주름 감소 |
| **탄력도 (R2)** | Cutometer | ≥10% 증가 | 탄력 증가 |
| **피부 두께** | Ultrasound | ≥5% 증가 | 진피 충실도 |
| **콜라겐 밀도** | Dermoscopy | 증가 | 콜라겐 합성 |

### 3.3 보습 (Moisturizing)

| 지표 | 측정 장비 | 유의미한 변화 | 임상적 의미 |
|-----|----------|--------------|------------|
| **Corneometer** | Corneometer | ≥10% 증가 | 수분 함량 증가 |
| **TEWL** | Tewameter | ≥10% 감소 | 장벽 기능 향상 |
| **Skin hydration** | MoistureMeter | ≥10% 증가 | 보습 효과 |
| **Skin capacitance** | Corneometer | 증가 | 전기용량 증가 |

### 3.4 진정 (Soothing/Anti-inflammatory)

| 지표 | 측정 장비 | 유의미한 변화 | 임상적 의미 |
|-----|----------|--------------|------------|
| **Erythema index** | Mexameter | ≥15% 감소 | 홍반 감소 |
| **a* value** | Chromameter | 감소 | 붉음 완화 |
| **TEWL** | Tewameter | ≥10% 감소 | 장벽 회복 |
| **Itching VAS** | VAS scale | ≥30% 감소 | 가려움 완화 |
| **Stinging VAS** | VAS scale | ≥30% 감소 | 따가움 완화 |

### 3.5 항산화 (Antioxidant)

| 지표 | 측정 방법 | 유의미한 변화 | 임상적 의미 |
|-----|----------|--------------|------------|
| **MDA (피부)** | 생검 분석 | 감소 | 산화 스트레스 감소 |
| **Lipid peroxidation** | 피부 측정 | 감소 | 지질 과산화 억제 |
| **Sebum oxidation** | Sebutape 분석 | 감소 | 피지 산화 감소 |

## 4. 메타 분석 기준

### 4.1 연구 포함 기준 (PICOS)

| 요소 | 권장 기준 |
|-----|----------|
| **P (Population)** | 성인, 특정 피부 상태 |
| **I (Intervention)** | 특정 성분, 명시된 농도 |
| **C (Comparison)** | 위약 또는 무처치 |
| **O (Outcomes)** | 객관적 측정 지표 |
| **S (Study design)** | RCT 우선 |

### 4.2 이질성 평가

| I² 값 | 해석 | 권장 조치 |
|------|------|----------|
| 0-25% | 낮음 | 고정효과 모델 |
| 25-50% | 중간 | 하위그룹 분석 |
| 50-75% | 높음 | 변량효과 모델 |
| >75% | 매우 높음 | 메타 분석 부적절 |

## 5. 등급 산정 알고리즘

### 5.1 의사결정 트리

```
START
│
├─ RCT ≥ 3 AND n > 200 AND consistency > 80%?
│   └─ YES → Grade A
│
├─ (RCT ≥ 1 AND n > 100) OR trials ≥ 3?
│   └─ YES → Grade B
│
├─ Clinical trial ≥ 1 AND n ≥ 20?
│   └─ YES → Grade C
│
├─ Preclinical data only?
│   └─ YES → Grade D
│
└─ OTHERWISE → Grade E
```

### 5.2 등급 상향/하향 조정

**상향 조정 요인**:
- 매우 큰 효과 크기 (RR > 2 또는 < 0.5)
- 용량-반응 관계 확인
- 여러 독립 연구팀에서 재현

**하향 조정 요인**:
- 심각한 비뚤림 위험
- 결과 불일관
- 간접 근거
- 부정확한 추정
- 출판 비뚤림 의심

### 6.1 근거 테이블 표준 형식

```markdown
| Study | Year | Design | N | Duration | Conc. | Key Finding | Quality |
|-------|------|--------|---|----------|-------|-------------|---------|
| Author et al. | 20XX | RCT | XX | Xwk | X% | Finding | High/Med/Low |
```

### 6.2 근거 요약문 형식

```

**총 연구**: N건 (RCT: N건)
**총 피험자**: N명

### 주요 발견
- Finding 1
- Finding 2

### 최적 사용 조건
- 농도: X%
- 기간: X주

### 제한점
- Limitation 1
- Limitation 2
```
