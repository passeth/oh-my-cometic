# Consumer Survey Methodology Guide

화장품 소비자 조사를 위한 설문 설계 및 분석 방법론입니다.

## 1. 설문 설계 원칙

### 조사 목적 정의

설문 설계 전 명확한 조사 목적을 정의해야 합니다.

**조사 유형별 목적**

| 조사 유형 | 주요 목적 | 핵심 질문 |
|----------|----------|----------|
| 탐색 조사 | 시장 기회 발굴 | 충족되지 않은 니즈는? |
| 기술 조사 | 현황 파악 | 얼마나, 누가, 어디서? |
| 인과 조사 | 원인 규명 | 왜 그런 행동을 하는가? |
| 평가 조사 | 성과 측정 | 얼마나 만족하는가? |

**목적 정의 체크리스트**
```
[ ] 알고 싶은 것이 무엇인가? (핵심 질문)
[ ] 그 정보로 무엇을 할 것인가? (의사결정)
[ ] 누구에게 물어야 하는가? (대상)
[ ] 어떻게 물어야 하는가? (방법)
[ ] 결과를 어떻게 분석할 것인가? (분석 계획)
```

### 설문 구조 설계

**표준 설문 구조**

```
1. 스크리닝 (Screening) - 적격 응답자 선별
   └ 화장품 구매/사용 경험
   └ 인구통계 (연령, 성별)
   └ 제외 조건 (경쟁사 직원 등)

2. 웜업 (Warm-up) - 주제 환기
   └ 전반적인 뷰티 관심도
   └ 스킨케어 루틴 개요

3. 본조사 (Main Body) - 핵심 질문
   └ 사용 행태 (Usage & Attitude)
   └ 인식/태도 (Perception & Attitude)
   └ 선호도/만족도 (Preference & Satisfaction)

4. 민감 질문 (Sensitive Questions) - 주의 필요
   └ 소득/지출
   └ 피부 트러블 경험

5. 인구통계 (Demographics) - 분류 기준
   └ 상세 인구통계
   └ 라이프스타일

6. 마무리 (Closing)
   └ 추가 의견
   └ 감사 인사
```

**섹션별 권장 문항 수**

| 섹션 | 권장 문항 | 비고 |
|------|----------|------|
| 스크리닝 | 3-5문항 | 필수 |
| 웜업 | 2-3문항 | 선택 |
| 본조사 | 15-20문항 | 핵심 |
| 민감 질문 | 2-3문항 | 필요시 |
| 인구통계 | 5-8문항 | 분석용 |
| **총계** | **25-35문항** | 15-20분 |

### 문항 작성 원칙

**좋은 문항의 조건**
```
1. 명확성: 한 가지 의미만 가짐
2. 단순성: 쉬운 단어, 짧은 문장
3. 중립성: 특정 답변 유도 않음
4. 관련성: 조사 목적과 직결
5. 응답 가능성: 응답자가 대답할 수 있음
```

**피해야 할 문항 유형**

| 문제 유형 | 잘못된 예 | 수정된 예 |
|----------|----------|----------|
| 이중 질문 | 이 제품의 효과와 가격에 만족하십니까? | 효과/가격 분리 질문 |
| 유도 질문 | 대부분의 여성처럼 주름이 걱정되십니까? | 피부 고민 중 주름이 있습니까? |
| 이중 부정 | 보습제를 사용하지 않는 것에 동의하지 않습니까? | 보습제를 사용하십니까? |
| 모호한 단어 | 자주 화장품을 구매하십니까? | 월 평균 화장품 구매 횟수는? |
| 가정 질문 | 신제품이 나오면 구매하시겠습니까? | 최근 3개월 내 신제품 구매 경험? |
| 전문 용어 | AHA/BHA를 사용하십니까? | 각질 제거제를 사용하십니까? |

### 척도 설계

**리커트 척도 (Likert Scale)**

```
5점 척도 (가장 보편적):
1 = 전혀 그렇지 않다
2 = 그렇지 않다
3 = 보통이다
4 = 그렇다
5 = 매우 그렇다

7점 척도 (더 세밀한 측정):
1 = 전혀 동의하지 않는다
2 = 동의하지 않는다
3 = 다소 동의하지 않는다
4 = 보통이다
5 = 다소 동의한다
6 = 동의한다
7 = 매우 동의한다
```

**척도 선택 가이드**

| 척도 | 장점 | 단점 | 권장 상황 |
|------|------|------|----------|
| 5점 | 단순, 익숙함 | 중간 집중 | 일반 조사 |
| 7점 | 세밀한 구분 | 복잡함 | 정밀 측정 |
| 10점 | NPS 등 표준 | 기준 모호 | 만족도 지수 |
| 짝수 점 | 중립 회피 | 강제 선택 | 의견 명확화 |

**기타 척도 유형**

```
순위 척도 (Ranking):
"다음 중 가장 중요한 것부터 순서대로 선택해 주세요"
→ 상대적 중요도 측정

의미분별 척도 (Semantic Differential):
저렴하다 ___:___:___:___:___ 비싸다
촌스럽다 ___:___:___:___:___ 세련됐다
→ 이미지/포지셔닝 측정

NPS (Net Promoter Score):
"이 브랜드를 추천할 의향은? (0-10점)"
9-10: 추천자 / 7-8: 수동적 / 0-6: 비추천자
NPS = 추천자% - 비추천자%
→ 충성도 지수
```

## 2. 샘플 크기 산정

### 표본 크기 계산 공식

**단순 무작위 표본**

```
n = Z² × p × (1-p) / e²

n: 필요 표본 크기
Z: 신뢰수준 Z값
   - 90% → 1.645
   - 95% → 1.960
   - 99% → 2.576
p: 예상 비율 (모를 경우 0.5)
e: 허용 오차 (margin of error)
```

**계산 예시**

```
조건: 95% 신뢰수준, ±5% 오차, 비율 모름
n = 1.96² × 0.5 × 0.5 / 0.05²
n = 3.84 × 0.25 / 0.0025
n = 384명

조건: 95% 신뢰수준, ±3% 오차
n = 1.96² × 0.5 × 0.5 / 0.03²
n = 1,067명
```

### 조사 유형별 권장 표본

**정량 조사 표본 크기**

| 조사 목적 | 최소 | 권장 | 정밀 |
|----------|------|------|------|
| 단순 빈도 분석 | 100 | 200 | 400 |
| 세그먼트 비교 (2개) | 200 | 400 | 600 |
| 세그먼트 비교 (4개) | 400 | 600 | 1,000 |
| 다변량 분석 | 200 | 300 | 500 |
| 구조방정식 | 200 | 300 | 500 |

**하위 그룹 분석 고려**

```
분석 계획:
- 전체 분석 + 연령대별 (4개) + 피부타입별 (4개)

필요 표본:
- 각 하위 그룹 최소 30명
- 연령대 4개 × 30명 = 120명
- 피부타입 4개 × 30명 = 120명
- 중복 고려 → 총 400-500명 권장
```

### 표본 할당 방법

**비례 할당 (Proportional)**
```
모집단 비율에 따라 표본 배분

예시: 20대 30%, 30대 40%, 40대 30%
총 표본 300명일 때:
- 20대: 90명
- 30대: 120명
- 40대: 90명
```

**균등 할당 (Equal)**
```
각 그룹에 동일 표본 배분

예시: 연령대별 분석이 중요할 때
총 표본 300명:
- 20대: 100명
- 30대: 100명
- 40대: 100명
```

**최소 할당**
```
분석 필요 그룹에 최소 표본 확보

예시: 희소 세그먼트 분석 시
- 민감성 피부: 최소 50명 확보 (오버샘플링)
- 기타 피부: 비례 할당
```

## 3. 바이어스 최소화

### 주요 바이어스 유형

**표본 바이어스 (Sampling Bias)**

| 유형 | 원인 | 대응 방안 |
|------|------|----------|
| 선택 바이어스 | 특정 집단만 참여 | 다양한 채널로 모집 |
| 자기 선택 바이어스 | 관심 있는 사람만 응답 | 인센티브 균등 제공 |
| 비응답 바이어스 | 특정 집단 응답 거부 | 응답률 관리, 가중치 |
| 생존자 바이어스 | 이탈자 미포함 | 이탈자 추적 조사 |

**응답 바이어스 (Response Bias)**

| 유형 | 원인 | 대응 방안 |
|------|------|----------|
| 사회적 바람직성 | 좋게 보이려는 응답 | 익명성 보장, 간접 질문 |
| 묵인 바이어스 | 긍정에 동의 경향 | 역코딩 문항 포함 |
| 극단 회피 | 중간값 선호 | 척도 범위 조정 |
| 순서 효과 | 앞 문항 영향 | 문항 순서 랜덤화 |
| 피로 효과 | 후반 성의 없는 답 | 설문 길이 제한 |

### 바이어스 통제 방법

**설문 설계 단계**
```
1. 문항 순서 랜덤화
   - 보기 순서 무작위 배치
   - 유사 문항 떨어뜨려 배치

2. 역코딩 문항 포함
   정문: "이 제품에 만족한다"
   역문: "이 제품에 불만이 있다"
   → 일관성 체크 및 바이어스 감소

3. 강제 선택 vs 중립 옵션
   민감 주제: 중립 옵션 제공
   의견 필요: 강제 선택 (짝수 척도)

4. 필터 질문 활용
   예: 실제 사용자만 상세 평가
```

**데이터 수집 단계**
```
1. 응답 시간 모니터링
   - 너무 빠른 응답 (스피더) 제거
   - 기준: 평균의 1/3 미만

2. 일관성 체크
   - 트랩 질문 (attention check)
   - 역코딩 문항 교차 검증

3. 응답 패턴 확인
   - 직선 응답 (straight-lining) 제거
   - 극단값만 선택 패턴 제거
```

**분석 단계**
```
1. 가중치 적용
   - 모집단 비율과 표본 비율 차이 보정
   - 비응답 그룹 가중치 조정

2. 이상치 처리
   - 극단값 확인 및 처리
   - 민감도 분석 수행

3. 일관성 검증
   - 신뢰도 분석 (Cronbach's alpha)
   - 문항 간 상관 분석
```

### 품질 관리 체크리스트

```
설문 전:
[ ] 파일럿 테스트 실시 (10-20명)
[ ] 문항 이해도 확인
[ ] 소요 시간 측정
[ ] 스크리닝 로직 검증

설문 중:
[ ] 응답률 모니터링
[ ] 할당 충족 확인
[ ] 이상 응답 실시간 체크

설문 후:
[ ] 스피더 제거 (기준: 평균의 1/3)
[ ] 직선 응답 제거
[ ] 트랩 질문 실패자 제거
[ ] 논리적 불일치 응답 검토
```

## 4. 데이터 분석 방법

### 기초 통계 분석

**빈도 분석 (Frequency Analysis)**
```
목적: 각 응답의 분포 파악
적용: 단일 선택, 복수 선택 문항

결과 해석:
- 최빈값 (Mode): 가장 많은 응답
- 분포 형태: 정규, 편포, 이봉분포
- Top Box / Bottom Box 비율
```

**기술 통계 (Descriptive Statistics)**
```
목적: 중심 경향 및 산포 파악
적용: 척도 문항

주요 지표:
- 평균 (Mean): 중심 경향
- 표준편차 (SD): 응답 분산
- 중위수 (Median): 이상치 영향 적음
- 왜도/첨도: 분포 형태
```

### 비교 분석

**교차 분석 (Cross-tabulation)**
```
목적: 두 범주형 변수 관계 분석
적용: 세그먼트별 차이 분석

예시:
연령대 × 선호 채널
- 20대: 온라인 70%, 오프라인 30%
- 40대: 온라인 40%, 오프라인 60%

검정: 카이제곱 검정 (Chi-square)
- p < 0.05 → 유의한 차이 있음
```

**평균 비교 (Mean Comparison)**
```
두 집단 비교 (t-test):
- 독립 t-test: 두 그룹 비교
- 대응 t-test: 사용 전후 비교

다집단 비교 (ANOVA):
- 3개 이상 그룹 평균 비교
- 사후 검정: Scheffe, Tukey 등

예시:
연령대별 브랜드 만족도 비교
F(3, 396) = 4.52, p < 0.01
→ 연령대별 만족도 차이 유의
```

### 고급 분석

**요인 분석 (Factor Analysis)**
```
목적: 다수 변수에서 숨은 요인 추출
적용: 태도/인식 문항 그룹핑

예시:
20개 브랜드 이미지 문항 → 4개 요인 추출
- 요인 1: 품질/효능 (신뢰, 효과, 과학)
- 요인 2: 감성/경험 (세련, 즐거움, 특별함)
- 요인 3: 가격/가치 (합리, 가성비)
- 요인 4: 편의/접근 (구하기 쉬움, 편리)

적합도:
- KMO > 0.6
- Bartlett p < 0.05
- 설명력 > 60%
```

**군집 분석 (Cluster Analysis)**
```
목적: 유사한 응답자 그룹핑
적용: 세그먼테이션

방법:
1. 계층적 군집 (Hierarchical)
   - 덴드로그램으로 군집 수 결정

2. K-평균 군집 (K-means)
   - 사전에 군집 수 지정
   - 각 군집 중심으로 분류

결과 해석:
- 군집별 프로필 비교
- 군집 크기 및 특성
- 군집별 차별적 특성 도출
```

**회귀 분석 (Regression)**
```
목적: 변수 간 인과관계 분석
적용: 만족도 영향 요인, 구매 의도 예측

예시:
재구매 의도 = b0 + b1(효능 만족) + b2(가격 만족) + b3(사용감 만족)

결과 해석:
- R²: 설명력 (높을수록 좋음)
- 표준화 계수 (β): 상대적 영향력
- 유의확률 (p): 통계적 유의성
```

### 분석 결과 보고

**표준 분석 표 형식**

```
[표 1. 응답자 특성]
| 구분 | 항목 | 응답자 수 | 비율(%) |
|------|------|----------|---------|
| 성별 | 여성 | 350 | 70.0 |
|      | 남성 | 150 | 30.0 |
| 연령 | 20대 | 100 | 20.0 |
|      | 30대 | 150 | 30.0 |
|      | 40대 | 150 | 30.0 |
|      | 50대 | 100 | 20.0 |

[표 2. 만족도 평가 (5점 척도)]
| 항목 | 평균 | 표준편차 | Top2Box(%) |
|------|------|----------|------------|
| 효능 | 3.85 | 0.92 | 68.5 |
| 사용감 | 4.12 | 0.78 | 78.2 |
| 가격 | 3.42 | 1.05 | 52.3 |

[표 3. 연령대별 만족도 비교]
| 항목 | 20대 | 30대 | 40대 | 50대 | F값 | p값 |
|------|------|------|------|------|-----|-----|
| 효능 | 3.65 | 3.80 | 3.95 | 4.02 | 3.45 | 0.016* |
```

## 6. 조사 실행 가이드

### 파일럿 테스트

**목적**
```
- 문항 이해도 확인
- 소요 시간 측정
- 로직/분기 오류 확인
- 응답 분포 사전 파악
```

**실행 방법**
```
대상: 목표 응답자와 유사한 10-20명
방법:
1. 실제와 동일 조건으로 응답
2. 각 문항에 대한 피드백 수집
3. 이해하기 어려운 문항 파악
4. 빠진 보기 확인

체크리스트:
[ ] 평균 소요 시간 15-20분 이내
[ ] 이해 안 되는 문항 없음
[ ] 보기가 충분히 포괄적
[ ] 분기 로직 정상 작동
```

### 데이터 클리닝

**제거 기준**
```
1. 스피더 (Speeder)
   - 기준: 평균 응답 시간의 1/3 미만
   - 예: 평균 15분 → 5분 미만 제거

2. 직선 응답 (Straight-liner)
   - 기준: 매트릭스 문항 80% 이상 동일 응답
   - 예: 10개 문항 중 8개 이상 "3"

3. 트랩 질문 실패
   - 예: "이 문항에는 2번을 선택해 주세요" → 오답

4. 논리적 불일치
   - 예: "사용 경험 없음" → 상세 평가 응답

5. 불성실 응답
   - 주관식 무성의 (예: "ㅇㅇ", "모름")
   - 극단값만 연속 선택
```

**클리닝 보고**
```
총 수집: 550명
- 스피더 제거: 25명 (4.5%)
- 직선 응답 제거: 18명 (3.3%)
- 트랩 실패 제거: 12명 (2.2%)
- 불완전 응답: 15명 (2.7%)
------------------------
유효 응답: 480명 (87.3%)
```

### 보고서 작성

**표준 보고서 구성**
```
1. Executive Summary (1-2페이지)
   - 핵심 발견사항 3-5개
   - 전략적 시사점
   - 권고 사항

2. 조사 개요 (1페이지)
   - 조사 목적
   - 조사 방법
   - 조사 기간
   - 표본 특성

3. 주요 결과 (본문)
   - 조사 목적별 결과
   - 그래프/표 활용
   - 통계적 유의성 표기

4. 시사점 및 제언 (1-2페이지)
   - 결과의 의미
   - 실행 방안
   - 추가 조사 제안

5. 부록
   - 상세 데이터 표
   - 설문지
   - 응답자 특성
```

**시각화 가이드**
```
그래프 선택:
- 비율 비교: 막대 그래프 (가로/세로)
- 시계열: 꺾은선 그래프
- 구성비: 파이/도넛 차트
- 분포: 히스토그램
- 관계: 산점도

주의사항:
- Y축은 0부터 시작
- 명확한 제목과 범례
- 핵심 수치 직접 표기
- 과도한 색상 사용 자제
```
