화장품 마케팅을 위한 소비자 페르소나 개발 가이드입니다.

## 2. 데이터 기반 페르소나

### 정량 데이터에서 페르소나 도출

**Step 1: 군집 분석**
```
데이터 준비:
- 설문 응답 데이터
- 변수: 사용 행태, 태도, 인구통계
- 결측치 처리, 표준화

군집 분석 수행:
1. 계층적 군집으로 적정 군집 수 탐색
2. K-means로 최종 군집 도출
3. 군집별 프로필 비교

결과 검증:
- 군집 간 유의한 차이 확인
- 군집 크기 적정성 (최소 10%)
- 군집 해석 가능성
```

**Step 2: 군집 프로파일링**
```
각 군집별:
- 인구통계 분포
- 사용 행태 특성
- 태도/인식 특성
- 브랜드 선호
- 채널 선호

군집 명명:
- 핵심 특성 반영한 이름
- 예: "효능 추구형", "가성비 실용형"
```

**Step 3: 페르소나 구체화**
```
군집 데이터 → 페르소나 전환:

군집 특성:
- 30대 여성 비율 높음 (45%)
- 직장인 비율 높음 (68%)
- 복합성 피부 다수 (52%)
- 모공/피지 고민 (61%)
- 올리브영 주 이용 (73%)
- 인스타그램 정보 탐색 (82%)

↓ 페르소나화

이름: 김민지 (32세)
직업: 대기업 마케터
피부: 복합성, 모공/피지 고민
채널: 올리브영, 인스타그램
```

### 정성 데이터에서 페르소나 도출

**FGI/IDI 데이터 분석**
```
Step 1: 전사 및 코딩
- 인터뷰 전사
- 주제별 코딩 (태도, 행동, 니즈, 페인포인트)

Step 2: 패턴 발견
- 반복되는 테마 추출
- 응답자 간 공통점/차이점

Step 3: 유형화
- 유사한 특성 가진 응답자 그룹핑
- 각 유형의 대표 특성 정의

Step 4: 페르소나 구성
- 각 유형을 대표하는 페르소나 개발
- 실제 응답 인용으로 생생함 부여
```

**인용문(Verbatim) 활용**
```
페르소나 리얼리티를 위한 실제 응답 활용:

니즈 관련:
"아침에 시간이 없어서 빨리 흡수되는 게 제일 중요해요"
→ 빠른 흡수, 시간 효율 니즈

페인포인트 관련:
"좋다는 제품 다 써봤는데 뭐가 진짜 좋은 건지 모르겠어요"
→ 정보 과잉, 선택 어려움

브랜드 기대:
"믿을 수 있는 성분으로, 확실하게 효과 나는 거요"
→ 신뢰성, 효능 증명 기대
```

### 복합 데이터 통합

**데이터 소스 삼각검증**
```
정량 데이터:
- 설문 조사 (N=500)

정성에서 발견된 인사이트를 정량으로 검증
```

## 3. 페르소나 검증

### 내부 검증

**체크리스트**
```
현실성:
[ ] 실제 존재할 법한 인물인가?
[ ] 모순되는 특성이 없는가?
[ ] 너무 이상적이지 않은가?

차별성:
[ ] 다른 페르소나와 명확히 구분되는가?
[ ] 각 페르소나가 의미 있는 세그먼트를 대표하는가?

실용성:
[ ] 제품/마케팅 의사결정에 활용 가능한가?
[ ] 팀원들이 쉽게 이해하고 공감하는가?

데이터 기반:
[ ] 조사 데이터에 근거하는가?
[ ] 추측이나 편견이 아닌 사실 기반인가?
```

### 외부 검증

**소비자 검증**
```
방법 1: 페르소나 카드 테스트
- 타겟 소비자에게 페르소나 카드 제시
- "이 사람과 얼마나 비슷하세요?" 질문
- 유사성 점수 및 피드백 수집

방법 2: 자기 분류

|------|------|
| 피부 타입 | 지성 (T존 특히) |

| 정보 채널 | 틱톡, 인스타 릴스, 화해 앱 |

## 라이프스타일
SNS를 통해 새로운 정보를 습득하고, 친구들과 공유하는 것을 좋아함.
가성비를 중요시하지만, 화제가 되는 제품은 꼭 써보고 싶어함.
환경과 동물 복지에 관심이 있어 비건 제품에 호감.

## 핵심 니즈
1. 모공과 피지 관리 (기능적)
2. "피부 좋다"는 칭찬 듣기 (사회적)
3. 스킨케어가 재미있고 만족스러운 경험 (감정적)

## 페인포인트
1. 어떤 제품이 내 피부에 맞는지 모르겠음
2. 좋다는 제품이 너무 많아서 선택이 어려움
3. 효과가 나타나는지 판단하기 어려움

## 구매 결정 요인
1. 틱톡/인스타 리뷰 (실제 사용 영상)
2. 가격 (3만원 이하 선호)
3. 화해 앱 성분 평가

## 대표 Quote
"틱톡에서 대박 났다는 거 다 써봐야 직성이 풀려요"
"성분 빨간색 많으면 일단 패스"
"진짜 효과 있는 건지 모르겠어서 계속 바꿔보게 돼요"

## 메시지 전략
"모공 고민 끝, 진짜 효과를 직접 확인하세요"
- 숏폼 영상으로 즉각적인 효과 시각화
- 성분 안전성 강조 (화해 연동)
- 합리적 가격 어필
```

- 프리미엄 이미지 + 과학적 근거
