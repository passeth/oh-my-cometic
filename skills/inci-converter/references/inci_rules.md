# INCI Naming Rules (INCI 명명 규칙)

INCI(International Nomenclature of Cosmetic Ingredients) 명명 체계의 공식 규칙과 표기법을 정리한 문서입니다.

## 1. 기본 원칙

### 1.1 대문자 규칙

**모든 INCI명은 대문자로 표기합니다.**

```
올바른 표기:
- NIACINAMIDE
- SODIUM HYALURONATE
- GLYCERIN

잘못된 표기:
- Niacinamide
- sodium hyaluronate
- Glycerin
```

### 1.2 언어 기준

- **화학명**: 영어 또는 수정된 라틴어 사용
- **식물명**: 라틴어 학명 사용
- **일반명**: 영어 사용

### 1.3 공백 및 구두점

```
공백 사용:
- HYALURONIC ACID (O)
- HYALURONICACID (X)

하이픈 사용:
- ALPHA-TOCOPHEROL (O)
- 숫자 연결: PEG-40 (O)

슬래시 사용:
- 복합명에서: 일반적으로 사용하지 않음
```

## 2. 식물 성분 명명법

### 2.1 식물명 구조

식물 성분의 INCI명은 다음 구조를 따릅니다:

```
[학명 (일반명)] [부위] [형태]

예시:
CAMELLIA SINENSIS (GREEN TEA) LEAF EXTRACT
         ↓              ↓        ↓       ↓
      학명          일반명    부위    형태
```

### 2.2 학명 표기

**속명(Genus) + 종명(Species)** 순서로 표기:

```
Camellia sinensis → CAMELLIA SINENSIS
Rosa damascena → ROSA DAMASCENA
Lavandula angustifolia → LAVANDULA ANGUSTIFOLIA
Aloe barbadensis → ALOE BARBADENSIS
Centella asiatica → CENTELLA ASIATICA
```

### 2.3 일반명 표기 (선택적)

학명 뒤 괄호 안에 일반명 표기 가능:

```
SIMMONDSIA CHINENSIS (JOJOBA) SEED OIL
MELALEUCA ALTERNIFOLIA (TEA TREE) LEAF OIL
PRUNUS AMYGDALUS DULCIS (SWEET ALMOND) OIL
BUTYROSPERMUM PARKII (SHEA) BUTTER
COCOS NUCIFERA (COCONUT) OIL
```

### 2.4 식물 부위 표기

| 영문 | 한글 | 예시 |
|------|------|------|
| LEAF | 잎 | CAMELLIA SINENSIS LEAF EXTRACT |
| FLOWER | 꽃 | ROSA DAMASCENA FLOWER WATER |
| ROOT | 뿌리 | GLYCYRRHIZA GLABRA ROOT EXTRACT |
| SEED | 씨앗 | SIMMONDSIA CHINENSIS SEED OIL |
| FRUIT | 열매 | CITRUS LIMON FRUIT EXTRACT |
| BARK | 껍질 | CINNAMOMUM CASSIA BARK EXTRACT |
| STEM | 줄기 | BAMBUSA VULGARIS STEM EXTRACT |
| PEEL | 과피 | CITRUS AURANTIUM DULCIS PEEL OIL |
| BERRY | 열매 | VACCINIUM MYRTILLUS BERRY EXTRACT |
| BUD | 새싹 | ROSA CANINA BUD EXTRACT |
| WHOLE PLANT | 전초 | CENTELLA ASIATICA |

### 2.5 추출물 형태 표기

| 영문 | 한글 | 설명 |
|------|------|------|
| EXTRACT | 추출물 | 일반 추출물 |
| OIL | 오일 | 유지 성분 |
| WATER | 워터 | 증류수/플로럴워터 |
| POWDER | 분말 | 건조 분말 |
| JUICE | 주스 | 압착액 |
| WAX | 왁스 | 왁스 성분 |
| BUTTER | 버터 | 식물성 버터 |

### 2.6 식물 추출물 예시

```
녹차 추출물:
CAMELLIA SINENSIS LEAF EXTRACT

알로에 추출물:
ALOE BARBADENSIS LEAF EXTRACT
ALOE BARBADENSIS LEAF JUICE

장미 추출물:
ROSA DAMASCENA FLOWER EXTRACT
ROSA DAMASCENA FLOWER WATER
ROSA DAMASCENA FLOWER OIL

라벤더:
LAVANDULA ANGUSTIFOLIA FLOWER EXTRACT
LAVANDULA ANGUSTIFOLIA FLOWER OIL
LAVANDULA ANGUSTIFOLIA FLOWER WATER

호호바:
SIMMONDSIA CHINENSIS (JOJOBA) SEED OIL

티트리:
MELALEUCA ALTERNIFOLIA (TEA TREE) LEAF OIL

아르간:
ARGANIA SPINOSA KERNEL OIL
```

## 3. 숫자 및 기호 사용

### 3.1 숫자 표기 규칙

**숫자는 하이픈으로 연결하거나 공백으로 분리합니다:**

```
PEG 계열:
- PEG-40 HYDROGENATED CASTOR OIL
- PEG-100 STEARATE
- PEG-8 DIMETHICONE

PPG 계열:
- PPG-15 STEARYL ETHER
- PPG-26-BUTETH-26

폴리소르베이트:
- POLYSORBATE 20
- POLYSORBATE 60
- POLYSORBATE 80

숫자 접두사:
- 1,2-HEXANEDIOL
- 1,3-BUTYLENE GLYCOL
```

### 3.2 그리스 문자

그리스 문자는 영문으로 표기:

```
α (alpha) → ALPHA
β (beta) → BETA
γ (gamma) → GAMMA

예시:
- ALPHA-TOCOPHEROL
- BETA-CAROTENE
- GAMMA-LINOLENIC ACID
```

### 3.3 화학 접두사

```
소듐(Sodium): Na →
- SODIUM HYALURONATE
- SODIUM LAURYL SULFATE
- SODIUM CHLORIDE

포타슘(Potassium): K →
- POTASSIUM SORBATE
- POTASSIUM HYDROXIDE

칼슘(Calcium): Ca →
- CALCIUM CARBONATE
- CALCIUM PANTOTHENATE

마그네슘(Magnesium): Mg →
- MAGNESIUM SULFATE
- MAGNESIUM ASCORBYL PHOSPHATE
```

## 4. 색소 표기 (CI 번호)

### 4.1 CI 번호 체계

**CI(Colour Index) 번호**를 사용하여 색소를 표기합니다:

```
형식: CI + 5자리 숫자

예시:
- CI 77891 (티타늄디옥사이드, 백색)
- CI 77491 (산화철, 적색)
- CI 77492 (산화철, 황색)
- CI 77499 (산화철, 흑색)
- CI 42090 (블루 1)
- CI 15985 (옐로우 6)
- CI 19140 (타르트라진)
```

### 4.2 일반 색소 CI 번호

| 색상 | CI 번호 | 성분명 |
|------|---------|--------|
| 백색 | CI 77891 | TITANIUM DIOXIDE |
| 적색 | CI 77491 | IRON OXIDES (RED) |
| 황색 | CI 77492 | IRON OXIDES (YELLOW) |
| 흑색 | CI 77499 | IRON OXIDES (BLACK) |
| 청색 | CI 42090 | BLUE 1 |
| 청색 | CI 77007 | ULTRAMARINES |
| 적색 | CI 15850 | RED 6, RED 7 |
| 적색 | CI 45410 | RED 27, RED 28 |
| 황색 | CI 19140 | YELLOW 5 |
| 녹색 | CI 77288 | CHROMIUM OXIDE GREENS |

### 4.3 색소 표기 예시

```
제품 라벨 표기:
MAY CONTAIN (+/-): CI 77891, CI 77491, CI 77492, CI 77499

또는:
TITANIUM DIOXIDE (CI 77891)
IRON OXIDES (CI 77491, CI 77492, CI 77499)
```

## 5. 특수 표기

### 5.1 나노물질 표기

나노 형태의 물질은 (NANO) 표기 추가:

```
TITANIUM DIOXIDE (NANO)
ZINC OXIDE (NANO)
```

### 5.2 혼합물/컴플렉스 표기

```
슬래시(/) 사용:
거의 사용하지 않음

AND 사용:
CETEARYL OLIVATE AND SORBITAN OLIVATE
(Olivem 1000 - 올리브유래 유화제)

콤마(,) 사용:
개별 성분으로 나열
```

### 5.3 향료 및 착향제

```
향료:
FRAGRANCE (미국/한국)
PARFUM (유럽)

아로마:
AROMA (식품 향료)
```

### 5.4 정제수 표기

```
유럽:
AQUA

미국/한국:
WATER

국제 표준:
AQUA 권장
```

## 6. 지역별 차이

### 6.1 한국 vs 미국 vs EU

| 항목 | 한국 | 미국 | EU |
|------|------|------|-----|
| 물 | WATER/AQUA | WATER | AQUA |
| 향료 | FRAGRANCE | FRAGRANCE | PARFUM |
| 색소 | CI 번호 | FD&C/D&C | CI 번호 |
| 대소문자 | 대문자 | 대문자 | 대문자 |

### 6.2 표기 통일 권장사항

국제 호환성을 위해:
- **물**: AQUA 사용
- **향료**: PARFUM 또는 FRAGRANCE (시장별 선택)
- **색소**: CI 번호 병기

## 7. 흔한 오류 및 수정

### 7.1 철자 오류

```
잘못된 표기 → 올바른 표기
NIACIAMIDE → NIACINAMIDE
HYALORONIC → HYALURONIC
TOCOFEROL → TOCOPHEROL
GLYCERINE → GLYCERIN
PROPYLENE GLYCOLE → PROPYLENE GLYCOL
```

### 7.2 순서 오류

```
잘못된 표기 → 올바른 표기
ACID HYALURONIC → HYALURONIC ACID
EXTRACT LEAF GREEN TEA → CAMELLIA SINENSIS LEAF EXTRACT
OIL JOJOBA → SIMMONDSIA CHINENSIS SEED OIL
```

### 7.3 불완전한 표기

```
불완전한 표기 → 올바른 표기
GREEN TEA EXTRACT → CAMELLIA SINENSIS LEAF EXTRACT
ALOE EXTRACT → ALOE BARBADENSIS LEAF EXTRACT
ROSE OIL → ROSA DAMASCENA FLOWER OIL
```

## 8. INCI 등록

### 8.1 PCPC (Personal Care Products Council)

- INCI명의 공식 등록 기관
- INCI Dictionary 발행
- 신규 성분 등록 절차 관리

### 8.2 등록 절차

1. 성분 정보 제출
2. 화학 구조 검토
3. 기존 INCI명 중복 확인
4. 새 INCI명 부여 또는 기존명 사용 안내
5. INCI Dictionary 업데이트

### 8.3 임시 명칭

공식 INCI명이 없는 경우:
```
회사명 + 성분 설명

예시:
[COMPANY NAME] PEPTIDE COMPLEX
[COMPANY NAME] FERMENT FILTRATE
```
