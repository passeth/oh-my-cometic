# 화장품 전달 시스템 종합 가이드

## 개요

전달 시스템(Delivery System)은 활성 성분을 피부 타겟 부위에 효과적으로 전달하고,
성분의 안정성과 효능을 극대화하기 위한 기술입니다.

## 1. 소포체 시스템 (Vesicular Systems)

### 1.1 리포좀 (Liposome)

#### 구조
```
     ┌─────────────────────────┐
     │    수용성 성분 (내부)    │
     │  ┌───────────────────┐  │
     │  │   ○ ○ ○ ○ ○ ○    │  │
     │  │  인지질 이중층     │  │
     │  │   ○ ○ ○ ○ ○ ○    │  │
     │  └───────────────────┘  │
     │     지용성 성분         │
     │   (이중층 사이)         │
     └─────────────────────────┘
```

#### 특성
| 항목 | 내용 |
|------|------|
| 크기 | 50-500 nm (SUV: 20-100nm, LUV: 100-500nm, MLV: 500nm+) |
| 구성 | 인지질 (레시틴, PC) + 콜레스테롤 |
| 캡슐화 | 수용성 (내부) + 지용성 (이중층) 모두 가능 |
| 전하 | 음전하/양전하/중성 조절 가능 |

#### 리포좀 종류
| 종류 | 크기 | 층수 | 봉입률 | 용도 |
|------|------|------|--------|------|
| SUV (Small Unilamellar) | 20-100nm | 단일 | 낮음 | 침투 중시 |
| LUV (Large Unilamellar) | 100-500nm | 단일 | 높음 | 범용 |
| MLV (Multilamellar) | 500nm+ | 다중 | 매우 높음 | 서방형 |
| GUV (Giant Unilamellar) | 1-100μm | 단일 | - | 연구용 |

#### 제조 방법
```
1. 박막법 (Thin Film Method)
   인지질 + 클로로포름 → 감압 증발 → 박막 형성 → 수화 → 초음파/압출

2. 역상 증발법 (REV)
   W/O 에멀전 형성 → 유기용매 제거 → 리포좀

3. 에탄올 주입법 (Ethanol Injection)
   에탄올에 용해된 인지질 → 수상에 주입 → 자발적 형성

4. 고압 균질화법 (HPH)
   조리포좀 → 고압 균질기 통과 → 균일 크기 SUV
```

#### 적합 성분
- 비타민C (아스코르브산)
- 레티놀/레티노이드
- 펩타이드 (Matrixyl, Argireline)
- CoQ10 (유비퀴논)
- 히알루론산 (저분자)
- 니아신아마이드
- 카페인

#### 시판 리포좀 원료
| 제품명 | 제조사 | 활성 성분 | 특징 |
|--------|--------|-----------|------|
| Natipide II | Nikkol | 빈 리포좀 | 자체 캡슐화용 |
| TEGO® Liposomes | Evonik | 다양 | 고품질 |
| PhytoSolve® | DSM | 식물 추출물 | 생체이용률↑ |
| Lipoid S75 | Lipoid | 대두 레시틴 | 75% PC |
| PHOSPHOLIPON 90G | Lipoid | 대두 레시틴 | 90% PC |

### 1.2 니오좀 (Niosome)

#### 구조
비이온성 계면활성제로 형성된 소포체 (리포좀의 대안)

#### vs 리포좀 비교
| 특성 | 리포좀 | 니오좀 |
|------|--------|--------|
| 구성 물질 | 인지질 | 비이온 계면활성제 |
| 안정성 | 산화 취약 | 화학적 안정 |
| 비용 | 높음 | 낮음 |
| 피부 침투 | 우수 | 우수 |
| 저장 안정성 | 짧음 | 김 |
| 대량 생산 | 어려움 | 용이 |

#### 구성 성분
```
계면활성제 (Span 류):
├── Span 60 (Sorbitan Monostearate) - HLB 4.7
├── Span 40 (Sorbitan Monopalmitate) - HLB 6.7
├── Span 80 (Sorbitan Monooleate) - HLB 4.3
└── Span 20 (Sorbitan Monolaurate) - HLB 8.6

보조 계면활성제 (Tween 류):
├── Tween 20 - HLB 16.7
├── Tween 60 - HLB 14.9
└── Tween 80 - HLB 15.0

안정화제:
├── 콜레스테롤 (20-50%)
├── Dicetyl Phosphate (음전하 부여)
└── Stearylamine (양전하 부여)
```

#### 제조 방법
```
표준 배합 (예시):
Span 60           47.5%
Cholesterol       47.5%
DCP (선택적)       5.0%

공정:
1. 지질상 70°C 용융
2. 약물 용해 수상 첨가
3. 교반하며 수화
4. 초음파 또는 압출
```

#### 적합 성분
- 레티노이드
- 항산화제 (비타민E, 아스타잔틴)
- 자외선 차단제
- 항염증 성분

### 1.3 에토좀 (Ethosome)

#### 특성
```
구성:
├── 인지질 (2-5%)
├── 에탄올 (20-45%) ← 핵심 차별점
├── 물 (나머지)
└── 활성 성분

특징:
├── 피부 침투력 리포좀 대비 5-15배↑
├── 연성(flexible) 소포체
├── 에탄올이 각질층 지질 유동화
└── 세포간 경로 침투 증가
```

#### 침투 메커니즘
```
1단계: 에탄올이 각질층 지질 구조 교란
     ↓
2단계: 유연한 에토좀이 세포간 공간 통과
     ↓
3단계: 진피층 도달 및 성분 방출
```

#### 적합 성분
- 미백제 (알부틴, 코직산)
- 항진균제
- 항염증제
- 트레티노인
- 민옥시딜

#### 제조 배합 (예시)
```
Phosphatidylcholine    2.0%
Ethanol               30.0%
활성 성분              적량
정제수                 to 100%

공정: 인지질을 에탄올에 용해 → 활성 용해 → 물 서서히 첨가 → 교반
```

### 1.4 트랜스퍼좀 (Transfersome)

#### 특성
```
구성:
├── 인지질 (주성분)
├── 가장자리 활성제 (Edge Activator) 10-25%
│   ├── Sodium Cholate
│   ├── Sodium Deoxycholate
│   ├── Span 80
│   ├── Tween 80
│   └── Dipotassium Glycyrrhizinate
└── 물

핵심 특징:
├── 초변형성 (Ultra-deformable)
├── 자신보다 5-10배 작은 기공 통과 가능
├── 피부 수분 구배 이용한 자가 침투
└── 비침습적 경피 전달
```

#### 침투 메커니즘
```
수분 구배 구동 (Hydration Gradient):

피부 표면 (건조, ~20% 수분)
          ↑
    트랜스퍼좀 이동 (삼투압 구동)
          ↑
진피층 (습윤, ~70% 수분)
```

#### vs 리포좀/에토좀
| 특성 | 리포좀 | 에토좀 | 트랜스퍼좀 |
|------|--------|--------|------------|
| 변형성 | 낮음 | 중간 | 매우 높음 |
| 침투 깊이 | 표피 | 진피 | 진피/전신 |
| 침투 기전 | 융합 | 에탄올 | 변형 |
| 알코올 | 없음 | 20-45% | 없음 |

#### 적합 성분
- 거대 분자 (펩타이드, 단백질)
- 인슐린 (경피 전달 연구)
- 성장인자
- 백신 항원
- 고분자량 히알루론산

```
       ┌─────────────────┐

       │     복합체       │
       │  ┌───────────┐  │

       │  │ (수소결합) │  │
```

## 2. 지질 기반 시스템 (Lipid-based Systems)

### 2.1 고체 지질 나노입자 (SLN)

#### 구조
```
  ┌───────────────────┐
  │ 고체 지질 매트릭스 │
  │   ┌─────────┐     │
  │   │ 활성물질│     │
  │   └─────────┘     │
  │  계면활성제 코팅   │
  └───────────────────┘
      50-1000 nm
```

#### 구성 성분
```
고체 지질 (융점 > 40°C):
├── 트리글리세리드: Tristearin, Tripalmitin, Trimyristin
├── 글리세리드: Glyceryl Monostearate, Glyceryl Behenate
├── 지방산: Stearic Acid, Palmitic Acid
├── 왁스: Cetyl Palmitate, Beeswax, Carnauba Wax
└── 기타: Compritol 888 ATO, Precirol ATO 5

계면활성제:
├── Poloxamer 188, 407
├── Polysorbate 80
├── Lecithin
└── Sodium Cholate
```

#### 약물 봉입 모델
```
1. 균질 매트릭스 모델 (Homogeneous Matrix)
   - 약물이 매트릭스 전체에 균일 분포
   - 서방형 방출

2. 약물 강화 외각 모델 (Drug-enriched Shell)
   - 약물이 외각에 집중
   - 초기 방출 빠름

3. 약물 강화 코어 모델 (Drug-enriched Core)
   - 약물이 중심에 집중
   - 서방형 방출
```

#### 장단점
| 장점 | 단점 |
|------|------|
| 서방형 방출 | 봉입률 한계 |
| 피부 보호막 | 젤화 경향 |
| 안정성 양호 | 결정화 문제 |
| 폐쇄효과 (보습) | 약물 방출 |

### 2.2 나노구조 지질 담체 (NLC)

#### vs SLN 개선점
```
SLN 문제점:
├── 결정 형성 → 약물 방출
├── 봉입률 한계
└── 저장 중 약물 방출

NLC 해결책:
├── 고체 + 액체 지질 혼합 (70:30 ~ 90:10)
├── 불완전 결정 구조
└── 높은 봉입률 (최대 95%+)
```

#### NLC 유형
| 유형 | 구조 | 특징 |
|------|------|------|
| Type I (불완전 결정) | 액체 지질이 결정 결함 생성 | 기본형 |
| Type II (비결정) | 고체+액체 혼합, 결정 없음 | 최고 봉입률 |
| Type III (다중) | 액체 나노구획 포함 | 서방형 최적화 |

#### 구성 예시
```
고체 지질 (70%):
├── Precirol ATO 5
├── Compritol 888 ATO
└── Glyceryl Monostearate

액체 지질 (30%):
├── Miglyol 812 (MCT)
├── Oleic Acid
├── Squalene
└── Isopropyl Myristate

계면활성제 (1-5%):
├── Poloxamer 188
└── Tween 80
```

#### 적합 성분
- 레티노이드 (레티놀, 레티날데하이드)
- CoQ10
- 비타민E
- 커큐민
- 세라마이드
- 자외선차단제

### 2.3 나노에멀전 (Nanoemulsion)

#### 특성
| 항목 | 나노에멀전 | 마이크로에멀전 |
|------|------------|----------------|
| 크기 | 20-200 nm | 10-100 nm |
| 열역학 | 불안정 (동역학 안정) | 열역학 안정 |
| 외관 | 투명/반투명 | 투명 |
| 계면활성제 | 낮음 (3-10%) | 높음 (15-30%) |
| 제조 | 고에너지 필요 | 자발적 가능 |

#### 제조 방법

**고에너지법:**
```
1. 고압 균질기 (HPH)
   - 500-2000 bar
   - 3-10 사이클

2. 마이크로플루이다이저
   - 미세 채널 통과
   - 균일한 크기 분포

3. 초음파 처리
   - 20-25 kHz
   - 캐비테이션 효과
```

**저에너지법:**
```
1. 자발적 유화 (Spontaneous Emulsification)
   - 용매 확산
   - 계면 불안정

2. 상전이 온도법 (PIT)
   - POE 계면활성제 사용
   - 온도 변화로 HLB 변경

3. 상전이 조성법 (PIC)
   - 물 적정으로 상전이
   - 상온 제조 가능
```

#### 적합 성분
- 지용성 비타민 (A, D, E, K)
- 카로티노이드
- 플라보노이드
- 정유 (에센셜 오일)
- 향료

---

### 2.4 마이크로에멀전 (Microemulsion)

#### 특성
```
정의:
├── 열역학적 안정
├── 등방성 (isotropic)
├── 투명
└── 자발적 형성

구성 (Winsor 시스템):
├── 오일상
├── 수상
├── 계면활성제 (10-30%)
└── 보조 계면활성제 (co-surfactant)
```

#### 구조 유형
```
Winsor I: O/W 마이크로에멀전 + 과잉 오일
Winsor II: W/O 마이크로에멀전 + 과잉 물
Winsor III: 바이컨티뉴어스 + 과잉 오일/물
Winsor IV: 단일상 마이크로에멀전 (이상적)
```

#### 장점
- 열역학 안정 (장기 저장)
- 자발적 형성 (저에너지)
- 높은 용해력
- 빠른 피부 침투

#### 단점
- 높은 계면활성제 함량 → 자극 가능
- 희석 시 불안정
- 성분 호환성 제한

| 고분자 | 특성 | 분해성 |
|--------|------|--------|

#### 제조 방법
```

   CO2 초임계 조건에서 형성
```

#### 크기 범위

- 나노캡슐: 10-1000 nm

#### 캡슐화 방법
| 방법 | 원리 | 적합 물질 | 특징 |
|------|------|-----------|------|
| 코아세르베이션 | 상분리 | 정유, 향료 | 고봉입률 |
| 스프레이 드라이 | 분무 건조 | 수용성 | 대량 생산 |
| 계면 중합 | 계면 반응 | 다양 | 단단한 벽 |
| 유동층 코팅 | 분무 코팅 | 고체 | 균일 코팅 |
| 솔-겔 | 무기 캡슐 | 효소, 세포 | 실리카 벽 |

#### 방출 메커니즘
```
1. 기계적 파괴 (Mechanical)
   압력, 마찰 → 벽 파괴 → 즉시 방출
   예: 향수, 스크럽

2. 확산 (Diffusion)
   벽을 통한 점진적 확산
   예: 서방형 활성 성분

3. 용해 (Dissolution)
   벽 물질 용해 → 방출
   예: 수용성 벽 (젤라틴)

4. 팽윤/삼투 (Swelling/Osmotic)
   수분 흡수 → 팽윤 → 방출
   예: pH 반응성

5. 온도 반응 (Thermoresponsive)
   특정 온도에서 방출
   예: 피부 온도에서 활성화
```

## 9. 안정성 평가 방법

### 필수 테스트
```
1. 입자 크기 분포
   - 동적 광산란 (DLS)
   - 나노입자 추적 분석 (NTA)
   - 레이저 회절

2. 제타 전위
   - ±30 mV 이상: 안정
   - ±20-30 mV: 불안정 가능
   - <±20 mV: 불안정

3. 봉입률 (EE%)
   EE% = (봉입된 약물 / 총 약물) × 100
   - 리포좀: 20-60%
   - SLN/NLC: 70-95%
```
