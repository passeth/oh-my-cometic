# 중국 NMPA 화장품 규제 상세 가이드

## 개요

**NMPA (National Medical Products Administration, 국가약품감독관리국)**는 중국의 화장품 규제를 총괄합니다. 2021년 새로운 「화장품감독관리조례」 시행으로 규제가 대폭 강화되었습니다.

### 기본 정보
| 항목 | 내용 |
|------|------|
| **규제 기관** | NMPA (국가약품감독관리국) |
| **근거 법령** | 화장품감독관리조례 (2021.1.1 시행) |
| **하위 규정** | 화장품등록비안관리방법, 화장품안전기술규범 등 |
| **적용 범위** | 중국 내 판매 모든 화장품 |
| **등록 포털** | https://zwfw.nmpa.gov.cn |

### 규제 체계
```
법령 체계:
├── 화장품감독관리조례 (条例)
│   └── 국무원령, 최상위 법규
├── 화장품등록비안관리방법 (办法)
│   └── NMPA령, 등록/비안 절차
├── 화장품안전기술규범 (技术规范)
│   └── 성분 제한, 테스트 방법
├── 화장품표식관리방법 (办法)
│   └── 라벨 요건
└── 기타 기술 지침/공고
```

## 화장품 분류

중국은 화장품을 **일반(普通)**과 **특수(特殊)**로 이원화 관리

### 분류 체계

| 분류 | 중문 | 관리 방식 | 심사 기관 |
|------|------|-----------|-----------|
| **일반화장품** | 普通化妆品 | 비안 (备案) | 성급 약감국 |
| **특수화장품** | 特殊化妆品 | 주책 (注册) | NMPA |

### 일반화장품 (普通化妆品)

**비안 (备案, Filing)** 제도 적용

```
일반화장품 범주:
├── 기초화장품 (스킨케어)
│   ├── 클렌저, 토너, 에센스
│   ├── 로션, 크림
│   └── 마스크팩
├── 색조화장품
│   ├── 파운데이션, 컨실러
│   ├── 립스틱, 립글로스
│   ├── 아이섀도, 아이라이너
│   └── 블러셔
├── 향수
├── 바디케어
├── 네일케어
└── 헤어케어 (염색/파마 제외)
```

### 특수화장품 (特殊化妆品)

**주책 (注册, Registration)** 제도 적용 (9개 카테고리)

| 번호 | 유형 | 중문 | 비고 |
|------|------|------|------|
| 1 | 염발 제품 | 染发 | 산화성/비산화성 |
| 2 | 파마 제품 | 烫发 | 퍼머넌트 웨이브 |
| 3 | 피부 미백 | 祛斑美白 | 멜라닌 억제 클레임 |
| 4 | 자외선차단 | 防晒 | SPF/PA 표시 |
| 5 | 탈모 방지 | 防脱发 | 모발 성장 촉진 |
| 6 | 체취 방지 | 祛臭 | 발한억제 포함 |
| 7 | 여드름 치료 | 祛痘 | 여드름 완화 |
| 8 | 제모 | 脱毛 | 영구/일시적 |
| 9 | 가슴 미용 | 美乳 | 가슴 형태 개선 |

## 등록/비안 절차

### 일반화장품 비안 (备案) 절차

```
국내 생산:
├── 1. 제품 개발 및 테스트
├── 2. 성급 약감국에 비안 자료 제출
├── 3. 자료 검토 (약 5-10 영업일)
├── 4. 비안 번호 발급
└── 5. 시장 출시

수입:
├── 1. 수입 비안인 지정 (경내책임인)
├── 2. 테스트 (중국 검사기관 필수)
├── 3. NMPA에 비안 자료 제출
├── 4. 자료 검토 (약 5-10 영업일)
├── 5. 비안 번호 발급
└── 6. 시장 출시
```

#### 비안 제출 자료
```
필수 자료:
├── 비안 신청서
├── 제품 배합표 (전성분, 농도)
├── 제조공정 개요
├── 제품 집행 기준
├── 안전성 평가 자료 (CSAR)
├── 제품 라벨 샘플
├── 제품 품질 테스트 보고서
├── 제조사 GMP 증명
└── 위탁 생산 시 계약서

수입 추가:
├── 수입 비안인 위임서
├── 원산지 증명
├── 자유판매증명서 (FSC)
└── 중국어 라벨
```

### 특수화장품 주책 (注册) 절차

```
등록 절차:
├── 1. 기술 심사 자료 준비
├── 2. 중국 검사기관에서 테스트
│   ├── 위생학 검사
│   ├── 독성학 검사
│   └── 인체 효능 시험 (필요 시)
├── 3. NMPA에 주책 신청서 제출
├── 4. 기술 심사 (약 90-120 영업일)
│   ├── 행정 심사
│   └── 기술 심사
├── 5. 주책증 발급 (유효기간 5년)
└── 6. 시장 출시
```

#### 주책 제출 자료
```
필수 자료:
├── 주책 신청서
├── 제품 배합표 (상세)
├── 제조공정 상세
├── 안전성 평가 보고서 (CSAR)
├── 효능 평가 자료
├── 검사기관 테스트 보고서
│   ├── 위생학 검사
│   ├── 미생물 검사
│   ├── 중금속 검사
│   ├── 독성학 검사 (필요 시)
│   └── 인체 시험 (필요 시)
├── GMP 증명
├── 라벨 샘플
└── 제품 집행 기준

수입 추가:
├── 경내책임인 위임서
├── 원산지 증명
├── 자유판매증명서
├── 제조 공장 정보
└── 제조국 규제 적합성 증명
```

## CSAR (화장품안전평가보고서)

2021년부터 시행된 새로운 안전성 평가 체계

### CSAR 요구 사항

```
CSAR 구성:
├── 1. 안전성 위험물질 식별
│   ├── 성분별 위험성 평가
│   └── 불순물/오염물질 평가
├── 2. 노출 평가
│   ├── 사용 부위, 빈도, 양
│   └── 체내 흡수율
├── 3. 용량-반응 평가
│   ├── NOAEL/LOAEL 확인
│   └── 참조 용량 산출
├── 4. 위험 특성화
│   ├── 안전역 (MoS) 계산
│   └── 위험성 판정
└── 5. 안전성 결론
    └── 종합 평가 결과
```

### 안전성 평가자 자격

```
자격 요건:
├── 화학, 생물학, 의학, 약학 관련 전공
├── 5년 이상 화장품 안전성 평가 경력
├── NMPA 등록 자격자
└── 중국 내 근무 (외국인도 가능)
```

### 간소화 안전성 평가

특정 조건 충족 시 간소화된 평가 허용:
```
적용 조건:
├── 이미 안전성이 입증된 원료만 사용
├── 고시 원료 규격 내 사용
├── 과거 안전 사용 이력
└── 새로운 위험 정보 없음

간소화 내용:
├── 독성학 데이터 생략 가능
└── 기존 평가 자료 인용 가능
```

## 동물시험 요건

### 현행 규정 (2021년 이후)

| 제품 유형 | 동물시험 | 조건 |
|-----------|----------|------|
| 일반화장품 (국내) | **면제** | 2021년 5월부터 |
| 일반화장품 (수입) | **면제 가능** | 조건 충족 시 |
| 특수화장품 | **대부분 필수** | 일부 면제 가능 |
| 영유아용 | **필수** | 추가 테스트 |

### 동물시험 면제 조건 (수입 일반화장품)

```
면제 요건:
├── 1. 안전성 평가 자료 제출 (CSAR)
├── 2. 제조 공장 GMP 인증
│   ├── CGMP 또는
│   └── ISO 22716
├── 3. 3년간 무부작용 이력 (원산지에서)
├── 4. 원료 안전성 자료 확보
│   ├── 고시 원료 사용 또는
│   └── 개별 원료 안전성 입증
└── 5. 제조국 시장 판매 이력

추가 조건:
├── 경내책임인 지정
├── 불시 검사 협조 동의
└── 부작용 발생 시 동물시험 수행 동의
```

### 동물시험 필수 항목 (특수화장품)

```
필수 테스트 (해당 시):
├── 급성 경피 독성
├── 피부 자극/감작성
├── 광독성/광감작성
├── 안점막 자극성
├── 경피 흡수
└── 반복 투여 독성 (필요 시)

면제 가능 항목:
├── 기존 안전성 데이터 있는 경우
├── 동일 처방 이전 등록 이력
└── 대체시험법 인정 항목
```

## 성분 규제

### 화장품안전기술규범 (2021년판)

#### 금지 성분 (禁用成分)
```
주요 금지 성분:
├── 하이드로퀴논 (일부 예외)
├── 수은 및 수은화합물
├── 납 및 납화합물
├── 비소 및 비소화합물
├── 카드뮴 및 카드뮴화합물
├── 메탄올 (변성 알코올 예외)
├── 클로로포름
├── 포름알데히드 (방부제 목적 제한적 허용)
└── 약 1,400+ 종
```

#### 제한 성분 (限用成分)
```
주요 제한 성분 예시:

살리실산 (水杨酸):
├── Rinse-off: max 3%
├── Leave-on: max 2%
└── 3세 미만 금지

레티놀 (视黄醇):
├── 스킨케어: max 0.5%
└── 경고문 필수

과산화수소 (过氧化氢):
├── 헤어케어: max 12%
├── 스킨케어: max 4%
└── 경고문 필수

알파하이드록시산 (α-羟基酸):
├── Leave-on: max 6%
├── pH ≥ 3.5
└── 자외선차단 권고 표시
```

#### 허용 방부제 (准用防腐剂)
```
주요 방부제:
├── 페녹시에탄올: max 1.0%
├── 메틸파라벤: max 0.4% (단독)
├── 프로필파라벤: max 0.14%
├── 벤질알코올: max 1.0%
└── 총 51종 목록
```

#### 허용 UV 필터 (准用防晒剂)
```
주요 UV 필터:
├── 에칠헥실메톡시신나메이트: max 10%
├── 부틸메톡시디벤조일메탄: max 5%
├── 옥토크릴렌: max 10%
├── 티타늄디옥사이드: max 25%
├── 징크옥사이드: max 25%
└── 총 27종 목록
```

#### 허용 색소 (准用着色剂)
```
색소 분류:
├── 염료 (染料): 수용성
├── 안료 (颜料): 불용성
├── 레이크 (色淀): 염료 침전

사용 제한:
├── 눈 주변 허용 색소
├── 입술 허용 색소
├── 외부 사용만 허용
└── 순도 기준 준수
```

## 라벨 요건

### 화장품표식관리방법

#### 필수 표시 사항
```
1. 제품명 (产品名称)
   ├── 상표명 + 통용명 + 속성명
   └── 예: "XX牌 美白 精华液"

2. 전성분 (成分)
   ├── 함량 순 (1% 이상)
   ├── 1% 미만: 순서 무관
   └── INCI명 또는 중문명

3. 순 함량 (净含量)
   ├── g, mL, 개수
   └── 법정 단위 사용

4. 사용기한/생산일자
   ├── 생산일자 + 유효기간 또는
   └── 생산일자 + 사용기한
   └── 30개월 이상 시 개봉 후 사용기한

5. 생산기업 정보
   ├── 기업명, 주소
   └── 전화번호

6. 생산허가증 번호
   └── 국내 생산 시

7. 비안/주책 번호
   └── 해당 제품

8. 경내책임인 정보 (수입)
   ├── 기업명, 주소
   └── 전화번호

9. 원산지 (수입)
   └── 국가명

10. 사용방법/주의사항
    └── 해당 시
```

#### 중문 라벨 요건
```
수입 화장품:
├── 중문 라벨 필수
├── 원 라벨 위 부착 가능
├── 모든 필수 정보 중문 표기
└── 글자 크기 규정 준수

글자 크기:
├── 제품명: ≥ 3.5mm
├── 기타 정보: ≥ 2mm
└── 경고문: 눈에 띄게
```

## 검사기관 테스트

### 지정 검사기관

NMPA 지정 화장품 검사기관에서만 유효한 테스트 수행 가능

```
주요 검사기관:
├── 중국식품약품검정연구원
├── 광동약품검정연구원
├── 상해약품검정연구원
├── 북경시약품검정연구원
└── 기타 성급 검정기관
```

### 테스트 항목

#### 일반화장품
```
필수 테스트:
├── 미생물 검사
│   ├── 총호기성생균수
│   └── 병원성 균 (대장균, 녹농균 등)
├── 중금속 검사
│   ├── 납, 비소, 수은, 카드뮴
│   └── 특정 성분 (해당 시)
├── pH 측정
└── 안정성 확인 (자체)
```

#### 특수화장품
```
추가 테스트:
├── 효능 시험
│   ├── 미백: 멜라닌 억제 시험
│   ├── 자외선차단: SPF/PA 측정
│   ├── 탈모방지: 모발 성장 평가
│   └── 기타 효능별 시험
├── 독성학 시험 (필요 시)
│   ├── 피부 자극
│   ├── 안 자극
│   ├── 감작성
│   └── 광독성
└── 인체 시험 (필요 시)
```

## 경내책임인 (境内责任人)

### 정의 및 역할

수입 화장품의 **중국 내 법적 책임자**

```
경내책임인 자격:
├── 중국 법인 (내자 또는 외자)
├── 화장품 관련 영업 범위
├── 품질 안전 관리 역량
└── NMPA 등록

주요 의무:
├── 등록/비안 신청 대행
├── 제품 품질 안전 관리
├── 부작용 보고 및 대응
├── 리콜 실행
├── 당국 검사 협조
├── 기록 보관 (최소 5년)
└── 연락 창구 역할
```

## 최근 규제 변화 (2021-2025)

### 2021년 변화
```
화장품감독관리조례 시행:
├── CSAR 도입
├── 경내책임인 제도 강화
├── 동물시험 면제 확대
├── 처벌 강화 (과태료 증가)
└── 신원료 관리 강화
```

### 2022-2023년 변화
```
세부 규정 정비:
├── 화장품안전기술규범 개정
├── 효능 클레임 관리 강화
├── 어린이 화장품 규정 강화
└── 온라인 판매 관리 강화
```

### 향후 예정
```
예상 변화:
├── 신원료 등록 절차 간소화
├── 동물시험 대체 확대
├── 국제 상호 인증 추진
└── 디지털화 (전자 라벨 등)
```

## 실무 가이드

### 중국 시장 진입 체크리스트

```
□ 제품 분류 확인 (일반/특수)
□ 경내책임인 선정
□ 성분 규제 검토
  □ 금지 성분 확인
  □ 제한 성분 농도 확인
  □ 신원료 해당 여부
□ 동물시험 요건 확인
  □ 면제 조건 충족 여부
  □ 필요 시 테스트 계획
□ CSAR 준비
□ 검사기관 테스트
  □ 검사기관 선정
  □ 샘플 준비 및 제출
  □ 테스트 완료 대기
□ 등록/비안 신청
  □ 자료 준비
  □ 온라인 제출
  □ 심사 대응
□ 라벨 준비
  □ 중문 라벨 제작
  □ 필수 정보 확인
□ 시장 출시
```

## 요약

중국 화장품 규제의 핵심:

1. **이원화 체계**: 일반(비안) vs 특수(주책)
2. **CSAR**: 2021년부터 안전성 평가 의무화
3. **동물시험**: 일반화장품 면제 확대 (조건부)
4. **경내책임인**: 수입 시 중국 내 법적 책임자 필수
5. **검사기관 테스트**: NMPA 지정 기관 필수
6. **라벨**: 중문 라벨 및 모든 필수 정보 표기

중국 시장은 규제가 복잡하고 변화가 빠르므로, 전문 RA 컨설턴트 활용을 권장합니다.
