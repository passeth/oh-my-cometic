# 한국 화장품 규제 (MFDS) 상세 가이드

## 개요

**식품의약품안전처 (MFDS, Ministry of Food and Drug Safety)**는 한국의 화장품 규제를 총괄합니다.

### 법령 체계
```
화장품법
├── 화장품법 (법률)
│   └── 국회 제정, 기본 원칙
├── 화장품법 시행령 (대통령령)
│   └── 법률 위임 사항 세부 규정
├── 화장품법 시행규칙 (총리령)
│   └── 절차, 서식, 기준 규정
└── 고시/가이드라인
    ├── 화장품 안전기준 등에 관한 규정
    ├── 기능성화장품 기준 및 시험방법
    ├── 화장품 표시·광고 실증에 관한 규정
    └── 화장품 사용 시 주의사항 표시에 관한 규정
```

## 화장품 정의 및 분류

### 화장품의 정의 (화장품법 제2조)

> "화장품"이란 인체를 청결·미화하여 매력을 더하고 용모를 밝게 변화시키거나 피부·모발의 건강을 유지 또는 증진하기 위하여 인체에 바르고 문지르거나 뿌리는 등 이와 유사한 방법으로 사용되는 물품으로서 인체에 대한 작용이 경미한 것

### 분류 체계

| 분류 | 정의 | 규제 방식 |
|------|------|-----------|
| **일반화장품** | 기능성화장품이 아닌 화장품 | 사후 관리, 제조판매업 신고 |
| **기능성화장품** | 특정 효능·효과를 표방하는 화장품 | 사전 심사/신고/보고 |
| **유기농화장품** | 유기농 원료 일정 비율 이상 함유 | 별도 인증 |
| **천연화장품** | 천연 원료 일정 비율 이상 함유 | 별도 인증 |
| **맞춤형화장품** | 개인 맞춤 혼합·소분 | 맞춤형화장품조제관리사 필수 |

## 기능성화장품 상세

### 기능성화장품 정의 (화장품법 제2조 제2호)

다음 중 하나에 해당하는 화장품:
1. 피부의 미백에 도움을 주는 제품
2. 피부의 주름개선에 도움을 주는 제품
3. 피부를 곱게 태워주거나 자외선으로부터 피부를 보호하는 데에 도움을 주는 제품
4. 모발의 색상 변화·제거 또는 영양공급에 도움을 주는 제품
5. 체모를 제거하는 데에 도움을 주는 제품
6. 그 밖에 대통령령으로 정하는 제품

### 기능성화장품 13종 상세

| 번호 | 기능 | 고시 원료 예시 | 심사 유형 |
|------|------|----------------|-----------|
| 1 | **피부 미백** | 나이아신아마이드, 알부틴, 아스코빌글루코사이드 | 신고/허가 |
| 2 | **주름 개선** | 레티놀, 아데노신, 폴리에톡실레이티드레틴아마이드 | 신고/허가 |
| 3 | **자외선차단** | 에칠헥실메톡시신나메이트, 징크옥사이드 | 보고 |
| 4 | **피부 탄력 개선** | 고시 원료 없음 (허가 필요) | 허가 |
| 5 | **모발 색상 변화** | 염모제 성분 | 신고/허가 |
| 6 | **체모 제거** | 치오글리콜산칼슘 등 | 신고/허가 |
| 7 | **여드름 피부 완화** | 살리실릭애씨드, 레조시놀 | 신고/허가 |
| 8 | **아토피 피부 보습** | 판테놀, 세라마이드 류 | 신고/허가 |
| 9 | **탈모 증상 완화** | 덱스판테놀, 비오틴 | 신고/허가 |
| 10 | **튼살 붉은선 완화** | 고시 원료 없음 | 허가 |
| 11 | **체취 방지** | 염화알루미늄 등 | 신고/허가 |
| 12 | **피부장벽 기능 회복** | 세라마이드 NP 등 | 신고/허가 |
| 13 | **비듬/가려움 완화** | 징크피리치온, 살리실릭애씨드 | 신고/허가 |

### 심사 유형별 절차

#### 1. 품목허가 (심사형)
```
대상: 신규 성분 또는 고시 외 배합
기간: 약 60-90일
제출 서류:
├── 기능성화장품 심사 신청서
├── 안전성 자료 (독성시험, 인체적용시험)
├── 유효성 자료 (효능시험 결과)
├── 제조방법 및 기준/시험방법
├── 기원 및 개발경위
├── 안정성 시험 자료
└── 관련 자료 요약본
```

#### 2. 품목신고 (고시형)
```
대상: 고시 원료 규격 준수
기간: 약 10일
제출 서류:
├── 기능성화장품 신고서
├── 제조방법 및 기준/시험방법
├── 원료 규격 적합 입증 자료
└── 효능/효과 입증 자료 (일부 면제 가능)
```

#### 3. 보고 (자외선차단제)
```
대상: 고시 자외선차단제 원료 사용
기간: 즉시 (온라인 제출)
제출 서류:
├── 기능성화장품 보고서
├── SPF/PA 시험 결과
└── 자외선차단지수 표시 자료
```

## 전성분 표시 기준

### 기본 원칙

1. **표시 순서**: 함량 많은 순
2. **1% 미만**: 순서 무관하게 표시 가능
3. **착향제**: "향료"로 통합 표기 가능
4. **색소**: 개별 또는 "+/-" 표기

### 표시 방법

```
표시 예시:
정제수, 글리세린, 나이아신아마이드, 부틸렌글라이콜,
1,2-헥산다이올, 카보머, 트로메타민, 향료
```

### 성분명 표기

| 구분 | 표기 방법 |
|------|-----------|
| **INCI명** | 국제 표준명 사용 |
| **표준화성분명** | 대한화장품협회 표준 한글명 |
| **혼합원료** | 개별 성분 또는 혼합원료명 |
| **나노물질** | 성분명 뒤 "(나노)" 추가 |

### 표기 예외

- **영업비밀 성분**: 식약처 승인 시 일부 생략 가능
- **향료 구성성분**: 알레르기 유발 성분은 개별 표기 필수

## 화장품 표시/광고 규제

### 금지 표현

1. **의약품 오인** 유발 표현
   - "치료", "치유", "효과", "약효" 등
2. **기능성화장품 미심사** 효능 표방
3. **허위/과장** 광고
4. **비방** 광고
5. **의사/의료기관** 추천 표현 (의약품 오인)

### 실증 대상 표시/광고

다음 표현 사용 시 입증 자료 구비 필수:
- 효능/효과 관련 표현
- 수치/통계 사용
- 비교 광고
- 시험/연구 인용

### 주의사항 표시

특정 성분 함유 시 필수 표시:
```
- 과산화수소: "눈에 들어가지 않도록 주의"
- AHA (글리콜릭애씨드 등): "자외선차단제 사용 권장"
- 살리실릭애씨드: "3세 이하 어린이 사용금지"
- 레티놀: "임산부 사용 주의"
```

## 영업 신고 및 관리

### 영업 유형

| 유형 | 정의 | 신고처 |
|------|------|--------|
| 화장품제조업 | 직접 제조 | 지방 식약청 |
| 화장품책임판매업 | 제조 위탁/수입 판매 | 지방 식약청 |
| 맞춤형화장품판매업 | 혼합/소분 판매 | 지방 식약청 |

### 제조업 시설 기준

```
필수 시설:
├── 제조 작업소
│   ├── 원료 보관소
│   ├── 제조실
│   └── 충전/포장실
├── 품질관리 시설
│   ├── 품질검사실
│   └── 보관실
└── 위생 시설
    ├── 손 세척 시설
    └── 탈의/세탁 시설
```

### CGMP (우수화장품 제조 및 품질관리 기준)

ISO 22716 기반, 권장 사항이나 일부 해외 수출 시 필수

## 실무 체크리스트

### 신제품 개발 시
```
□ 제품 유형 결정 (일반/기능성/유기농/천연/맞춤형)
□ 배합 금지 원료 확인 (별표 1)
□ 배합 제한 원료 확인 (별표 2)
□ 기능성화장품 해당 여부 확인
□ 기능성 심사 유형 결정 (허가/신고/보고)
□ 효능 입증 시험 계획
□ 안정성 시험 계획
□ 전성분 표시 준비
□ 표시/광고 문구 검토
□ 주의사항 표시 확인
```

### 시장 출시 전
```
□ 기능성 심사/신고/보고 완료
□ 제품 등록 완료
□ 라벨 검토 완료
□ 광고 소재 법적 검토
□ 제조/수입 기록 문서화
```

## 요약

한국 화장품 규제의 핵심:

1. **기능성화장품 체계**: 효능 표방 시 사전 심사/신고/보고 필수
2. **안전기준**: 배합금지/제한 원료 철저 확인
3. **전성분 표시**: INCI명 또는 표준화성분명 사용
4. **광고 규제**: 의약품 오인, 과장광고 금지
5. **영업 신고**: 제조업/책임판매업 신고 필수

한국 시장 진출 또는 국내 제조 시 반드시 화장품법 체계를 이해하고 준수해야 합니다.
