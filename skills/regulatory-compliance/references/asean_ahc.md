# ASEAN 화장품 규제 (ACD) 상세 가이드

## 개요

**ASEAN Cosmetic Directive (ACD)**는 ASEAN 10개국의 화장품 규제를 통일화하는 지역 지침으로, EU 규정을 기반으로 합니다.

### 기본 정보
| 항목 | 내용 |
|------|------|
| **지침명** | ASEAN Cosmetic Directive (ACD) |
| **근거** | ASEAN Cosmetic Committee (ACC) |
| **모델** | EU Regulation (EC) 1223/2009 기반 |
| **회원국** | 10개국 |
| **시행** | 2008년 1월 1일 |

### ASEAN 회원국

```
회원국 (알파벳순):
├── Brunei (브루나이)
├── Cambodia (캄보디아)
├── Indonesia (인도네시아)
├── Laos (라오스)
├── Malaysia (말레이시아)
├── Myanmar (미얀마)
├── Philippines (필리핀)
├── Singapore (싱가포르)
├── Thailand (태국)
└── Vietnam (베트남)
```

## ACD 핵심 원칙

### 1. 상호인증체계 (MRA)

```
상호인증 원칙:
├── 한 회원국에서 합법적 판매 제품은
│   타 회원국에서도 판매 가능
├── 성분 목록 통일 (금지/제한)
├── 안전성 평가 기준 공유
└── 라벨 요건 조화

실제 적용:
├── 완전한 자유 이동은 아님
├── 각국 등록/통지 여전히 필요
└── 국가별 추가 요건 존재
```

### 2. 사후 관리 체계

```
규제 방식:
├── 사전 승인 불필요 (대부분 국가)
├── 통지/등록 의무
├── 안전성 책임은 제조사/수입사
└── 사후 시장 감시 (PMS)

예외:
├── 일부 국가 특정 제품 사전 심사
└── 일부 기능성 클레임 검증 요구
```

### 3. 통일된 기술 문서

```
제품 정보 파일 (PIF):
├── EU PIF와 유사한 구조
├── 안전성 평가 자료
├── 성분 정보
├── 제조 정보
└── 효능 입증 자료 (클레임 시)
```

## ASEAN 공통 규정

### ASEAN Cosmetic Ingredient Listing

EU Annex 체계를 기반으로 함

#### 금지 성분 (Negative List)
```
EU Annex II 기반:
├── 약 1,300+ 종 금지
├── 주요 금지 성분:
│   ├── Hydroquinone (대부분 금지)
│   ├── Mercury compounds
│   ├── Lead compounds
│   ├── Antibiotics
│   └── Hormones
└── 정기 업데이트 (EU 연동)
```

#### 제한 성분 (Restricted List)
```
EU Annex III 기반:
├── 약 300+ 종 제한
├── 농도/용도 제한
├── 주요 제한 성분:
│   ├── Salicylic Acid
│   ├── Retinol
│   ├── Alpha-hydroxy Acids
│   ├── Hydrogen Peroxide
│   └── Hair dye ingredients
└── 라벨 경고문 요건
```

#### 허용 방부제 (Positive List - Preservatives)
```
EU Annex V 기반:
├── 59종 허용 방부제
├── 최대 농도 규정
├── 주요 방부제:
│   ├── Phenoxyethanol: 1.0%
│   ├── Parabens: 0.4%/0.8%
│   ├── Benzyl Alcohol: 1.0%
│   └── MIT/CMIT: 제한적
└── 목록 외 방부제 금지
```

#### 허용 UV 필터 (Positive List - UV Filters)
```
EU Annex VI 기반:
├── 28종 허용 UV 필터
├── 최대 농도 규정
├── 주요 UV 필터:
│   ├── Titanium Dioxide: 25%
│   ├── Zinc Oxide: 25%
│   ├── Octinoxate: 10%
│   ├── Avobenzone: 5%
│   └── Octocrylene: 10%
└── 목록 외 UV 필터 금지
```

#### 허용 색소 (Positive List - Colorants)
```
EU Annex IV 기반:
├── 허용 색소 목록
├── 사용 부위 제한
├── 순도 기준
└── CI 번호 체계 사용
```

## ASEAN Cosmetic Claim Guideline

### 허용 클레임

```
허용 클레임 유형:
├── 미용적 효과
│   ├── "Moisturizes skin"
│   ├── "Softens skin"
│   ├── "Improves skin texture"
│   └── "Enhances radiance"
├── 감각적 특성
│   ├── "Non-greasy"
│   ├── "Light texture"
│   └── "Fresh feeling"
├── 제품 특성
│   ├── "Long-lasting"
│   ├── "Water-resistant"
│   └── "Quick-absorbing"
└── 성분 관련
    ├── "Contains Vitamin E"
    └── "With natural extracts"
```

### 금지 클레임

```
금지 클레임 유형:
├── 의약품적 효능
│   ├── "Treats eczema"
│   ├── "Cures acne"
│   └── "Heals wounds"
├── 질병 예방
│   ├── "Prevents skin cancer"
│   └── "Stops hair loss"
├── 신체 구조/기능 변화
│   ├── "Increases collagen production"
│   └── "Regenerates cells"
├── 절대적 주장
│   ├── "100% effective"
│   └── "Guaranteed results"
└── 허위/오해 유발
    ├── 증명되지 않은 주장
    └── 경쟁사 비방
```

### 조건부 클레임

```
입증 필요 클레임:
├── 자외선차단
│   ├── SPF 테스트 필수
│   └── PA 테스트 (UVA)
├── 미백/화이트닝
│   ├── 일부 국가 특수 분류
│   └── 효능 입증 요구
├── 안티에이징
│   ├── 효능 테스트 권장
│   └── 과장 금지
└── 수치 클레임
    ├── 테스트 데이터 필요
    └── 방법론 명시
```

## 국가별 규제 상세

### 인도네시아 (BPOM)

**BPOM (Badan Pengawas Obat dan Makanan)**

```
기본 정보:
├── 규제 기관: BPOM
├── 등록 시스템: e-Registration (AeRO)
├── 등록 유형: 통지 (Notification)
├── 유효 기간: 5년

등록 절차:
├── 1. BPOM e-Registration 계정 생성
├── 2. 제품 정보 입력
├── 3. 서류 업로드
│   ├── 안전성 평가 자료
│   ├── 성분 목록 (INCI명)
│   ├── 제조 정보
│   ├── 라벨 샘플
│   └── GMP 증명 (ISO 22716)
├── 4. 검토 (약 14-30 영업일)
├── 5. 등록번호 발급 (MD/ML 번호)
└── 6. 시장 출시

특이사항:
├── 할랄 인증 권장 (의무화 예정)
├── 일부 성분 추가 제한
├── 인도네시아어 라벨 필수
└── 현지 수입자(Importer) 필요
```

### 태국 (Thai FDA)

**Thai FDA (Food and Drug Administration)**

```
기본 정보:
├── 규제 기관: Thai FDA
├── 등록 시스템: e-Submission
├── 등록 유형: 통지 (Notification)
├── 유효 기간: 3년

화장품 분류:
├── Controlled Cosmetics (특별 관리)
│   ├── 자외선차단제
│   ├── 염모제
│   └── 일부 미백 제품
├── General Cosmetics (일반)
│   └── 기타 화장품
└── 각 분류별 요건 상이

등록 절차:
├── 1. Thai FDA 계정 생성
├── 2. 제품 분류 확정
├── 3. 서류 제출
│   ├── 안전성 평가
│   ├── 성분 분석 증명서 (CoA)
│   ├── 제조 증명
│   └── 태국어 라벨
├── 4. 심사 (약 15-60 영업일)
├── 5. 등록번호 발급
└── 6. 시장 출시

특이사항:
├── 특정 성분 추가 테스트 요구
├── 태국어 라벨 필수
├── 현지 수입자 필요
└── 비안 후 샘플 검사 가능
```

### 베트남 (MOH)

**베트남 보건부 (Ministry of Health)**

```
기본 정보:
├── 규제 기관: MOH / Drug Administration of Vietnam (DAV)
├── 등록 시스템: Online Portal
├── 등록 유형: 통지 (Notification)
├── 유효 기간: 5년 (신규), 무기한 (갱신)

등록 절차:
├── 1. 현지 수입자 지정
├── 2. 서류 준비
│   ├── 통지 신청서
│   ├── 제품 요약서
│   ├── 안전성 평가
│   ├── GMP 증명
│   ├── Free Sale Certificate
│   └── 베트남어 라벨
├── 3. 서류 제출 (온라인)
├── 4. 심사 (약 15-30 영업일)
├── 5. 등록번호 발급
└── 6. 시장 출시

특이사항:
├── 베트남어 라벨 필수
├── 현지 수입자 필수
├── FSC (Free Sale Certificate) 필수
└── 일부 성분 추가 제한
```

### 필리핀 (FDA Philippines)

```
기본 정보:
├── 규제 기관: FDA Philippines
├── 등록 시스템: e-Services / e-Registration
├── 등록 유형: CPR (Certificate of Product Registration)
├── 유효 기간: 5년

화장품 분류:
├── With Cosmetic Claim
│   └── 미용적 효과만
├── With Drug Claim (OTC)
│   └── 치료적 효과 (FDA Drug 규정)
└── 분류에 따라 요건 상이

등록 절차:
├── 1. FDA 계정 생성
├── 2. LTO (License to Operate) 취득
├── 3. CPR 신청
│   ├── 제품 정보
│   ├── 안전성 자료
│   ├── 제조 정보
│   └── 라벨
├── 4. 심사 (약 30-90 영업일)
├── 5. CPR 발급
└── 6. 시장 출시

특이사항:
├── LTO 먼저 취득 필요
├── 영어 + 필리핀어 라벨
├── 현지 수입자/유통자 필요
└── 일부 클레임 입증 요구
```

### 말레이시아 (NPRA)

**NPRA (National Pharmaceutical Regulatory Agency)**

```
기본 정보:
├── 규제 기관: NPRA
├── 등록 시스템: QUEST3+
├── 등록 유형: 통지 (Notification)
├── 유효 기간: 영구 (변경 시 업데이트)

등록 절차:
├── 1. QUEST3+ 계정 생성
├── 2. 제품 통지 제출
│   ├── 통지 양식
│   ├── 성분 목록
│   ├── 안전성 평가
│   ├── GMP 증명
│   └── 라벨 (말레이어 + 영어)
├── 3. 자동 승인 (요건 충족 시)
├── 4. 통지번호 발급 (NOT 번호)
└── 5. 시장 출시

특이사항:
├── 말레이어 + 영어 라벨 필수
├── 할랄 인증 중요 (무슬림 시장)
├── 현지 통지 보유자 필요
├── 일부 성분 추가 제한
└── 미백 제품 주의 (민감 클레임)
```

### 싱가포르 (HSA)

**HSA (Health Sciences Authority)**

```
기본 정보:
├── 규제 기관: HSA
├── 등록 시스템: PRISM
├── 등록 유형: 통지 (Notification)
├── 유효 기간: 영구 (변경 시 업데이트)

규제 특징:
├── ASEAN 중 가장 선진화된 체계
├── 사후 관리 중심
├── 신속한 통지 절차
└── 높은 규제 투명성

등록 절차:
├── 1. PRISM 계정 생성
├── 2. 화장품 통지 제출
│   ├── 제품 정보
│   ├── 성분 목록
│   ├── 안전성 선언
│   └── 라벨
├── 3. 자동 승인 (요건 충족 시)
├── 4. 통지 확인
└── 5. 시장 출시

특이사항:
├── 영어 라벨 (공용어)
├── 신속한 처리 (1-3 영업일)
├── 높은 규정 준수 기대
├── PMS (사후 시장 감시) 활발
└── EU 기준과 거의 동일
```

## 라벨 요건

### ASEAN 공통 라벨 요건

```
필수 표시 사항:
├── 1. 제품명
├── 2. 용도/기능
├── 3. 성분 목록 (INCI명)
│   └── 함량 순서
├── 4. 사용 방법
├── 5. 주의사항/경고문
├── 6. 제조사/수입사 정보
├── 7. 순 중량/용량
├── 8. 제조일/사용기한
├── 9. 배치 번호
└── 10. 원산지

언어 요건:
├── 현지어 필수 (국가별 상이)
├── 영어 병기 권장
└── 원 라벨 위 스티커 부착 가능
```

## 실무 가이드

### ASEAN 시장 진입 전략

```
권장 접근법:
├── 1단계: 싱가포르/말레이시아 진출
│   └── 비교적 간단한 절차, 영어 사용
├── 2단계: 태국/필리핀 확장
│   └── 중간 난이도, 시장 규모 큼
├── 3단계: 인도네시아 진출
│   └── 가장 큰 시장, 복잡한 요건
└── 4단계: 기타 국가 확장
    └── 베트남, 캄보디아 등
```

### ASEAN 진입 체크리스트

```
□ 성분 검토 (ASEAN Cosmetic Ingredient Listing)
  □ 금지 성분 확인
  □ 제한 성분 농도 확인
  □ 국가별 추가 제한 확인
□ 안전성 평가 준비
  □ PIF 작성
  □ 안전성 평가 보고서
□ GMP 증명 확보
  □ ISO 22716 인증 또는
  □ 동등 GMP 증명
□ 현지 대리인/수입자 선정
  □ 국가별 요건 확인
  □ 신뢰할 수 있는 파트너
□ 라벨 준비
  □ 현지어 번역
  □ 필수 정보 포함
```

## 요약

ASEAN 화장품 규제의 핵심:

1. **ACD 기반**: EU 규정을 모델로 한 통일 지침
2. **상호인증**: 원칙적으로 상호 인정, 실제로는 국가별 등록 필요
3. **통일 성분 목록**: 금지/제한/허용 성분 ASEAN 공통
4. **국가별 차이**: 등록 절차, 언어, 추가 요건 상이
5. **할랄**: 인도네시아, 말레이시아에서 중요

ASEAN 시장은 EU 기준을 충족하면 진입이 용이하나, 국가별 특수 요건(언어, 현지 대리인, 할랄 등)에 주의가 필요합니다.
