# Equipment Capacity Reference

## Overview

화장품 제조 설비의 용량과 사양을 이해하는 것은 효율적인 배치 크기 결정과 생산 계획 수립에 필수적입니다. 이 문서는 주요 제조 설비의 용량, 사양, 선택 가이드를 제공합니다.

## Mixing Equipment

### 1. 유화 믹서 (Emulsification Mixer)

화장품 제조의 핵심 설비로, 유화 및 혼합을 수행합니다.

#### 용량별 분류

```
유화 믹서 용량 등급:
─────────────────────────────────────────────────────
등급          공칭 용량    유효 용량*   용도
─────────────────────────────────────────────────────
Lab           1-5L        0.5-3L       연구개발
Bench         10-50L      7-35L        소량 생산, 파일럿
Pilot         100-300L    70-210L      파일럿 생산
Small Prod.   500-1000L   350-700L     소규모 생산
Medium Prod.  2000-5000L  1400-3500L   중규모 생산
Large Prod.   10000L+     7000L+       대규모 생산
─────────────────────────────────────────────────────
* 유효 용량 = 공칭 용량 × 0.7 (유화 팽창, 교반 공간 고려)
```

#### 주요 사양 항목

| 항목 | 설명 | 일반적인 범위 |
|------|------|--------------|
| 공칭 용량 | 탱크 최대 용량 | 1L - 20,000L |
| 유효 용량 | 실제 작업 용량 | 공칭의 60-80% |
| 앵커 속도 | 메인 교반기 속도 | 10-100 rpm |
| 호모 속도 | 균질기 속도 | 1,000-10,000 rpm |
| 가열 방식 | 스팀/전기/온수 | - |
| 최대 온도 | 가열 가능 온도 | 90-150°C |
| 냉각 방식 | 냉각수/브라인 | - |
| 진공 기능 | 탈포용 진공 | -0.08 ~ -0.095 MPa |

### 설비 용량 매칭

배치 크기와 설비 용량을 맞추는 가이드입니다.

```python
def match_equipment_capacity(
    target_batch_kg: float,
    formulation_type: str,
    density: float = 1.0
) -> dict:
    """
    목표 배치에 적합한 설비 용량 추천

    Args:
        target_batch_kg: 목표 배치 크기 (kg)
        formulation_type: 제형 타입
        density: 제품 밀도 (kg/L)

    Returns:
        추천 설비 사양
    """
    # 배치 부피 계산
    batch_volume_L = target_batch_kg / density

    # 유화기 용량 (70% 채움 기준)
    mixer_capacity = batch_volume_L / 0.70

    # 근처 표준 용량으로 반올림
    standard_capacities = [5, 10, 20, 50, 100, 200, 300, 500, 1000, 2000, 3000, 5000]
    recommended_capacity = min(
        [c for c in standard_capacities if c >= mixer_capacity],
        default=standard_capacities[-1]
    )

    # 실제 유효 용량
    effective_volume = recommended_capacity * 0.70
    effective_batch = effective_volume * density

    return {
        "target_batch_kg": target_batch_kg,
        "target_volume_L": round(batch_volume_L, 1),
        "recommended_mixer_L": recommended_capacity,
        "effective_volume_L": round(effective_volume, 1),
        "effective_batch_kg": round(effective_batch, 1),
        "fill_ratio": round(batch_volume_L / recommended_capacity * 100, 1),
        "formulation_type": formulation_type
    }

# 예시
# match_equipment_capacity(350, "cream", density=0.98)
# => recommended_mixer_L: 500L
# => effective_batch_kg: 343kg
# => fill_ratio: 71.4%
```

## Summary

설비 선택과 용량 결정의 핵심 원칙:

1. **여유 용량 확보**: 공칭 용량의 70-80%를 실제 배치 크기로 설정
2. **확장성 고려**: 향후 생산량 증가를 고려한 용량 선택
3. **제형 적합성**: 제품 특성에 맞는 설비 타입 선택
