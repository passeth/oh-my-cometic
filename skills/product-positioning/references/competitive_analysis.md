# Competitive Analysis Methodology for Cosmetics

## Overview

Effective competitive analysis in cosmetics requires systematic evaluation of market players, their strategies, strengths, and vulnerabilities. This guide provides a comprehensive methodology for conducting competitive analysis.

## 1. Competitor Identification Framework

### Layers of Competition

```
┌─────────────────────────────────────────────────────────┐
│                    BUDGET COMPETITION                    │
│  (All products competing for same consumer dollars)      │
│  ┌─────────────────────────────────────────────────┐    │
│  │              GENERIC COMPETITION                 │    │
│  │  (All skincare/beauty solutions)                │    │
│  │  ┌─────────────────────────────────────────┐   │    │
│  │  │         CATEGORY COMPETITION             │   │    │
│  │  │  (Same product category, e.g., serums)  │   │    │
│  │  │  ┌─────────────────────────────────┐    │   │    │
│  │  │  │     DIRECT COMPETITION          │    │   │    │
│  │  │  │  (Same segment, same target)    │    │   │    │
│  │  │  │                                 │    │   │    │
│  │  │  └─────────────────────────────────┘    │   │    │
│  │  └─────────────────────────────────────────┘   │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

### Competitor Classification Matrix

| Type | Definition | Example (Vitamin C Serum) |
|------|------------|---------------------------|
| **Direct** | Same category, same segment, same target | Klairs Vitamin C, Some By Mi Galactomyces |
| **Indirect** | Same need, different solution | Niacinamide serums, brightening toners |
| **Potential** | Could enter category | Derma brands expanding into brightening |
| **Substitute** | Alternative solutions | Professional treatments, supplements |

## Korean Market Competitive Landscape

### Major Player Profiles

#### Amorepacific Group
- **Brands**: Sulwhasoo, Hera, Laneige, Mamonde, Innisfree, Etude
- **Strength**: Full tier coverage, R&D capability, global presence
- **Strategy**: Brand portfolio diversification, premiumization

#### LG H&H
- **Brands**: Whoo, Ohui, Su:m37, Belif, VDL, The Face Shop
- **Strength**: Heritage positioning, duty-free dominance
- **Strategy**: Luxury focus, Chinese market emphasis

#### Olive Young Private Label
- **Brands**: Round Lab, Medicube (exclusive partnerships)
- **Strength**: Distribution control, trend responsiveness
- **Strategy**: Curation power, DTC hybrid

### Competitive Dynamics by Tier

| Tier | Competition Level | Key Success Factors |
|------|-------------------|---------------------|
| Mass | Very High | Price, availability, trend speed |
| Masstige | High | Quality perception, brand story |
| Prestige | Medium | Efficacy proof, brand prestige |
| Luxury | Lower | Heritage, exclusivity, experience |

## Output: Competitive Analysis Report Structure

### Recommended Format

1. **Executive Summary** (1 page)
   - Key findings
   - Strategic implications
   - Priority recommendations

2. **Market Overview** (2-3 pages)
   - Category size and growth
   - Key trends
   - Channel dynamics

3. **Competitive Landscape** (5-7 pages)
   - Major player analysis
   - Strategic group mapping
   - Positioning comparison

4. **Detailed Competitor Profiles** (2-3 pages each)
   - Full SWOT analysis
   - Product portfolio
   - Strategy assessment

5. **Competitive Opportunities** (2-3 pages)
   - White space analysis
   - Vulnerability assessment
   - Positioning opportunities

6. **Recommendations** (2-3 pages)
   - Strategic options
   - Action priorities
   - Response planning
